DATA CLASSIFICATION
═══════════════════════════════════════════════════════════════════════════════

Stored Data (PVC):
  - Type: Inference payloads, model predictions, feature data
  - Sensitivity: Medium to High (depends on model inputs)
  - Encryption at Rest: Depends on storage provider configuration
  - Access Control: Filesystem permissions (pod user context)
  - Retention: User-defined (manual cleanup required)

In-Transit Data:
  - External to Route: HTTPS TLS 1.2+ (encrypted)
  - Route to OAuth Proxy: HTTPS TLS 1.2+ service-ca (encrypted)
  - OAuth Proxy to TrustyAI Service: HTTP (plaintext, localhost)
  - InferenceService to TrustyAI: HTTP (plaintext, cluster-internal)
  - Operator to Kubernetes API: HTTPS TLS 1.2+ (encrypted)

Metrics Data:
  - Type: Aggregated fairness and bias metrics (trustyai_spd, trustyai_dir)
  - Sensitivity: Low to Medium
  - Exposure: Prometheus scraping (cluster-internal)
  - Retention: Prometheus retention policy (default 15 days)
