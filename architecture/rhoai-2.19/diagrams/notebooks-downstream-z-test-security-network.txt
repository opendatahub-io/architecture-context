  - code-server (8080/TCP): OAuth proxy / kube-rbac-proxy (injected sidecar), per-user RBAC

Secrets:
  - rhel-subscription-secret (Opaque) - RHEL subscription for RStudio RHEL9 BuildConfig

Container Security:
  - All images run as non-root (UID 1001)
  - Base images: Red Hat UBI9/RHEL9 with security updates
  - Build scanning: Clair (vulnerabilities), ClamAV (malware), SAST (Snyk, Coverity)
  - RPM signature scanning, deprecated image checks, ecosystem cert preflight
  - Images pinned by SHA256 digest for reproducibility
