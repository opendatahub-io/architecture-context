SECURITY CONFIGURATION
======================

TLS Configuration (MetadataStoreServerConfig.SSLConfig):
  - server_key: PEM-encoded private server key
  - server_cert: PEM-encoded public server certificate
  - custom_ca: PEM-encoded custom certificate authority
  - client_verify: When true, requires and verifies client certs (mTLS)
  - Default: Insecure (plaintext gRPC) when no ssl_config provided

Authentication & Authorization:
  - Default: No authentication (InsecureServerCredentials)
  - Optional: mTLS via SSLConfig with client_verify=true
  - No application-level authorization (all RPCs accessible if connected)
