  - RUNTIME_TLS_SERVER_CERT: Server TLS certificate

  - RUNTIME_TLS_SERVER_CERT: Server TLS certificate
  - RUNTIME_TLS_CLIENT_CERT: Client certificate for mTLS verification
  - Provisioned by: Platform / cert-manager
  - Auto-rotate: No (manual)

  - client_key_file: Client key for TGIS backend mTLS
