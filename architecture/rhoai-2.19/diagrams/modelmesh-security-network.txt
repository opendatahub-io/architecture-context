TLS Configuration:
================================================================================
| Setting                    | Env Var                      | Purpose                                          |
|----------------------------|------------------------------|--------------------------------------------------|
| Private Key Path           | MM_TLS_PRIVATE_KEY_PATH      | PKCS8 private key for external gRPC & inter-pod   |
| Certificate Path           | MM_TLS_KEY_CERT_PATH         | Certificate for TLS                                |
| Client Auth Mode           | MM_TLS_CLIENT_AUTH           | NONE, REQUIRE, or OPTIONAL mTLS                   |
| Trust Certs                | MM_TLS_TRUST_CERT_PATH       | CA certs for client verification                   |
| Internal Trust Certs       | MM_INTERNAL_TRUST_CERTS      | Additional certs for inter-pod (cert rotation)     |
| Intracluster Client Auth   | MM_INTRACLUSTER_CLIENTAUTH   | Client auth for internal litelinks TLS             |
| FIPS Disabled              | -Dcom.redhat.fips=false      | Java FIPS mode disabled (breaks with TLS)          |
