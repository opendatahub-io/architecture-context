Container Security:
===================

- Non-root execution: All containers (UIDs: 1001, 2000, 5000, 65532)
- Base images: Red Hat UBI 8/9 or RHEL 9
- Image scanning: Clair, ClamAV, SAST (Snyk, Coverity) via Konflux
- Image signing: Enterprise Contract verification
- Image pinning: SHA256 digest for all 27 RELATED_IMAGE references
- Disconnected support: All 27 images via RELATED_IMAGE env vars
