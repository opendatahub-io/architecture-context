## Purpose
**Short**: A Kubernetes operator that manages the lifecycle of Data Science Pipelines (DSP) applications and their associated infrastructure components on OpenShift.

**Detailed**: The Data Science Pipelines Operator (DSPO) provides a declarative way to deploy and manage complete Data Science Pipelines installations through a single Custom Resource (DataSciencePipelinesApplication / DSPA). When a user creates a DSPA CR in a namespace, the operator reconciles and deploys all required sub-components: the DSP API Server, Persistence Agent, Scheduled Workflow controller, ML Metadata (MLMD) services, and optionally a database (MariaDB) and object storage (Minio).

The operator uses a template-based approach via the `manifestival` library, rendering Go templates from `config/internal/` into Kubernetes resources. It performs health checks against the database and object storage backends before deploying pipeline components, and publishes Prometheus metrics for each component's readiness state. The operator supports both managed (MariaDB/Minio) and external (user-provided DB/S3) backend configurations, pod-to-pod TLS for inter-component encryption, custom CA bundles for enterprise TLS environments, and OAuth proxy integration for external API access through OpenShift Routes.

The operator is deployed as part of Red Hat OpenShift AI (RHOAI) and is managed by the parent RHOAI operator through the DataScienceCluster CR's `datasciencepipelines` component.

| MariaDB | Deployment (templated, optional) | Default managed database for pipeline metadata storage (3306/TCP) |
| Minio | Deployment (templated, optional) | Default managed object storage for pipeline artifacts (9000/TCP) |
| Workflow Controller | Deployment (templated, optional) | Argo Workflow controller for pipeline execution orchestration |
| ML Pipeline UI | Deployment (templated, optional) | KFP UI frontend (unsupported, for development/testing) |
| OAuth Proxy | Sidecar container | OpenShift oauth-proxy sidecar on API Server and MLMD Envoy for authenticated external access (8443/TCP) |
