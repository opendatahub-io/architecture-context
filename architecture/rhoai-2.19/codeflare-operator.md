## Purpose
**Short**: The CodeFlare Operator manages the lifecycle of RayCluster and AppWrapper custom resources, providing OAuth proxy integration, mTLS certificate management, and network policy enforcement for distributed computing workloads on OpenShift.

**Detailed**: The CodeFlare Operator is a controller-runtime based Kubernetes operator that extends OpenShift/Kubernetes clusters with distributed computing capabilities. It watches RayCluster custom resources (from the KubeRay project) and dynamically creates supporting infrastructure including OpenShift Routes for dashboard access, OAuth proxy sidecars for authentication, mTLS certificates for inter-node communication, NetworkPolicies for workload isolation, and Ingress resources for vanilla Kubernetes environments.

The operator also embeds an optional AppWrapper controller (from the project-codeflare/appwrapper project) that integrates with Kueue for workload queuing and quota management. When enabled, AppWrappers provide a mechanism to wrap multiple Kubernetes resources (Deployments, Jobs, RayClusters, PyTorchJobs, etc.) under a single unit of scheduling, enabling gang scheduling and quota-aware resource allocation. The operator uses mutating and validating webhooks to inject OAuth proxy sidecars and mTLS init containers into RayCluster head pods, and to validate AppWrapper resources.

Configuration is driven by a ConfigMap (`codeflare-operator-config`) that controls features like Ray Dashboard OAuth enablement, mTLS, ingress domain, and AppWrapper settings. The operator auto-detects whether it's running on OpenShift (creating Routes) or vanilla Kubernetes (creating Ingresses).

| AppWrapper Validating Webhook | Admission Webhook | Validates AppWrapper resources on create/update with SubjectAccessReview |
| Cert Rotator | Certificate Management | Manages webhook TLS certificates using open-policy-agent/cert-controller, auto-generates and rotates certs for webhook server |
| Operator Manager | Go Binary | Main entry point; configures controller manager, leader election, health probes, and dynamically detects APIs (RayCluster, Kueue Workload) |
