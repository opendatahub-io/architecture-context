## Purpose
**Short**: Central build configuration repository that defines the RHOAI operator bundle, FBC (File-Based Catalog) fragments, and image references for all RHOAI 2.19 shipped components.

**Detailed**: RHOAI-Build-Config is the authoritative source for the Red Hat OpenShift AI (RHOAI) product build pipeline. It does not contain application source code or operator controllers; instead, it orchestrates the assembly of the RHOAI operator bundle and OLM catalog artifacts that are shipped to customers.

The repository manages the ClusterServiceVersion (CSV), CRD manifests, related image references (27 container images), and FBC catalog entries for the rhods-operator across multiple supported OpenShift versions (v4.14, v4.16, v4.17, v4.18, v4.19). It integrates with the Konflux CI/CD system via Tekton PipelineRuns and GitHub Actions workflows to automate bundle processing, catalog generation, nightly builds, and release artifact creation.

The repository also contains PNC (Project Newcastle) build configurations for specific Java-based components (ModelMesh, TrustyAI) that require Maven-based builds, and stores release artifacts with Enterprise Contract verification pipelines for supply chain security.
