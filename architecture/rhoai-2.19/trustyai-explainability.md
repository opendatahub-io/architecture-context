## Purpose
**Short**: TrustyAI is a Java library and Quarkus REST service providing AI fairness metrics, bias tracking, model drift detection, and explainability algorithms for ML models served via ModelMesh and KServe.

**Detailed**: TrustyAI Explainability is a comprehensive toolkit for Responsible AI within the Red Hat OpenShift AI (RHOAI) platform. It provides fairness metrics (Statistical Parity Difference, Disparate Impact Ratio), model drift detection (KS Test, Approximate KS Test, Fourier MMD, Meanshift), and local/global explainability algorithms (LIME, SHAP, Counterfactual, Partial Dependence Plots, Time Series Saliency).

The service integrates with ModelMesh and KServe model serving platforms to intercept inference payloads (inputs and outputs) via REST endpoints and CloudEvents. These payloads are reconciled and stored for continuous fairness monitoring and drift detection. Metrics are published to Prometheus via a Micrometer registry and scraped by a ServiceMonitor. The service supports multiple storage backends including PVC-based flat file storage, MinIO object storage, and MariaDB/MySQL databases via Hibernate ORM.

In RHOAI, the service is deployed per-namespace alongside model serving infrastructure. It consumes inference data from ModelMesh's payload processing pipeline and from KServe's inference logger CloudEvents, enabling continuous bias monitoring and model explainability without modifying the model serving workflow.
