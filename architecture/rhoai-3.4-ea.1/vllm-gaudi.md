## Purpose
**Short**: vLLM hardware plugin that enables optimized large language model inference on Intel Gaudi AI accelerators.

**Detailed**: The vllm-gaudi repository provides the vLLM Hardware Plugin for Intel Gaudi accelerators (HPUs). It integrates Intel Gaudi AI accelerators with the vLLM inference engine, implementing hardware-specific optimizations for attention backends, KV-cache management, distributed communication, model quantization (FP8, compressed-tensors, AWQ, GPTQ), and speculative decoding on HPU devices.

The plugin follows vLLM's pluggable hardware architecture, registering itself via Python entry points (`vllm.platform_plugins` and `vllm.general_plugins`). It provides a custom `HpuPlatform` class, HPU-specific worker and model runner implementations, custom attention backends (including unified attention and MLA), and HCCL-based distributed communication. The container image ships as an OpenAI-compatible API server running on Red Hat UBI 9.6 with Intel Synapse AI 1.23.0 and PyTorch 2.9.0.

Within the RHOAI platform, this component provides the serving runtime container image for deploying LLM inference workloads on Intel Gaudi hardware. It is used by KServe as an inference runtime, exposing an OpenAI-compatible HTTP API endpoint for model serving.
