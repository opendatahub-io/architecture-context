## Purpose
**Short**: Argo Workflows is a container-native workflow engine for orchestrating parallel jobs on Kubernetes, used as the execution backend for Data Science Pipelines in RHOAI.

**Detailed**: Argo Workflows provides the core workflow execution engine that powers Data Science Pipelines in Red Hat OpenShift AI. It defines workflows as Kubernetes Custom Resources (CRDs) and executes them by orchestrating pods on the cluster. Each step in a workflow runs as a separate container, with the workflow controller managing the lifecycle, dependencies, and state transitions.

The component produces two container images for RHOAI: the **workflow-controller** (which watches Workflow CRs, creates pods, and manages execution state) and the **argoexec** executor (which runs as a sidecar/init container in workflow pods to manage artifacts, wait for conditions, and report results). The upstream argo-server (API/UI server) is also present in the source but is not shipped as a separate RHOAI image -- the Data Science Pipelines Operator manages the API layer instead.

Both images are built with FIPS compliance support via `GOEXPERIMENT=strictfipsruntime` and are based on UBI9 (ubi-minimal). The Konflux build system is used for production builds.

## Architecture Components

| Component | Type | Purpose |
|-----------|------|---------|
| workflow-controller | Go Controller | Watches Workflow CRs, creates and manages pods for workflow steps, handles scheduling, retries, artifact GC, cron workflows, and TTL cleanup. Uses leader election via Kubernetes Leases. |
| argoexec | Go Executor (sidecar) | Runs inside workflow step pods to manage artifacts (upload/download), wait for container completion, capture outputs, and report task results back to the controller. |
| argo-server | Go API Server (upstream only) | Provides REST/gRPC API and web UI for workflow management. Not shipped as a separate RHOAI image. |
| argo CLI | Go CLI (upstream only) | Command-line tool for interacting with Argo Workflows. Not shipped as a separate RHOAI image. |
| Workflow UI | React/TypeScript | Web-based UI for visualizing and managing workflows. Bundled into argo-server (upstream only). |
