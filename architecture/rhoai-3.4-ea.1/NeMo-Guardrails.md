## Purpose
**Short**: NeMo Guardrails is an open-source toolkit for adding programmable guardrails to LLM-based conversational systems.

**Detailed**: NeMo Guardrails, developed by NVIDIA, provides an intermediary layer between application code and LLMs that enforces programmable safety, security, and behavior controls (called "rails"). It supports input rails (filtering user messages), output rails (filtering LLM responses), dialog rails (steering conversation flow), and retrieval rails (controlling RAG behavior). The system uses a domain-specific language called Colang (v1.0 and v2.x) to define guardrail policies declaratively.

In the RHOAI distribution, NeMo Guardrails is shipped as a containerized server (built via Konflux) that exposes an OpenAI-compatible chat completions API (`/v1/chat/completions`) augmented with guardrail-specific extensions. The RHOAI build uses an "opensource" profile that excludes closed-source third-party guardrail integrations (e.g., ActiveFence, Cleanlab, Fiddler). The container ships with pre-downloaded NLP models (spaCy, sentence-transformers, FastEmbed, NLTK) for offline guardrail evaluation, and supports multi-architecture builds (amd64, arm64, ppc64le, s390x) with architecture-specific build stages for PyTorch, ONNX Runtime, and OpenBLAS on ppc64le.
