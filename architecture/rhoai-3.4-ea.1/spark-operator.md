## Purpose
**Short**: Kubernetes operator for managing Apache Spark application lifecycle on OpenShift/Kubernetes clusters.

**Detailed**: The Spark Operator (Kubeflow Spark Operator v2) automates the deployment, scaling, and lifecycle management of Apache Spark applications on Kubernetes. It provides Custom Resource Definitions (CRDs) for declaring Spark applications as Kubernetes-native resources, enabling users to submit, schedule, and monitor Spark jobs declaratively.

The operator consists of two main components: a controller that watches SparkApplication, ScheduledSparkApplication, and SparkConnect CRs to manage Spark driver/executor pods via spark-submit, and a webhook server that mutates and validates pods and SparkApplication resources. The controller handles the full application lifecycle including submission, monitoring, retry, and cleanup. It also supports optional Spark UI ingress, multiple batch schedulers (Volcano, YuniKorn, kube-scheduler), and Prometheus metrics.

For RHOAI, the operator is deployed in the `redhat-ods-applications` namespace via kustomize overlays, with images provided through the RHOAI operator's parameter substitution mechanism. The operator uses Apache Spark 4.0.1 as its base image.

## Architecture Components

| Component | Type | Purpose |
|-----------|------|---------|
| Controller Deployment | Go Operator (controller-runtime) | Watches SparkApplication/ScheduledSparkApplication/SparkConnect CRs; creates driver pods via spark-submit; manages Services and Ingresses for Spark UI |
| Webhook Deployment | Admission Webhook Server | Mutates pods (injects Spark config, volumes, env vars), validates SparkApplication/ScheduledSparkApplication resources, manages resource quotas |
| SparkApplication Controller | Reconciler | Manages full lifecycle state machine: New -> Submitted -> Running -> Succeeding/Failing -> Completed/Failed; creates driver pods, services, and ingresses |
| ScheduledSparkApplication Controller | Reconciler | Cron-based scheduling of SparkApplications with concurrency policies (Allow/Forbid/Replace) and run history management |
| SparkConnect Controller | Reconciler | Manages Spark Connect server pods and services; creates ConfigMaps for executor pod templates |
| Spark Driver ServiceAccount | RBAC | ServiceAccount used by Spark driver pods to manage executor pods, configmaps, services, and PVCs |
| Scheduler Registry | Internal | Pluggable batch scheduler framework supporting kube-scheduler, Volcano, and YuniKorn |
