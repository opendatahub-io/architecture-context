## Purpose
**Short**: A central repository for storing, managing, and serving ML model metadata including registered models, model versions, artifacts, inference services, serving environments, and experiments.

**Detailed**: Model Registry is a Kubeflow-originated service that provides a REST API (OpenAPI v1alpha3) for managing ML model lifecycle metadata. It serves as the single source of truth for model metadata across the RHOAI platform, enabling model developers and data scientists to register, version, and track models and their associated artifacts.

The component consists of several sub-components: (1) the core REST API proxy server that exposes an OpenAPI-based HTTP API backed by an embedded metadata datastore (EmbedMD) using either MySQL or PostgreSQL, (2) an InferenceService controller that watches KServe InferenceService CRs and synchronizes their state with the model registry, (3) a CSI storage initializer that enables KServe to pull model artifacts from the registry using `model-registry://` URIs, (4) an async upload job for copying models between storage backends, and (5) a Model Catalog service that provides a curated catalog API for browsing model sources and MCP servers.

The service integrates with Istio service mesh for mTLS communication and with KServe for inference service lifecycle management. It is built with FIPS compliance using Go's `strictfipsruntime` build tag and ships as a UBI9-based container image.

| InferenceService Controller | Go Controller (controller-runtime) | Watches KServe InferenceService CRs and syncs their state (create/update/delete) with Model Registry API; manages finalizers |
| CSI Storage Initializer | Go CLI Tool | KServe storage initializer for `model-registry://` URIs; downloads model artifacts from registry for inference |
| Model Catalog Server | Go REST API Service | Catalog API server (v1alpha1) providing curated model browsing, MCP server catalog, and performance metrics; uses PostgreSQL with leader election via pglock |
| Async Upload Job | Python K8s Job | Copies models between storage backends (e.g., HuggingFace, S3, OCI) and registers them in Model Registry |
| Model Registry UI | Web Application | Frontend for browsing and managing model registry entries |
| EmbedMD Datastore | Internal Go Library | Embedded metadata storage layer using GORM ORM; supports MySQL 8.3+ and PostgreSQL 16+ backends with schema migrations |
