When annotation odh.ray.io/secure-trusted-network: "true" is set:

  Head Pod NetworkPolicy ({cluster}-head):
  - Rule 1: Allow ALL ports from pods in same RayCluster (label match)
  - Rule 2: Allow ports 10001, 8265 from any pod in same namespace
  - Rule 3: Allow ports 8265, 10001 from KubeRay operator namespace
  - Rule 4a (opt): Allow port 8080 from monitoring namespace (MONITORING_NAMESPACE env)
  - Rule 4b (opt): Allow port 8265 from gateway namespace (GATEWAY_NAMESPACE env, default: openshift-ingress)
  - Rule 5: Allow port 8443 from anywhere (kube-rbac-proxy secured)

  Worker Pod NetworkPolicy ({cluster}-workers):
  - Allow ALL ports from pods in same RayCluster ONLY (strict isolation)
