Container Security:
- Runs as: non-root UID 1001
- Base Image: UBI9 (Red Hat Universal Base Image) with security updates
- Vulnerability Mitigation: dnf upgrade --refresh in every build layer
- Dependency Pinning: Hash-verified Python packages via pylock.toml with --require-hashes
- Build Isolation: Hermetic Konflux builds with prefetched dependencies
- CVE Constraints: dependencies/cve-constraints.txt for pinning security-critical versions

Authentication & Authorization:
- JupyterLab UI (8888/TCP): kube-rbac-proxy sidecar on 8443/TCP, Kubernetes RBAC
- RStudio Server UI (8787/TCP): kube-rbac-proxy sidecar on 8443/TCP, Kubernetes RBAC
- VS Code Server UI (8787/TCP): kube-rbac-proxy sidecar on 8443/TCP, Kubernetes RBAC
- Note: Auth is NOT handled within notebook images; kube-rbac-proxy sidecar is injected by controller

Secrets:
- rhel-subscription-secret (Opaque) - RHEL subscription for RStudio BuildConfig builds, Manual provisioning, No auto-rotation

Services (created dynamically by ODH Notebook Controller):
- notebook-{name}: ClusterIP, 8888/TCP, HTTP, internal only
- notebook-{name}-tls: ClusterIP, 8443/TCP, HTTPS (kube-rbac-proxy), internal only

Ingress (created dynamically by ODH Notebook Controller):
- notebook-{name}: HTTPRoute (Gateway API), 443/TCP, HTTPS, TLS 1.2+ SIMPLE mode
  Routes via data-science-gateway to kube-rbac-proxy sidecar
