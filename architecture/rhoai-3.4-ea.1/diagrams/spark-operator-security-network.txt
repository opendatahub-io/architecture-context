Notes
================================================================================

- No NetworkPolicy resources defined
- No Gateway API / HTTPRoute usage
- No oauth-proxy or kube-rbac-proxy usage
- No service mesh (Istio/OSSM) integration
- Webhook TLS certs are self-signed and self-managed by the webhook controller
- Metrics endpoints (8080/TCP) are plaintext HTTP with no authentication
- SparkConnect gRPC endpoint (15002/TCP) is plaintext with no authentication
- Spark Web UI Ingress TLS is optional and configurable per SparkApplication
