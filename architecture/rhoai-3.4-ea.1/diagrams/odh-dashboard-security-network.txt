Authentication Flow:
  External Request
    → Gateway (TLS termination, 443/TCP)
    → HTTPRoute
    → kube-rbac-proxy (8443, re-encrypts, validates via TokenReview + SubjectAccessReview)
    → dashboard app (8080, plaintext)
  User identity forwarded via X-Auth-Request-User and X-Auth-Request-Groups headers
