Authentication & Authorization:

- All endpoints: plaintext HTTP, no auth

Custom Detector Security (Built-in Detector):
- Static code analysis on user-provided Python detector files
- Forbidden imports: os (except os.environ/os.getenv), subprocess, sys, shutil
- Forbidden calls: eval, exec, open, compile, input
- Violation triggers ImportError and module rejection
