Network Policies:
- Allow ingress to TrustyAI Service (8080/TCP) from OAuth Proxy
- Allow ingress to EvalHub Service (8000/TCP) from authorized clients
- Allow egress to PostgreSQL (5432/TCP) from TrustyAI/EvalHub pods
- Allow egress to KServe pods (8080/TCP) from TrustyAI Service
- Allow egress to HuggingFace Hub (443/TCP) from Evaluation Job pods
- Allow egress to S3 Storage (443/TCP) from Evaluation Job pods
- Allow egress to MLFlow Server (5000/TCP) from Evaluation Job pods
- Allow ingress to Operator metrics (8443/TCP) from Prometheus
