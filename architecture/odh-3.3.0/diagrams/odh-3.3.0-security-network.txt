Key Ports & Protocols
=====================
- External Ingress: 80/TCP (HTTP redirect), 443/TCP (HTTPS TLS 1.2+)
- Kubernetes API: 6443/TCP (HTTPS TLS 1.2+)
- S3 Storage: 443/TCP (HTTPS TLS 1.2+)
- PyPI/HuggingFace: 443/TCP (HTTPS TLS 1.2+)
- Container Registry: 443/TCP (HTTPS TLS 1.2+)
- Service Mesh: mTLS (PERMISSIVE/STRICT)
- Operator Webhooks: 9443/TCP (HTTPS TLS 1.2+)
