## Purpose

**Short**: A Python NLP library built on the Caikit framework providing text generation, embedding, reranking, and classification capabilities for model serving in RHOAI.

**Detailed**: Caikit-NLP is a Python library that implements NLP "tasks" and "modules" using the Caikit AI framework. It provides capabilities for text generation (via prompt tuning, fine-tuning, and TGIS remote inference), text embedding (via sentence-transformers), cross-encoder reranking, text/token classification, and tokenization. The library serves as the NLP domain plugin for the Caikit runtime, which exposes models through both HTTP (port 8080) and gRPC (port 8085) interfaces.

In RHOAI, caikit-nlp is packaged as a container image and deployed as an inference runtime within the model serving infrastructure. It integrates with TGIS (Text Generation Inference Server) as a backend for large language model inference, allowing prompt-tuned and fine-tuned models to be served efficiently. The library supports HuggingFace model loading (AutoCausalLM, AutoSeq2SeqLM), PEFT-based prompt tuning and multitask prompt tuning, and local sentence-transformer embedding models with optimizations including PyTorch compile, IPEX, and autocast.
