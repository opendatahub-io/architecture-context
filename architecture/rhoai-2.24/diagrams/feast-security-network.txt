  - Verbs: create, delete, get, list, update, watch, patch (varies by resource)

- ClusterRole: feast-operator-metrics-auth-role
  - API Groups: authentication.k8s.io, authorization.k8s.io
  - Resources: tokenreviews, subjectaccessreviews
  - Verbs: create

- Dynamic Roles (per FeatureStore CR):
  - feast-{name}: RBAC roles/rolebindings read (for Kubernetes authz)
  - feast-{name}-cronjob: deployments get, pods list, pods/exec create

Role Bindings:
- feast-operator-manager-rolebinding (Cluster-scoped)
  → ClusterRole: feast-operator-manager-role
  → ServiceAccount: feast-operator-controller-manager

- feast-operator-leader-election-rolebinding (Operator namespace)
  → Role: feast-operator-leader-election-role
  → ServiceAccount: feast-operator-controller-manager

Authentication & Authorization:
- Online/Offline/Registry servers:
  - Methods: All
  - Auth: Kubernetes RBAC or OIDC (configurable via FeatureStore CR `authz` field)
  - Enforcement: Feast server (application-level)
- Operator metrics:
  - Methods: GET
  - Auth: TokenReview + SubjectAccessReview
  - Enforcement: controller-runtime metrics filter

Secrets:
- feast-{name}-{service}-tls (kubernetes.io/tls)
  - Purpose: TLS certificates for feast services
  - Provisioned by: OpenShift service serving cert signer (auto)
  - Auto-rotate: Yes (OpenShift managed)

- User-referenced secrets (DB persistence) (Opaque)
  - Purpose: Database connection parameters for online/offline/registry stores
  - Provisioned by: User
  - Auto-rotate: No

- User-referenced secrets (OIDC) (Opaque)
  - Purpose: OIDC client credentials (client_id, client_secret,
    auth_discovery_url, username, password)
  - Provisioned by: User
  - Auto-rotate: No

TLS Configuration:
- OpenShift: Automatic via service serving certificates
  - Annotation: service.beta.openshift.io/serving-cert-secret-name
  - CA injection: service.beta.openshift.io/inject-cabundle on ConfigMap
  - Custom CA: odh-trusted-ca-bundle mounted at /etc/pki/tls/custom-certs/ca-bundle.crt
- Non-OpenShift: Plain HTTP/gRPC (no automatic TLS)
