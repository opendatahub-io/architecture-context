Security Notes:
================================================================================

- Inter-node Ray communication (GCS 6379/TCP, client 10001/TCP) uses plaintext
- Inter-node PyTorch c10d communication (29500/TCP) uses plaintext
- Ray Dashboard internal traffic (8265/TCP) is plaintext
- All external egress uses HTTPS/TLS 1.2+
- Kubernetes API access uses TLS 1.2+ with ServiceAccount tokens
- Ray Dashboard exposed via edge TLS Route (TLS terminated at OpenShift router)
- No service mesh (mTLS) configured for distributed workload pods
