RBAC Summary:
- This component does NOT define RBAC resources
- RBAC for notebook pods is managed by:
  - ODH Notebook Controller (creates ServiceAccounts, injects auth sidecars)
  - User's OpenShift RBAC permissions (namespace access)
- Authentication enforcement:
  - kube-rbac-proxy (RHOAI 3.x): Validates bearer tokens against K8s RBAC
  - oauth-proxy (RHOAI 2.x): Validates OAuth tokens against OpenShift OAuth server

Secrets:
- rhel-subscription-secret (Opaque)
  - Purpose: RHEL subscription for RStudio BuildConfig
  - Provisioned By: Manual / Cluster Admin
  - Auto-Rotate: No

Container Security:
- Non-root: UID 1001
- FIPS-compliant: Yes (data science images, mongocli built with -tags strictfipsruntime)
- Base image hardening: UBI9 with dnf upgrade --refresh at build time
- Vulnerability mitigation: GPG pubkey RPMs removed, OS packages minimized
- Multi-arch: amd64, ppc64le, s390x with conditional build steps

- N (2025.1): Current recommended
