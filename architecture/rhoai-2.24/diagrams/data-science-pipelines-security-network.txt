Authentication & Authorization:
==============================================================================

- API Server (/apis/*):
  Auth: Kubeflow UserID Header (multi-user mode)
  Enforcement: API Server interceptor
  Policy: SubjectAccessReview against K8s RBAC

- Webhooks (/webhooks/*):
  Auth: Kubernetes API Server TLS client cert
  Enforcement: Kubernetes admission controller

- Persistence Agent → API Server:
  Auth: Projected ServiceAccount Token
  Audience: pipelines.kubeflow.org
  Expiry: 1 hour

- Scheduled Workflow → API Server:
  Auth: Projected ServiceAccount Token
  Audience: pipelines.kubeflow.org
  Expiry: 1 hour
