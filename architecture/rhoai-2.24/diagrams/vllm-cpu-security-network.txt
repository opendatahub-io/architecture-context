RBAC Summary:
- No RBAC resources defined by this component
- Runs as non-root user (UID 2000, GID 0) within container
- Managed by KServe/ModelMesh (inherits their RBAC)

Container Security:
- User ID: 2000 (non-root)
- Group ID: 0 (root group, OpenShift arbitrary UID support)
- HF_HUB_OFFLINE=1 (prevents runtime network calls to HuggingFace)
- VLLM_NO_USAGE_STATS=1 (disables telemetry/usage statistics)
- Base Image: UBI 9 Minimal (9.5-9.6)

Secrets:
- SSL key/cert (kubernetes.io/tls) - Optional TLS termination, cert-manager, hot-reload via --enable-ssl-refresh
- API Key (Opaque) - Optional Bearer token auth for API endpoints, no auto-rotation
- HuggingFace Token (Opaque) - Optional model download auth, no auto-rotation

Authentication & Authorization:
- /v1/* endpoints: API Key Bearer Token (optional, rejects 401 if set but not provided)
- /health, /ping, /metrics, /version: No auth (always accessible)
- gRPC TGIS (port 8033): No built-in authentication

Network Services:
- vllm-openai: ClusterIP, 8000/TCP, HTTP, Optional TLS, API Key, Internal
- vllm-grpc (TGIS): ClusterIP, 8033/TCP, gRPC, plaintext, None, Internal
