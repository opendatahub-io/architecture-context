1. Allow ingress to InferenceService pods from Istio IngressGateway:
   - From: istio-system/istio-ingressgateway
   - To: <user-namespace>/<isvc-name>-predictor
   - Ports: 8000/TCP

2. Allow egress from InferenceService pods to external storage:
   - From: <user-namespace>/<isvc-name>-predictor (storage-initializer)
   - To: Internet (S3/GCS/Azure endpoints)
   - Ports: 443/TCP

3. Allow egress from InferenceService pods to Kubernetes API:
   - From: <user-namespace>/<isvc-name>-predictor
   - To: default/kubernetes
   - Ports: 6443/TCP

4. Allow ingress to webhook server from Kubernetes API Server:
   - From: Kubernetes API Server IP range
   - To: kserve/kserve-webhook-server-service
   - Ports: 443/TCP

5. Allow egress from controller to Kubernetes API Server:
   - From: kserve/kserve-controller-manager
   - To: default/kubernetes
   - Ports: 6443/TCP
