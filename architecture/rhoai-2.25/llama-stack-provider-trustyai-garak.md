## Purpose
**Short**: An out-of-tree Llama Stack eval provider that integrates NVIDIA Garak for automated red-teaming and security testing of Large Language Models.

**Detailed**: This component implements the Llama Stack `Eval` API as an out-of-tree provider, wrapping the NVIDIA Garak red-teaming framework to provide automated security testing for LLMs deployed in RHOAI. It detects vulnerabilities including prompt injection, jailbreaks, toxicity, and bias through standardized compliance frameworks (OWASP LLM Top 10, AVID taxonomy).

The provider supports two execution modes: an **inline mode** where Garak scans run as local subprocess invocations within the Llama Stack server process, and a **remote mode** where scans are offloaded to Kubeflow Pipelines (KFP) for scalable Kubernetes-based execution. Both modes integrate with the Llama Stack Files API for scan artifact storage (reports in JSON, HTML, and log formats) and optionally with the Safety/Shields APIs for testing LLM behavior with and without guardrails.

The shield scanning subsystem (`SimpleShieldOrchestrator`) allows probes to be routed through input/output shields before/after LLM inference, enabling comparative security testing with guardrails like Prompt-Guard-86M and Llama-Guard. Scan results are scored on a 0.0-1.0 vulnerability scale with configurable thresholds.
