## Purpose
**Short**: Builds the GPU-accelerated vLLM inference runtime container image with NVIDIA CUDA and TGIS (Text Generation Inference Service) adapter for RHOAI.

**Detailed**: This repository is a Konflux build-definition repository that produces the `odh-vllm-cuda-rhel9` container image for Red Hat OpenShift AI (RHOAI). It does not contain application source code; instead, it wraps the upstream Red Hat AI Inference Server (RHAIIS) vLLM CUDA image (`registry.redhat.io/rhaiis/vllm-cuda-rhel9`) with configuration for the TGIS gRPC adapter (`vllm_tgis_adapter`).

The resulting container image provides a high-performance LLM inference server that exposes both an OpenAI-compatible HTTP API (port 8000) and a TGIS-compatible gRPC API (port 8033). It is designed to run on NVIDIA GPU-equipped nodes within OpenShift and is deployed by KServe or ModelMesh as an inference runtime for serving large language models. The image disables logprobs during speculative decoding by default to provide predictable behavior for hosted model endpoints.
