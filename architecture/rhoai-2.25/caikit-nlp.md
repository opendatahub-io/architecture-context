## Purpose

**Short**: Caikit-NLP is a Python library providing Natural Language Processing (NLP) capabilities — including text generation, prompt tuning, text embedding, reranking, and classification — built on top of the Caikit AI framework.

**Detailed**: Caikit-NLP extends the Caikit framework with NLP-specific modules that implement standardized "tasks" (TextGenerationTask, EmbeddingTask, RerankTask, etc.) and provides both local inference and remote TGIS (Text Generation Inference Server) backend support. It packages Hugging Face transformer models into a Caikit-compatible format, enabling prompt tuning (PEFT), fine-tuning, text embedding via sentence-transformers, cross-encoder reranking, sequence classification, and tokenization.

When deployed as a container image (odh-caikit-nlp-rhel9), the library is loaded by the Caikit runtime (`caikit.runtime`) which exposes both gRPC and HTTP serving endpoints. The container integrates into the RHOAI model serving stack, typically fronted by KServe or ModelMesh, and communicates with a remote TGIS backend over gRPC for large language model inference. The library itself does not define Kubernetes CRDs, controllers, or operator logic — it is a pure runtime library consumed by the model serving infrastructure.
