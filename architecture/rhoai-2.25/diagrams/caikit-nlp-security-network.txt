Authentication & Authorization:
- HTTP API (8080): Auth delegated to KServe/ModelMesh/Istio sidecar proxy
- gRPC API (8085): Auth delegated to KServe/ModelMesh/Istio sidecar proxy
- TGIS backend:    mTLS certificate-based authentication (optional)

- RUNTIME_TLS_SERVER_CERT  (file path via env var) - TLS certificate for serving endpoints
  Provisioned by: Platform / cert-manager | Auto-Rotate: No

- ca_cert_file             (config file path) - CA cert for TGIS backend

- client_key_file          (config file path) - Client key for TGIS mTLS
  Provisioned by: Platform / manual | Auto-Rotate: No
