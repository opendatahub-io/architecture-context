NETWORK POLICIES:
================================================================================

1. modelmesh-controller
   - Pod Selector: control-plane: modelmesh-controller
   - Ingress: Port 8443/TCP from any (metrics)

2. modelmesh-webhook
   - Pod Selector: control-plane: modelmesh-controller
   - Ingress: Port 9443/TCP from any (webhook from API server)

3. modelmesh-runtimes
   - Pod Selector: modelmesh-service label exists
   - Ingress:
     - Port 8033/TCP, 8080/TCP from modelmesh-controller pods (inter-pod coordination)
     - Port 8033/TCP, 8008/TCP from any (inference access)
     - Port 2112/TCP from any (Prometheus scraping)

4. etcd
   - Pod Selector: component: model-mesh-etcd
   - Ingress: Port 2379/TCP from modelmesh-enabled namespaces and same namespace

RBAC SUMMARY:
================================================================================

ClusterRoles:
- modelmesh-controller-role
  - serving.kserve.io: inferenceservices, predictors, servingruntimes,
    clusterservingruntimes (full CRUD + finalizers + status)
  - core: configmaps, secrets, services (full CRUD)
  - core: namespaces (get, list, patch, update, watch)
  - apps: deployments (full CRUD + finalizers)
  - monitoring.coreos.com: servicemonitors (full CRUD)
  - autoscaling: horizontalpodautoscalers (get, list, watch, create, delete, update)

- proxy-role
  - authentication.k8s.io: tokenreviews (create)
  - authorization.k8s.io: subjectaccessreviews (create)

- model-serving-edit (user-facing)
  - serving.kserve.io: inferenceservices, servingruntimes (full CRUD)

- model-serving-view (user-facing)
  - kubeflow.org: servingruntimes, inferenceservices (read-only)

- etcd-role
  - core: secrets (full CRUD)

Namespace Roles:
- modelmesh-controller-leader-election-role
  - coordination.k8s.io: leases (create, get, list, update)
  - core: configmaps (full CRUD), events (create)

- modelmesh-controller-restricted-scc-role
  - security.openshift.io: securitycontextconstraints/restricted (use)

Service Accounts:
- modelmesh-controller (controller manager pod)
- modelmesh-serving-sa (runtime inference pods)
- etcd-serviceaccount (etcd pod, secret creation)
