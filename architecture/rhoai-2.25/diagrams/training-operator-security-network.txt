Security Notes
================================================================================

- No external ingress (Routes, HTTPRoutes, Gateways) - cluster-internal only
- Webhook TLS: self-signed CA managed by open-policy-agent/cert-controller
- Istio sidecar explicitly disabled on operator pod
- Container runs as non-root user (UID 65532)
- FIPS mode enabled via GOEXPERIMENT=strictfipsruntime
- NetworkPolicy created per PyTorchJob for pod isolation
- Metrics endpoint (8080/TCP) is plaintext HTTP with no authentication
- Only egress: Kubernetes API server (443/TCP HTTPS)
