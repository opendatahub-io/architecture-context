Security Considerations:
═══════════════════════════════════════════════════════════════════════════
1. FIPS Compliance: Built with GOEXPERIMENT=strictfipsruntime for FIPS 140-2
2. Minimal RBAC: Separate roles for webhook cert management (namespace-scoped)
3. Network Isolation: Optional NetworkPolicies per training job
4. No Service Mesh mTLS: Explicitly disabled for training pods
5. Framework-level Security: Training communication security is framework-specific
6. Certificate Rotation: Automated webhook TLS cert management
7. Image Security: Training images pulled from trusted registries with pull secrets
8. Recent Security Improvements:
   - Secrets RBAC restricted from ClusterRole to namespace-scoped Role (CVE mitigation)
   - Updated Go toolset to 1.25 for CVE-2025-61726 fix
   - Regular base image updates for security patches
