Authentication & Authorization:
================================================================================

  - Model Registry REST API: OpenShift OAuth + SubjectAccessReview
    - Policy: User must have `get` on `services/{name}` in registry namespace
  - Model Catalog REST API: OpenShift OAuth + SubjectAccessReview
    - Policy: User must have `get` on `services/model-catalog` in catalog namespace
  - Operator Metrics (8443/TCP): Bearer Token + TokenReview + SubjectAccessReview
  - Webhooks (9443/TCP): k8s API server mutual TLS

  - {registry-name}-oauth-cookie-secret (Opaque)
