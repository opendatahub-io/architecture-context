Security Notes:
- API Server uses InsecureSkipVerify=true for internal gRPC-Gateway loopback (HTTP→gRPC translation, localhost only)
- Cache client explicitly sets InsecureSkipVerify=false
- FIPS: All binaries built with GOEXPERIMENT=strictfipsruntime, CGO_ENABLED=1 (except launcher has dual binary)
- Ingress (Routes/HTTPRoutes) managed externally by RHOAI Operator — not defined in this component
- No PeerAuthentication or AuthorizationPolicy defined in component (service mesh config managed by operator)
- Multi-user mode: Bearer token validated via K8s TokenReview, authorization via SubjectAccessReview per namespace
