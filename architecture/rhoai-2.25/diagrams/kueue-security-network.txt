RBAC Summary:
- ClusterRole: manager-role
  - Core: events (create,patch,update,watch), pods (delete,get,list,patch,update,watch),
    secrets (get,list,update,watch), limitranges/namespaces/nodes (get,list,watch)
  - Kueue CRDs: admissionchecks, clusterqueues, cohorts, localqueues, workloads
    (create,delete,get,list,patch,update,watch)
  - Job frameworks: batch/jobs, kubeflow.org/*, ray.io/*, jobset.x-k8s.io/*,
    workload.codeflare.dev/*, leaderworkerset.x-k8s.io/* (get,list,patch,update,watch)
  - Cluster resources: apps/deployments,replicasets,statefulsets (get,list,watch),
    scheduling.k8s.io/priorityclasses (get,list,watch)
  - Webhooks: admissionregistration.k8s.io (get,list,update,watch)

- ClusterRole: metrics-auth-role
  - authentication.k8s.io/tokenreviews (create)
  - authorization.k8s.io/subjectaccessreviews (create)

- ClusterRole: metrics-reader
  - Non-resource URL: /metrics (get)

Role Bindings:
- manager-rolebinding → controller-manager SA (cluster-scoped)
- leader-election-rolebinding → controller-manager SA (namespace-scoped, leader election)
- metrics-auth-rolebinding → controller-manager SA (cluster-scoped)
- controller-manager-read-metrics → controller-manager-metrics-reader SA (RHOAI, cluster-scoped)

Secrets:
- kueue-webhook-server-cert (kubernetes.io/tls)
  - Purpose: Webhook and metrics TLS certificates
  - Provisioned: Internal cert mgmt (self-signed rotation) or cert-manager
  - Rotation: Yes (internal) / Depends (cert-manager)

- controller-manager-metrics-token (kubernetes.io/service-account-token) [RHOAI]
  - Purpose: Bearer token for Prometheus metrics scraping
  - Provisioned: Kubernetes SA token controller
  - Rotation: Yes (automatic)

- MultiKueue kubeconfig secrets (Opaque)
  - Purpose: Remote cluster kubeconfig for MultiKueue
  - Provisioned: Manual
  - Rotation: No (manual management)

RHOAI-Specific Hardening:
- NetworkPolicy: Restricts webhook ingress to TCP/9443 only
- PrometheusRule alerts:
  - KueuePodDown (critical) - Pod liveness
  - LowClusterQueueResourceUsage (info) - Resource utilization
  - ResourceReservationExceedsQuota (info) - Over-reservation
  - PendingWorkloadPods (info) - Long-pending workloads
- Webhook namespace exclusions: kube-system, openshift-* system namespaces
- Feature gates: LocalQueueMetrics=true, VisibilityOnDemand=false, MultiKueue=false

FIPS Compliance:
- Build: GOEXPERIMENT=strictfipsruntime, CGO_ENABLED=1, -tags strictfipsruntime
- Linking: Dynamic (CGO_ENABLED=1, uses OpenSSL via UBI9)
- TLS: crypto/tls with certwatcher (inherits Go/OpenSSL FIPS defaults)
- No custom cipher suites or MinVersion set
- No InsecureSkipVerify in production code (test-only)
