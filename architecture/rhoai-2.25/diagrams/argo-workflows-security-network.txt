- No NetworkPolicies included in manifests
- No Ingress configured (controller is internal-only)
- Metrics (9090/TCP) and health (6060/TCP) endpoints are unauthenticated HTTP
- Network isolation relies on namespace/cluster-level controls
- No service mesh / mTLS configuration in manifests

FIPS Compliance Notes:
- workflow-controller: FIPS-only binary (CGO_ENABLED=1, strictfipsruntime)
- argoexec: DUAL binary - FIPS + non-FIPS, ENTRYPOINT defaults to non-FIPS
  * DSP operator must configure FIPS binary selection at deployment time
- InsecureSkipVerify used for SSO OIDC (argo-server only, not shipped)
