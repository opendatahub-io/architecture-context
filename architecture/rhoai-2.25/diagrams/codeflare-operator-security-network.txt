Trust Boundaries:
1. External → Ingress: OAuth/TLS termination at Route
2. Ingress → User Namespace: Pod-to-pod (no encryption by default)
3. Operator → K8s API: ServiceAccount Bearer Token (TLS 1.3)
4. K8s API → Webhooks: mTLS (mutual authentication)
5. User Namespace: NetworkPolicies enforce pod-level isolation
