================================================================================
Authentication & Authorization
================================================================================

- Inference endpoints (RHOAI 3.x):
  Auth: kube-rbac-proxy (TokenReview/SubjectAccessReview)
  Enforcement: kube-rbac-proxy sidecar container
  Policy: RBAC-based, annotation: security.opendatahub.io/enable-auth

- Webhook endpoints:
  Auth: Kubernetes API server TLS client cert
  Enforcement: Kubernetes API admission
  Policy: API server admission control

- LLM-d scheduler gRPC:
  Auth: mTLS (self-signed certs)
  Enforcement: TLS in ext_proc
  Policy: Certificate-based

- LLM-d routing sidecar:
  Auth: TLS (self-signed certs, --secure-proxy=true)
  Enforcement: Sidecar proxy
  Policy: Certificate-based
