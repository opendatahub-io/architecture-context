RBAC Summary:
=============

ClusterRoles:
- argo-cluster-role (bound to SA: argo)
  - pods, pods/exec: create, get, list, watch, update, patch, delete
  - configmaps: get, watch, list
  - persistentvolumeclaims: create, update, delete, get
  - argoproj.io workflows, workflowtasksets, workflowartifactgctasks: get, list, watch, update, patch, delete, create
  - argoproj.io workflowtemplates, clusterworkflowtemplates: get, list, watch
  - argoproj.io workflowtaskresults: list, watch, deletecollection
  - argoproj.io cronworkflows: get, list, watch, update, patch, delete
  - serviceaccounts: get, list
  - events: create, patch
  - poddisruptionbudgets: create, get, delete
  - secrets (argo-workflows-agent-ca-certificates): get

- argo-server-cluster-role (bound to SA: argo-server)
  - configmaps: get, watch, list
  - secrets: get, create
  - pods, pods/exec, pods/log: get, list, watch, delete
  - events: watch, create, patch
  - serviceaccounts: get, list, watch
  - argoproj.io (all workflow CRDs): create, get, list, watch, update, patch, delete

- argo-aggregate-to-view (label aggregation)
  - argoproj.io resources: get, list, watch

- argo-aggregate-to-edit (label aggregation)
  - argoproj.io resources: create, delete, deletecollection, get, list, patch, update, watch

- argo-aggregate-to-admin (label aggregation)
  - argoproj.io resources (incl. workflowtasksets): create, delete, deletecollection, get, list, patch, update, watch

Namespace Roles:
- argo-role (bound to SA: argo in install namespace)
  - coordination.k8s.io leases: create, get, update
  - secrets: get

Authentication & Authorization:
================================
- /api/* endpoints: Bearer Token (K8s SA), SSO (OIDC JWT), or Server mode (cluster SA)
- Enforcement: Gatekeeper interceptor (HTTP) / gRPC interceptor chain
- /metrics: Bearer Token (configurable, can disable via ARGO_SERVER_METRICS_AUTH=false)
- gRPC services: Same auth as HTTP API

Secrets:
========
- argo-workflows-agent-ca-certificates (Opaque) - CA certs for agent communication, Admin provisioned, No auto-rotate
- argo-server-sso (Opaque, optional) - SSO client credentials for OIDC, Admin provisioned, No auto-rotate
- (database credentials) (Opaque) - PostgreSQL/MySQL connection credentials, Admin provisioned, No auto-rotate
