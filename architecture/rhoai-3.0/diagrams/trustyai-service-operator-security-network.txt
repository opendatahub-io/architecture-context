Authentication & Authorization:
===============================
- TrustyAI Service API (8443/TCP): kube-rbac-proxy SubjectAccessReview
- GuardrailsOrchestrator API (8432/TCP): kube-rbac-proxy SubjectAccessReview
  (when security.opendatahub.io/enable-auth: true annotation set)
- GuardrailsOrchestrator Gateway (8490/TCP): kube-rbac-proxy SubjectAccessReview
  (when OAuth enabled and gateway enabled)

- DestinationRule: mTLS for TrustyAI service
- VirtualService: Traffic routing for TrustyAI service
