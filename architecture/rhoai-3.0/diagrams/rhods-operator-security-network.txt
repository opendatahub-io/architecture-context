Authentication Flow (RHOAI 3.x Gateway API)
================================================================================

  Request → data-science-gateway (Envoy, 443/TCP)
    → EnvoyFilter ext_authz → kube-auth-proxy (8443/TCP)
      → OpenShift OAuth / OIDC Provider (443/TCP)
    ← Auth result
    → Lua filter: inject Bearer token header
    → Backend service (via HTTPRoute)

  Unauthenticated: Redirect to OAuth login page
  Authenticated: Forward with Bearer token to backend
