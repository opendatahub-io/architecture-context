OPERATOR ──► MANAGED SERVER COMMUNICATION:
  - Port: 8321/TCP
  - Protocol: HTTP
  - Encryption: plaintext
  - Auth: None
  - Purpose: Health check (/v1/health), provider info (/v1/providers),
             version queries (/v1/version)
  - Note: Internal cluster communication only (ClusterIP service)

NO EXTERNAL INGRESS:
  - No Ingress, Route, or HTTPRoute resources are created
  - Managed Llama Stack servers are ClusterIP-only
  - Accessible only within the cluster
