  Enforcement: Kubeflow Pipelines server

- S3 Storage
  Methods: PUT/GET
  Auth: AWS IAM credentials (mounted as env vars in KFP pods)
  Enforcement: S3/MinIO server (IAM policy on bucket)

- Llama Stack Server
  Methods: POST/GET
  Auth: None (provider runs in-process or KFP pods call back)
  Enforcement: Application level (Llama Stack server configuration)
