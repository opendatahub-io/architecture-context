NETWORK POLICIES (Recommended)
═══════════════════════════════════════════════════════════════════════════

Ingress to Notebook Controller:
  - Allow from: Prometheus (for metrics scraping)
    - Port: 8080/TCP or 8443/TCP
  - Allow from: Kubernetes API Server (for health checks)
    - Ports: 8081/TCP

Egress from Notebook Controller:
  - Allow to: Kubernetes API Server
    - Port: 6443/TCP
  - Allow to: Notebook Pods (for culling)
    - Port: 8888/TCP

Ingress to Notebook Pods:
  - Allow from: Notebook Controller (for culling)
    - Port: 8888/TCP
  - Allow from: Istio Gateway (for user access)
    - Port: 8888/TCP
  - Allow from: Users (via Istio VirtualService)
    - Port: 80/TCP (via Service)

Egress from Notebook Pods:
  - Allow to: Internet (for package installation, model download)
  - Allow to: Internal services (databases, storage, etc.)
