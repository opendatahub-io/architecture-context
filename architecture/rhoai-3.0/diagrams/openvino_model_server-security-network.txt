RBAC Summary:
================================================================
OVMS does not define Kubernetes RBAC resources.
It runs as a regular container within pods created by KServe or ModelMesh controllers.
Platform-level RBAC is enforced by kube-rbac-proxy sidecar (RHOAI 3.x)
or oauth-proxy sidecar (RHOAI 2.x).

Container Security:
================================================================
- Run-as User: ovms (UID 5000)
- Run-as Group: ovms (GID 5000)
- Additional Groups: 39 (video), 44 (video1) for GPU device access
- Base Image: registry.access.redhat.com/ubi9-minimal:9.6
- FIPS Compliant: Yes
- No TLS implementation (plaintext internally)
- TLS termination at platform ingress layer

Secrets:
================================================================
- AWS_ACCESS_KEY_ID (env var) - S3 auth - User/Platform provisioned - No auto-rotate
- AWS_SECRET_ACCESS_KEY (env var) - S3 auth - User/Platform provisioned - No auto-rotate
- AWS_REGION (env var) - S3 region - User/Platform provisioned - No auto-rotate
- GOOGLE_APPLICATION_CREDENTIALS (env var/file) - GCS auth - User/Platform provisioned - No auto-rotate
- AZURE_STORAGE_CONNECTION_STRING (env var) - Azure auth - User/Platform provisioned - No auto-rotate
- HF_TOKEN (env var) - HuggingFace auth (optional) - User/Platform provisioned - No auto-rotate

Authentication & Authorization:
================================================================
- OVMS itself: No auth/TLS implementation (intentional design)
- RHOAI 3.x: kube-rbac-proxy sidecar enforces Bearer Token + Kubernetes RBAC
- RHOAI 2.x: oauth-proxy sidecar enforces OAuth 2.0
- Security boundary: Platform ingress layer, not OVMS container

Network Security Notes:
================================================================
- OVMS binds to 0.0.0.0 by default (all interfaces)
- gRPC uses InsecureServerCredentials (no TLS)
- REST uses plaintext HTTP (no TLS)
- All external communication (S3/GCS/Azure/HF) uses HTTPS TLS 1.2+
- Internal pod communication is plaintext (security relies on pod isolation + sidecar)
