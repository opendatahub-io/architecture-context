Notes:

1. Metrics endpoint currently uses HTTP (port 8080) - migration to HTTPS (8443) is planned
2. Model Registry TLS verification is configurable via MR_SKIP_TLS_VERIFY
3. Webhook certificates are auto-rotated by OpenShift Service CA
4. Controller uses label-based caching to reduce memory footprint:
   - Secrets: opendatahub.io/managed=true
   - Pods: component=predictor
5. NIM functionality can be completely disabled via NIM_STATE=removed
6. Model Registry integration is optional via MODELREGISTRY_STATE
