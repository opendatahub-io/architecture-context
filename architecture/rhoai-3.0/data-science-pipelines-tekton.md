## Purpose
**Short**: Kubeflow Pipelines backend on Tekton, providing ML pipeline orchestration, scheduling, and metadata tracking for RHOAI.

**Detailed**: Data Science Pipelines Tekton (DSP Tekton) is a fork of Kubeflow Pipelines that replaces the Argo Workflows engine with Tekton Pipelines as the workflow execution backend. It provides a complete ML pipeline platform including an API server for pipeline CRUD operations, a scheduler for recurring workflows, a persistence agent for tracking pipeline run status, a cache server for task-level caching, ML metadata services (MLMD) for artifact and experiment tracking, and a web UI for pipeline visualization and management.

The component is designed to run on OpenShift as part of the RHOAI (Red Hat OpenShift AI) platform. It uses Tekton CRDs (PipelineRun, TaskRun) along with custom Tekton extensions (PipelineLoop) to execute ML workflows. The system integrates with object storage (S3-compatible/Minio) for artifact persistence and a relational database (MySQL/MariaDB/PostgreSQL) for metadata and pipeline state storage. The OpenDataHub overlay manifests provide RHOAI-specific configuration including OAuth proxy authentication for the UI and OpenShift Route ingress.
