## Purpose
**Short**: A lightweight Rust HTTP server that detects PII and custom patterns in text using regex matching, serving as a detector backend for the FMS Guardrails Orchestrator.

**Detailed**: The Guardrails Regex Detector is a purpose-built microservice written in Rust that provides text content scanning via regular expression matching. It exposes an HTTP API that accepts text content and regex patterns, returning structured detection results with position information and confidence scores.

The service includes built-in detectors for common PII patterns including email addresses, Social Security numbers (SSNs), and credit card numbers. Additionally, it supports arbitrary custom regex patterns provided at request time. Each detection result includes the matched text, start/end character positions, detection type classification, and a score (always 1.0 for regex matches).

This component is designed to integrate with the [FMS Guardrails Orchestrator](https://github.com/foundation-model-stack/fms-guardrails-orchestrator), which routes content through various detector services to enforce guardrails on AI model inputs and outputs. The regex detector handles pattern-based content detection as one of potentially many detector backends.

### API Request/Response Schema

**POST `/api/v1/text/contents`** - Request:
```json
{
  "contents": ["text strings to scan"],
  "detector_params": {
    "regex": ["email", "ssn", "credit-card", "^custom-regex$"]
  }
}
```

**Built-in regex identifiers**:
| Identifier | Detection Type | Detection Name | Pattern |
|------------|----------------|----------------|---------|
| `email` | pii | EmailAddress | `[A-Za-z0-9._%+-]+@[A-Za-z0-9.-]+\.[A-Za-z]{2,}` |
| `ssn` | pii | SocialSecurity | SSN variants (XXX-XX-XXXX, etc.) |
| `credit-card` | pii | CreditCard | Major card number patterns (Visa, MC, Amex, etc.) |

Custom regex strings (not matching built-in identifiers) are treated as arbitrary regex patterns with `detection_type: "custom"` and `detection: "custom"`.

**Response**: Array of detection arrays:
```json
[[{
  "start": 0,
  "end": 16,
  "text": "matched text",
  "detection_type": "pii",
  "detection": "EmailAddress",
  "score": 1.0
}]]
```
