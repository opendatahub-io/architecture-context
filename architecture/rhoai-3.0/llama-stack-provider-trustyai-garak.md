## Purpose
**Short**: An out-of-tree Llama Stack provider that integrates NVIDIA's Garak red-teaming framework for automated security testing and vulnerability assessment of Large Language Models.

**Detailed**: The TrustyAI Garak provider (`trustyai_garak`) implements Llama Stack's Eval API to enable automated security testing and red teaming of LLMs. It wraps NVIDIA's Garak tool as a Llama Stack provider, enabling security scans to be orchestrated through the standard Llama Stack API surface. The provider supports two execution modes: inline (local execution within the Llama Stack server process) and remote (execution via Kubeflow Pipelines on Kubernetes).

The provider offers pre-defined compliance frameworks (OWASP LLM Top 10, AVID Security/Ethics/Performance) and scan profiles (quick, standard), as well as custom probe configurations. It can test LLMs with or without safety shields (input/output guardrails such as Prompt-Guard-86M and Llama-Guard-3-8B), comparing model vulnerability scores with and without protection layers. Results are scored on a 0.0-1.0 vulnerability scale and delivered as JSON, HTML reports, and hit logs via the Llama Stack Files API.

The remote execution mode leverages Kubeflow Pipelines (KFP) to run Garak scans as Kubernetes pipeline tasks, with S3/MinIO for artifact storage and result passing. This enables long-running compliance scans (up to 12 hours) to execute on dedicated cluster resources with GPU support.
