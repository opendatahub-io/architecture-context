## Purpose
**Short**: A pre-configured container image distribution of Meta's Llama Stack framework for running AI/ML inference, agents, evaluation, safety, and tool runtime services within the Red Hat OpenShift AI platform.

**Detailed**: The Llama Stack Distribution packages the open-source Llama Stack server into a production-ready container image optimized for Red Hat OpenShift AI (RHOAI). It provides a unified REST API gateway (port 8321) that aggregates multiple AI/ML capabilities including inference (via vLLM, AWS Bedrock, Azure, Vertex AI, WatsonX, OpenAI), vector I/O (Milvus, pgvector, Faiss, Qdrant), agents, evaluation (TrustyAI LM-Eval, RAGAS, Garak), safety (TrustyAI FMS), scoring, tool runtimes (Brave Search, Tavily, MCP, RAG), file management, and batch processing.

The distribution uses a plugin-based provider architecture where capabilities are enabled/disabled via environment variables at runtime. It ships with PostgreSQL as the default persistence backend (both key-value and SQL stores) and supports OpenTelemetry instrumentation for observability. The container image is built on UBI9 Python 3.12 and uses Konflux CI/CD for production builds, with wheel releases managed through GitLab artifact repositories. The image pre-downloads the `ibm-granite/granite-embedding-125m-english` model for local embedding capabilities.
