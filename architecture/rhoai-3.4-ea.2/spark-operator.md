## Purpose
**Short**: A Kubernetes operator that manages the lifecycle of Apache Spark applications on Kubernetes/OpenShift clusters.

**Detailed**: The Spark Operator automates the submission, monitoring, scheduling, and lifecycle management of Apache Spark applications running on Kubernetes. It provides Custom Resource Definitions (CRDs) that allow users to declaratively define Spark applications, scheduled Spark applications (cron-like), and Spark Connect server sessions.

The operator consists of two main deployments: a **controller** that watches SparkApplication, ScheduledSparkApplication, and SparkConnect custom resources and reconciles them into Kubernetes-native objects (driver pods, executor pods, services, ingresses), and a **webhook** that provides mutating and validating admission control for Spark pods and SparkApplication resources. The webhook injects Spark-specific configuration, volumes, and environment variables into pods at creation time.

This is a downstream fork of the Kubeflow Spark Operator, customized for Red Hat OpenShift AI (RHOAI) and Open Data Hub (ODH) deployments. It deploys into the `redhat-ods-applications` namespace for RHOAI, with kustomize overlays providing distribution-specific configuration including network policies and image references.

## Architecture Components

| Component | Type | Purpose |
|-----------|------|---------|
| Controller Deployment | Go Operator (controller-runtime) | Watches SparkApplication, ScheduledSparkApplication, SparkConnect CRs; creates/manages driver pods, executor pods, Services, Ingresses, ConfigMaps |
| Webhook Deployment | Admission Webhook Server | Mutates Spark pods (inject config, volumes, env vars), validates/defaults SparkApplication and ScheduledSparkApplication CRs |
| SparkApplication Controller | Reconciler | Manages full lifecycle of Spark applications: submission, monitoring, restart, TTL cleanup; creates WebUI Services and Ingresses |
| ScheduledSparkApplication Controller | Reconciler | Implements cron-based scheduling of SparkApplications with concurrency policies (Allow/Forbid/Replace) and run history management |
| SparkConnect Controller | Reconciler | Manages Spark Connect server pods, ConfigMaps (executor pod templates), and ClusterIP services exposing gRPC/RPC ports |
| MutatingWebhookConfiguration Controller | Reconciler | Manages webhook certificate injection into MutatingWebhookConfiguration |
| ValidatingWebhookConfiguration Controller | Reconciler | Manages webhook certificate injection into ValidatingWebhookConfiguration |
| Batch Scheduler Interface | Plugin | Pluggable scheduler support for Volcano and kube-scheduler-plugins |
