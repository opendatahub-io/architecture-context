## Purpose
**Short**: Implements Ragas evaluation framework as an out-of-tree Llama Stack evaluation provider for LLM quality assessment.

**Detailed**: The llama-stack-provider-ragas component integrates the [Ragas](https://github.com/explodinggradients/ragas) evaluation framework into the Llama Stack ecosystem as an external provider plugin. It enables users to evaluate LLM outputs using standardized metrics (answer relevancy, context precision, faithfulness, context recall) through the Llama Stack eval API.

The provider ships in two execution modes: an **inline** mode that runs Ragas evaluation in the same process as the Llama Stack server, and a **remote** mode that offloads evaluation to Kubeflow Pipelines for distributed execution. The remote mode submits KFP pipelines that retrieve datasets from Llama Stack, run Ragas evaluations in containerized components, and persist results to S3-compatible storage.

This component is part of the TrustyAI initiative within RHOAI and is built by Konflux using the `Dockerfile.konflux` which produces the `odh-trustyai-ragas-lls-provider-dsp-rhel9` container image based on UBI9/Python 3.12. The image serves as the base image for Kubeflow Pipeline components in the remote execution mode.

|-------------|---------------|---------|
