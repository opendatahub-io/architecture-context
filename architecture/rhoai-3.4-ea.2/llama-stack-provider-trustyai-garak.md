## Purpose
**Short**: Out-of-tree Llama Stack evaluation provider that performs automated LLM red-teaming and vulnerability scanning using NVIDIA Garak.

**Detailed**: This component implements TrustyAI's Garak-based red-teaming as an external evaluation provider for the Llama Stack framework. It enables automated vulnerability assessment of Large Language Models for prompt injection, jailbreaks, toxicity, bias, and other security concerns, with support for compliance frameworks including OWASP LLM Top 10, AVID taxonomy, and CWE.

The provider supports three deployment modes: (1) **Total Remote** -- the Llama Stack server runs on OpenShift/Kubernetes and Garak scans execute as Kubeflow Pipeline (KFP) jobs; (2) **Partial Remote** -- a local Llama Stack server submits KFP pipelines to a remote cluster; (3) **Total Inline** -- everything runs locally for development and testing. In remote mode, scan results are persisted to S3-compatible object storage via OpenShift AI Data Connection secrets, and KFP dashboard metrics/HTML reports are generated automatically.

The provider also integrates with the eval-hub framework, providing an adapter that can be deployed as a standalone service pod. It supports shield testing (input/output guardrail effectiveness measurement), synthetic data generation (SDG) for intent-based risk assessment, and produces detailed scoring including a Tier-Based Security Aggregate (TBSA) metric.
