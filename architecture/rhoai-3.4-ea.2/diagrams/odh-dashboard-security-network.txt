                     same namespace pods
  - Ports: 8443, 8043, 8143, 8243, 8343, 8543, 8643, 8743 (all TCP)

  - Pod Selector: app.kubernetes.io/managed-by=perses-operator

  - Allowed Sources: Pods with app.kubernetes.io/part-of=rhods-dashboard
                     in redhat-ods-applications namespace
  - Ports: 8080/TCP

Authentication & Authorization:
- All external traffic via kube-rbac-proxy (port 8443)
- AuthZ: User must be able to `list` `projects` in project.openshift.io
- K8s API calls use user impersonation (user's own RBAC permissions)
- Plugin containers receive user token via x-forwarded-access-token header
- /api/health endpoint is unsecured (health check only)

Secrets:
- dashboard-proxy-tls (kubernetes.io/tls) - TLS cert for kube-rbac-proxy,
  provisioned by OpenShift service serving cert signer, auto-rotated
