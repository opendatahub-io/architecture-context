    - events: create, patch
    - serviceaccounts: get, list
    - poddisruptionbudgets: create, get, delete
    - secrets (argo-workflows-agent-ca-certificates): get

  - argo-server-cluster-role (bound to SA: argo-server)
    - configmaps: get, watch, list
    - secrets: get, create
    - pods, pods/exec, pods/log: get, list, watch, delete
    - events: watch, create, patch
    - serviceaccounts: get, list, watch
    - argoproj.io (all CRDs): create, get, list, watch, update, patch, delete

  - argo-aggregate-to-admin (label-aggregated)
    - argoproj.io (all CRDs + tasksets): full CRUD

Namespace Roles:
  - argo-role
    - coordination.k8s.io/leases: create, get, update (leader election)
    - secrets: get

Secrets:
  - argo-workflows-agent-ca-certificates (Opaque) - CA certs, Admin/Operator, No auto-rotate
  - sso-client-secret (Opaque) - OIDC client credentials, Admin/Operator, No auto-rotate
  - argo-server-tls (kubernetes.io/tls) - TLS cert, Admin/cert-manager, Depends on provisioner

Container Security Contexts:
  - argo-server: runAsNonRoot=true, readOnlyRootFilesystem=true,
    allowPrivilegeEscalation=false, drop ALL
  - workflow-controller: runAsNonRoot=true, readOnlyRootFilesystem=true,
    allowPrivilegeEscalation=false, drop ALL
  - argoexec: runAsNonRoot=true (UID 2000)

Authentication & Authorization:
  - /api/v1/*: Bearer Token (ServiceAccount) → Kubernetes RBAC impersonation
  - /api/v1/*: SSO (OIDC) → OIDC token + optional RBAC mapping
  - gRPC Services: Bearer Token/SSO → Rate-limited, auth-gated
