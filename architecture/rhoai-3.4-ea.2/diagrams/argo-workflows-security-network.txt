  - argoproj.io: workflows, workflowtasksets, workflowartifactgctasks (full CRUD)
  - argoproj.io: workflowtemplates, clusterworkflowtemplates (get,list,watch)
  - argoproj.io: workflowtaskresults (list,watch,deletecollection)
  - argoproj.io: cronworkflows (full CRUD)
  - policy: poddisruptionbudgets (create,get,delete)

- argo-server-cluster-role (bound to SA: argo-server)
  - Core: configmaps (get,watch,list)
  - Core: secrets (get,create)
  - Core: pods, pods/exec, pods/log (get,list,watch,delete)
  - Core: events (watch,create,patch)
  - Core: serviceaccounts (get,list,watch)
  - argoproj.io: eventsources, sensors, workflows, workfloweventbindings,
    workflowtemplates, cronworkflows, clusterworkflowtemplates (full CRUD)

- argo-aggregate-to-view (aggregated)
  - argoproj.io: workflows, workfloweventbindings, workflowtemplates,
    cronworkflows, clusterworkflowtemplates, workflowtaskresults (get,list,watch)

- argo-aggregate-to-edit (aggregated)
  - argoproj.io: all workflow resources (full CRUD)

- argo-aggregate-to-admin (aggregated)
  - argoproj.io: all workflow resources + workflowtasksets (full CRUD)

Namespace Roles:
- argo-role (bound to SA: argo)
  - coordination.k8s.io: leases (create,get,update) - leader election
  - Core: secrets (get)

Service Accounts:
- argo: workflow-controller operations (pod management, CRD operations)
- argo-server: API server operations (pod log access, CRD management)

Secrets:
- argo-workflows-agent-ca-certificates (Opaque) - CA certs for agent tasks, Manual/Operator, No rotation
- SSO client secret (Opaque) - OIDC credentials, Manual/Operator, No rotation
- argo-server-tls (kubernetes.io/tls, optional) - TLS cert for HTTPS, Manual/cert-manager, Depends

Authentication Modes:
- Bearer Token: K8s SA token, impersonation with caller's identity
- SSO (OIDC JWT): Maps OIDC groups to K8s RBAC
- Server mode: Uses argo-server's own SA permissions
- Enforcement: Gatekeeper middleware on all /api/v1/* endpoints

FIPS Compliance:
- workflow-controller: CGO_ENABLED=1, GOEXPERIMENT=strictfipsruntime
- argoexec: Dual binary (FIPS + non-FIPS), strictfipsruntime tags
- Base image: registry.redhat.io/ubi9/ubi-minimal

Network Policies:
- None defined in static manifests (managed by DSP operator)

Ingress:
- No ingress resources defined (managed by DSP operator)
