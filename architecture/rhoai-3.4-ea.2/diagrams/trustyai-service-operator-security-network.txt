Authentication & Authorization:
================================================================================

- TrustyAI Service API (8443/TCP):
  kube-rbac-proxy sidecar performs TokenReview + SubjectAccessReview
  Conditionally enabled via: security.opendatahub.io/enable-auth annotation

- Guardrails Orchestrator (8432/TCP):
  kube-rbac-proxy sidecar (conditional, annotation-driven)

- NeMo Guardrails:
  kube-rbac-proxy sidecar (conditional, annotation-driven)

- EvalHub (8443/TCP):
  TLS client certificates (OpenShift serving certs)

- Operator Metrics (8443/TCP):
  Bearer Token (ServiceAccount) via kube-rbac-proxy
