Network Policies (Dynamic)
================================================================================

Per-TrainJob NetworkPolicy created by controller:

Rule 1: Inter-pod communication
  - Source: Pods with label jobset.sigs.k8s.io/jobset-name: <trainjob-name>
  - Destination: Same pods (all ports, TCP)
  - Purpose: NCCL/MPI/gRPC distributed training communication

Rule 2: Controller metrics access
  - Source: Controller pods (ns: opendatahub, app.kubernetes.io/name: trainer)
  - Destination: Port 28080/TCP
  - Purpose: Progression tracking (only when annotation is set)
