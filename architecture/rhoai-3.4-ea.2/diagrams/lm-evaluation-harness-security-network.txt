- EvalHub Callback API: POST, Bearer Token (file-mounted), validated by EvalHub service
- OCI Registry: Push, Docker registry auth (authfile or username/password), validated by Registry ACL
- HuggingFace Hub: GET, HF Token (optional), validated by HuggingFace
