================================================================================
CONTAINER SECURITY
================================================================================

All RHOAI containers (92 images):
- Run as non-root: Yes (UIDs: 1001, 2000, 5000, or OpenShift arbitrary)
- Capabilities: Drop ALL
- Privilege escalation: Disabled
- Seccomp profile: RuntimeDefault
- Base image: UBI 9 (ubi9-minimal or ubi9/python-312)
- FIPS compliance: Go components use GOEXPERIMENT=strictfipsruntime
- Read-only rootfs: Operators yes, workbenches no
- Build system: Konflux with SLSA provenance + Sigstore signing
