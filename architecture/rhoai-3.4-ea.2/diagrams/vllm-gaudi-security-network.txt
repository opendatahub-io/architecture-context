Authentication & Authorization:

  - In RHOAI deployment: Bearer Token / mTLS enforced by KServe + Istio sidecar
  - Enforcement point: Istio sidecar (mTLS) + KServe auth policy (Bearer Token)

Container Security:

  - Run As User: 2000 (non-root)

  - FIPS: openssl-fips-provider-so explicitly removed
  - License: Apache-2.0

Service Mesh Configuration:

  - RHOAI auth policy applied at gateway level
