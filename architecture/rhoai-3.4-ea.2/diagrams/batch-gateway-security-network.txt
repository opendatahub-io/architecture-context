================================================================================
SECURITY CONFIGURATION
================================================================================

Multi-Tenancy:
- Tenant isolation via configurable HTTP header (default: X-MaaS-User)
- All database queries filter by tenant ID
- Cross-tenant access returns 404 Not Found

Middleware Stack (outermost to innermost):
1. Recovery Middleware - catches panics
2. Request Middleware - request ID, tenant ID extraction, logging, metrics
3. Security Headers Middleware:
   - X-Content-Type-Options: nosniff
   - X-Frame-Options: DENY
   - X-XSS-Protection: 1; mode=block
   - CORS preflight handling

Authentication & Authorization:
- Currently: No authentication enforced (open access)
- Planned: API key/JWT authentication, authorization checks, rate limiting
- /health and /metrics: Always open access

TLS Configuration (when enabled):
- Minimum Version: TLS 1.2
- Cipher Suites:
  - TLS_ECDHE_RSA_WITH_AES_128_GCM_SHA256
  - TLS_ECDHE_RSA_WITH_AES_256_GCM_SHA384
  - TLS_ECDHE_ECDSA_WITH_AES_128_GCM_SHA256
  - TLS_ECDHE_ECDSA_WITH_AES_256_GCM_SHA384
  - TLS_ECDHE_RSA_WITH_CHACHA20_POLY1305
  - TLS_ECDHE_ECDSA_WITH_CHACHA20_POLY1305
