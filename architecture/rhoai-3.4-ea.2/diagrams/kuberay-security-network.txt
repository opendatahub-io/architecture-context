================================================================================
NETWORK POLICIES (Dynamic, per RayCluster)
================================================================================

Condition: Applied when odh.ray.io/secure-trusted-network=true

Head Node NetworkPolicy ({cluster}-head):
  - Intra-cluster:   Pods with ray.io/cluster={name}     → All ports
  - Same namespace:  All pods in same namespace            → 10001/TCP, 8265/TCP
  - Operator:        kuberay-operator pods                 → 8265/TCP, 10001/TCP
  - Monitoring:      Pods in MONITORING_NAMESPACE          → 8080/TCP
  - Gateway:         Pods in openshift-ingress namespace   → 8265/TCP
  - Secured ports:   All sources                           → 8443/TCP

Worker Node NetworkPolicy ({cluster}-workers):
  - Intra-cluster:   Pods with ray.io/cluster={name}     → All ports
  (Workers only accept traffic from same cluster)
