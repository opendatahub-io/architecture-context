## Purpose
**Short**: Argo Workflows is a Kubernetes-native workflow engine used as the execution backend for Data Science Pipelines in RHOAI.

**Detailed**: Argo Workflows provides the core workflow execution engine for the Red Hat OpenShift AI (RHOAI) Data Science Pipelines component. It orchestrates multi-step computational workflows on Kubernetes, where each step runs as a container. The workflow controller watches for Workflow custom resources and manages the full lifecycle of workflow execution, including creating pods for each workflow step, managing artifacts, handling retries, and garbage collecting completed resources.

In the RHOAI context, Argo Workflows serves as the backend execution engine for Kubeflow Pipelines (Data Science Pipelines). The argo-workflowcontroller manages workflow execution by creating and monitoring pods for each pipeline step, while the argoexec sidecar runs inside each workflow pod to manage artifact collection, script execution, and container lifecycle. The Argo Server component (not shipped separately for RHOAI) provides a REST/gRPC API and web UI. RHOAI ships two container images built via Konflux: the workflow controller and the argoexec executor, both built with FIPS-compliant Go runtime on UBI9.

## Architecture Components

| Component | Type | Purpose |
|-----------|------|---------|
| workflow-controller | Go Binary (Deployment) | Watches Workflow CRs and orchestrates pod creation/lifecycle for each workflow step. Uses leader election for HA. Exposes Prometheus metrics on port 9090/TCP and health/pprof on port 6060/TCP. |
| argoexec | Go Binary (Sidecar) | Executor sidecar injected into each workflow pod. Manages artifact collection, script execution, resource monitoring, and container wait logic. Emissary executor pattern. |
| argo-server | Go Binary (Deployment) | REST API + gRPC server + Web UI on port 2746/TCP. Provides workflow management API with SSO/Bearer/Server auth modes. Serves static React UI. Not separately shipped for RHOAI (used via DSP API server). |
| argo-ui | React/TypeScript SPA | Web-based UI for workflow visualization, submission, and management. Bundled into argo-server binary at build time. |
