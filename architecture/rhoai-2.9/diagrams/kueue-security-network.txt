Data Flow Summary:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. Job Submission:
   User → API Server (6443/TCP HTTPS) → Webhook (9443/TCP mTLS) →
   Controller creates Workload CRD

2. Workload Admission:
   Controller watches ClusterQueue → Checks quota → Admits Workload →
   Unsuspends Job → Pods scheduled

3. Metrics Collection:
   Prometheus → Metrics Service (8080/TCP HTTP) → Controller metrics

4. Autoscaling:
   Controller → API Server (6443/TCP HTTPS) → Creates ProvisioningRequest →
   cluster-autoscaler provisions nodes
