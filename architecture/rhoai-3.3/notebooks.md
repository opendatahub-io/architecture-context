## Purpose
**Short**: Provides pre-built workbench container images (Jupyter, Code Server, RStudio) and Elyra pipeline runtime images for RHOAI data science environments.

**Detailed**: The notebooks repository builds and maintains a comprehensive set of container images used as workbench environments in Red Hat OpenShift AI (RHOAI). These images provide interactive development environments for data scientists and ML engineers, including JupyterLab notebooks, VS Code (code-server), and RStudio Server. Each workbench type is available in multiple variants optimized for different accelerators (CPU, NVIDIA CUDA, AMD ROCm) and ML frameworks (PyTorch, TensorFlow, TrustyAI, LLM Compressor).

The repository does not contain any operator or controller code. Instead, it produces container images and kustomize manifests (OpenShift ImageStream definitions). The rhods-operator deploys these manifests via the `overlays/additional` kustomize overlay, which creates ImageStream resources in the OpenShift cluster. The ODH Notebook Controller (a separate component) then uses these ImageStreams to launch workbench pods for users. The repository maintains a versioned history of images (N through N-6), allowing users to select from current and previous workbench versions.

Elyra pipeline runtime images are also provided for executing notebook pipelines in headless mode, supporting Data Science Pipelines workflows. All images are built using Konflux CI/CD pipelines (110 Tekton PipelineRun definitions in `.tekton/`), targeting RHEL 9 (UBI9) base images and supporting multi-architecture builds (amd64, arm64, ppc64le, s390x).

## Image Versioning Strategy

Images follow a rolling N-version scheme:
- **N** (2025.2): Current/latest version, Python 3.12, `opendatahub.io/workbench-image-recommended: true`
- **N-1** (2025.1): Previous release, Python 3.11, `opendatahub.io/workbench-image-recommended: false`
- **N-2** through **N-6**: Historical versions marked with `opendatahub.io/image-tag-outdated: true`

N-1 images reference `registry.redhat.io/rhoai/` (shipped Red Hat images). N-2 through N-6 reference `quay.io/modh/` (community/pre-release images).
