    - leaderworkerset.x-k8s.io: leaderworkersets → full CRUD
    - monitoring.coreos.com: podmonitors, servicemonitors → full CRUD

  kserve-proxy-role:
    - authentication.k8s.io: tokenreviews → create

    - kserve-view: read-only on inferenceservices, llminferenceservices, servingruntimes

Authentication & Authorization
================================================================================

  /metrics (controller):
    Methods: GET
    Auth: kube-rbac-proxy (TokenReview + SubjectAccessReview)
    Enforcement: kube-rbac-proxy sidecar (8443/TCP)

  Webhook endpoint:
    Methods: POST
    Auth: TLS client verification
    Enforcement: Kubernetes API server

  InferenceService endpoints:
    Auth: Istio mTLS (mesh-internal) or Gateway TLS
    Enforcement: Istio / Gateway + PeerAuthentication

  InferenceGraph router:
    Auth: Bearer Token propagation
    Enforcement: Router binary (Authorization header propagation)
