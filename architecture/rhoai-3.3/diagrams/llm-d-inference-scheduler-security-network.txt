SSRF Protection:
================================================================================
- P/D sidecar maintains dynamic allowlist of valid prefill targets
- Watches InferencePool resources to discover pod selectors
- Watches matching Pods to build allowlist of pod IPs and hostnames
- Validates x-prefiller-host-port header against allowlist
- Returns HTTP 403 if prefill target is not in allowlist
- Enabled via: --enable-ssrf-protection flag
