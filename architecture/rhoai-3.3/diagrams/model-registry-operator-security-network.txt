RBAC SUMMARY:
================================================================================
ClusterRole: manager-role
  ServiceAccount: model-registry-operator-controller-manager

  Core API:
  - configmaps, pvcs, secrets, serviceaccounts, services: CRUD + watch
  - endpoints, pods, pods/log: get, list, watch
  - events: create, patch

  Apps API:
  - deployments: CRUD + watch

  ModelRegistry API (modelregistry.opendatahub.io):
  - modelregistries: CRUD + watch
  - modelregistries/status: get, patch, update
  - modelregistries/finalizers: update

  OpenShift APIs:
  - route.openshift.io: routes, routes/custom-host: CRUD + watch
  - user.openshift.io: groups: CRUD + watch
  - config.openshift.io: ingresses: get, list, watch

  RBAC API:
  - roles, rolebindings, clusterrolebindings: CRUD + watch

  Auth APIs:
  - authentication.k8s.io: tokenreviews: create
  - authorization.k8s.io: subjectaccessreviews: create

  Networking:
  - networking.k8s.io: networkpolicies: CRUD + watch

  Migration:
  - migration.k8s.io: storageversionmigrations: CRUD + watch

  Platform CRDs (read-only):
  - components.platform.opendatahub.io: modelregistries: get, list, watch
  - services.platform.opendatahub.io: auths: get, list, watch

Dynamically Created Roles:
  - registry-user-{name}: services/{name}, endpoints/{name} -> get
  - model-catalog: services/model-catalog, endpoints/model-catalog -> get
  - model-catalog-admin: configmaps/model-catalog-sources -> get, update, patch
