Security Notes:
- Llama Stack Server communication is plaintext HTTP (8321/TCP)
  - Security relies on cluster network isolation
  - No application-level authentication
- KFP pod-to-Llama Stack communication is plaintext HTTP
- External S3 communication uses HTTPS with TLS 1.2+
- KFP API communication uses HTTPS with Bearer Token auth
- Secrets are mounted as environment variables in KFP pods
