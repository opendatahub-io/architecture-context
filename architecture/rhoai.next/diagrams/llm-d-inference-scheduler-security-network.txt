Security Notes:
- EPP gRPC server supports TLS 1.2+ (enabled via --secure-serving, default true)
- PD sidecar supports TLS 1.2+ (enabled via --secure-proxy, default true)
- PD sidecar SSRF protection watches InferencePool for dynamic pod allowlist
- Cert reload on both EPP and PD sidecar via filesystem watching (fsnotify)
- Metrics endpoint auth delegates to Kubernetes RBAC via TokenReview/SAR
- No service mesh (Istio) requirement in base deployment (available as overlay)
