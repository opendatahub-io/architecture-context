RBAC Summary:
- No Kubernetes RBAC defined (library, not a deployed operator/service)
- No ClusterRoles, Roles, or RoleBindings
- No ServiceAccounts

Service Mesh Configuration:
- N/A (library runs within host process, not as a Kubernetes pod)
- If consumed within a service mesh pod, inherits pod's mesh configuration

Secrets:
- API_KEY (Environment variable) - Llama Stack server authentication
  - Provisioned by: Manual (user-provided .env file)
  - Auto-Rotate: No
  - Used by: LlamaStackClient, OpenAI client (Bearer token)
- BASE_URL (Environment variable) - Llama Stack server endpoint
  - Provisioned by: Manual (user-provided .env file)
  - Auto-Rotate: No
  - Used by: LlamaStackClient connection setup

Network Exposure:
- No inbound ports (library, not a service)
- No Kubernetes Services
- No Ingress resources
- Outbound only: Llama Stack Server, OpenAI-compatible APIs

Data Sensitivity:
- Processes user-provided benchmark data (questions, ground truth answers)
- Transmits document content to external embedding and LLM services
- Stores evaluation results locally (JSON) or in-memory (KFP handler)
- No persistent data storage within the library itself
