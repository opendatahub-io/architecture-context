===============================================================================
  RBAC SUMMARY

No RBAC resources defined in this repository.
RBAC is managed by KServe and RHOAI platform operators:
  - KServe creates ServiceAccounts for inference pods
  - RHOAI manages ClusterRoles/RoleBindings for model serving

===============================================================================
  CONTAINER SECURITY
===============================================================================

- Runs as non-root user (UID 2000)
- No TLS at container level (plaintext HTTP/gRPC)
- TLS termination at platform ingress layer
- FIPS compliance delegated to RHAIIS base image (UBI9 + OpenSSL)
- No compiled code in this layer (config-only Dockerfile)
- Multi-arch build: amd64 + arm64
- Build: Non-hermetic (hermetic: false), Konflux/Tekton pipeline
- Vulnerability scanning: Clair (in build pipeline)
