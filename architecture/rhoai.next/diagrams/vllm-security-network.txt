RBAC Summary:
- No RBAC resources defined by this component.
- This is a container image, not an operator.
- RBAC is managed externally by KServe (ServingRuntime / InferenceService).

Service Mesh Configuration:
- PeerAuthentication: STRICT (mTLS enforced by Istio sidecar)
- AuthN/AuthZ: Delegated entirely to KServe gateway and Istio
- Sidecar: Automatically injected by Istio into inference pods

Secrets:
- No secrets defined by this repository.
- Model access credentials are injected by KServe at runtime:
  - S3 credentials (AWS_ACCESS_KEY_ID, AWS_SECRET_ACCESS_KEY) — via KServe storage-config Secret
  - Hugging Face token (HF_TOKEN) — optional, via environment variable injection

Container Security:
- Runs as non-root user (USER 2000)
- No privilege escalation
- Base image: registry.redhat.io/rhaiis/vllm-cuda-rhel9:3.2.2 (Red Hat signed)
- FIPS compliance: Inherited from RHAIIS base image (UBI-based)

Network Exposure:
- No ports exposed externally by this container directly
- All external access is routed through KServe Gateway → Istio sidecar
- Pod-level ports (8000, 8033) accept only pod-internal or mTLS traffic
