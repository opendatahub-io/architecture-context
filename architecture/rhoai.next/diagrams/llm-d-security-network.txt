Production Note:
- llm-d does NOT install ClusterRoles in production deployments
- Security relies on network-level isolation and gateway-provider policies
- vLLM API is plaintext HTTP with no built-in authentication
- RDMA transport is unencrypted (L2)
- Gateway-level auth (Istio AuthorizationPolicy, etc.) is expected
  to be layered on by the deployment environment
