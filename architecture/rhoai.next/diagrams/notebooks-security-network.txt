Secrets:
- None defined by this component
- User-provisioned secrets may be mounted for:
  - Data source credentials (S3, MongoDB, ODBC)
  - Git configuration
  - Injected by Notebook Controller based on Dashboard UI configuration

FIPS Compliance:
- mongocli: CGO_ENABLED=1, -tags strictfipsruntime (Go FIPS-compliant build)
- Python/R/Node.js: System OpenSSL from UBI9 base image (FIPS mode via crypto-policies)
- OpenSSL: System trust store (/etc/pki/tls/certs/ca-bundle.crt)
- compat-openssl11: Explicitly installed in all images
- Risk: User-installed packages may introduce non-FIPS crypto at runtime

Build Hermeticity:
- All Konflux Dockerfiles use /cachi2/output/deps/ with --no-index flags
- Zero network access during builds
- Supply chain defense: exclude-newer = "3 days" (refuses packages < 3 days old)
- Lock files: RPM (rpms.lock.yaml), Python (pylock.*.toml), Go (go.sum), npm (package-lock.json)
