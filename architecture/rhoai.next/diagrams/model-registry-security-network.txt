Security Notes:
- No application-level authentication (delegated entirely to Istio)
- No rate limiting at application level
- Null byte injection protection in validation middleware
- TLS 1.2+ configurable for database connections (cipher suites, server cert verification)
- Insecure ciphers rejected by TLS configuration module
- HuggingFace gated/private model validation prevents unauthorized model deployment
