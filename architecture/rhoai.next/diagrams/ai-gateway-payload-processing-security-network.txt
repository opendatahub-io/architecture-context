- Attached to: Gateway/{inferenceGateway.name} (default: maas-default-gateway)

- Operation: INSERT_AFTER
- Anchor: configurable subfilter
- Effect: Routes inference traffic through ext_proc at 9004/TCP

Secrets:
- User-created Secrets (Opaque) - API keys for external model providers
  - Label: inference.networking.k8s.io/bbr-managed=true
  - Contents: Provider API keys (OpenAI, Anthropic, Azure, Bedrock, Vertex)
  - Provisioned by: User / MaaS controller
  - Auto-Rotate: No
  - Storage: Cached in-memory by secretStore (refreshed via Secret reconciler)

Security Notes:
- ext_proc gRPC (9004/TCP) runs WITHOUT TLS or authentication
  - Security relies on network isolation (same namespace as Gateway)
  - Only Envoy proxy should connect via EnvoyFilter targetRefs
- API keys held in-memory in secretStore (sensitive cloud provider credentials)
  - Persist only in process memory, refreshed via reconciler
- FIPS compliance enforced at build time: GOEXPERIMENT=strictfipsruntime
- No direct ingress; all traffic arrives via Envoy ext_proc filter
