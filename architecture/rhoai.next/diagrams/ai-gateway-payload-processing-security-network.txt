CRDs:
- ExternalModel (inference.opendatahub.io/v1alpha1, Namespaced)
  - Maps client-facing model name → external provider with target model, API format, credential ref
  - ExternalProviderRefs: MaxItems=1 (Phase 2: traffic splitting across multiple providers)

- ExternalProvider (inference.opendatahub.io/v1alpha1, Namespaced)
  - Defines connection to external LLM provider (endpoint FQDN, provider type, auth secret ref)

Secrets:
- Provider API key Secrets (Opaque)
  - Label: inference.networking.k8s.io/bbr-managed=true
  - Data field: api-key
  - Provisioned by: User or MaaS controller
  - Auto-Rotate: No
  - Security note: Client Authorization header is stripped by api-translation plugin
    and re-injected by apikey-injection plugin to prevent credential leakage

Security Notes:
- ⚠ NeMo Guardrails communication uses plaintext HTTP (no TLS)
  - Relies on network-level isolation for security
  - Review for environments requiring end-to-end encryption
- NeMo response size capped at 1 MiB to prevent DoS
- gRPC ExtProc is plaintext but pod-local (Envoy connects from same network)
- All external LLM provider traffic uses HTTPS TLS 1.2+
- FIPS compliant build: GOEXPERIMENT=strictfipsruntime, CGO_ENABLED=1, UBI9 base
- No custom TLS configuration; delegates to Go stdlib defaults (FIPS via OpenSSL)
- No non-FIPS crypto detected (no MD5/RC4/DES imports)
