FIPS Compliance:

- Linking: Dynamic (CGO_ENABLED=1)
- Base image: registry.access.redhat.com/ubi9/ubi-minimal (OpenSSL included)
- Crypto: stdlib crypto/tls only (FIPS via OpenSSL with strictfipsruntime)
- No third-party crypto imports
- ⚠ NeMo HTTP calls use plaintext (no TLS)
- External provider TLS delegated to Envoy proxy

Security Notes:
- ⚠ NeMo Guardrails communication is plaintext HTTP (relies on network isolation)
- Client authorization header is stripped by api-translation before forwarding
- Provider API keys are injected by apikey-injection plugin from cached Secrets
- NeMo response size capped at 1 MiB to prevent DoS
- ExternalModel CRDs are eventually consistent (in-memory store, non-shared across replicas)
