FIPS Notes:
  - Python service on UBI 9 base image (inherits OpenSSL FIPS config)
  - Risk: model_registry_skip_tls_verify bypasses TLS verification (dev only)
  - httpx uses system OpenSSL via Python ssl module

Safety Features:
  - Read-only mode blocks all mutations
  - Dangerous operation gating (deletes require explicit enablement)
  - Confirmation pattern (confirm=True for destructive operations)
  - Credential masking (S3 secret keys never returned in full)
  - GIT_TERMINAL_PROMPT=0 prevents credential prompts during git clone
