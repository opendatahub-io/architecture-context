ALTERNATIVE TRANSPORT: stdio (Local)
=====================================
- AI Agent connects via stdin/stdout pipe (no network)
- Used for Claude Desktop, local development
- No TLS, no Route — communication is process-local
- K8s auth via kubeconfig file on host

FIPS Notes
==========
- Base image: UBI9/python-312 (provides FIPS-compliant OpenSSL)
- No custom TLS configuration in application code
- Crypto delegated to system OpenSSL (httpx, kubernetes client)
- CONCERN: skip_tls_verify option for Model Registry bypasses cert validation
- No Konflux Dockerfile present (upstream branch)
