Build Hermeticity:

Custom Detector Security:
- Static code analysis gate enforces restrictions on custom Python detectors
- Forbidden imports: os, subprocess, sys, shutil
- Forbidden calls: eval, exec, open, compile, input
- Exception: os.environ and os.getenv are permitted
