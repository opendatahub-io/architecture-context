RBAC Summary:
- No ClusterRoles or RoleBindings defined by this component
- RBAC for workbench pods is managed by the ODH Notebook Controller
- Workbench pods run as non-root (UID 1001, GID 0)
- OpenShift random UID security model supported via chmod/chgrp operations

Authentication Model:
- CRITICAL: All workbench images DISABLE built-in auth (--ServerApp.token='')
- Authentication DELEGATED to platform:
  - RHOAI 3.x: kube-rbac-proxy sidecar injected by Notebook Controller
  - RHOAI 2.x: oauth-proxy sidecar injected by Notebook Controller
- Auth Enforcement Point: Sidecar container (NOT the workbench)
- Risk: Direct exposure without sidecar = UNAUTHENTICATED ACCESS

Secrets:
- rhel-subscription-secret (Opaque) - RHEL subscription creds for entitled builds
  - Provisioned by: Manual (build environment)
  - Auto-rotate: No
  - Usage: RStudio RHEL9 image builds only
- git_auth_secret (kubernetes.io/basic-auth) - Git auth for Konflux Tekton
  - Provisioned by: Konflux / Pipelines as Code
  - Auto-rotate: Yes
