RBAC Summary:
- Workbench images do NOT define RBAC resources
- RBAC is managed by odh-notebook-controller:
  - Creates per-notebook ServiceAccounts
  - Creates per-notebook RoleBindings
  - Injects kube-rbac-proxy sidecar for authentication

Secrets:
- rhel-subscription-secret (Opaque) - RHEL subscription credentials for RStudio RHEL9 builds, Manual provisioning, No auto-rotation
- git_auth_secret (kubernetes.io/basic-auth) - Git authentication for Konflux pipeline clones, Konflux/PaC provisioned, Auto-rotated

FIPS Compliance:
- mongocli binary: Built with CGO_ENABLED=1, tags=strictfipsruntime (dynamic linking)
- Python images: FIPS via UBI9 system OpenSSL
- TLS: Platform-managed termination (Gateway/Route); JupyterLab uses stdlib ssl with system defaults
- No non-FIPS crypto detected in workbench startup code
