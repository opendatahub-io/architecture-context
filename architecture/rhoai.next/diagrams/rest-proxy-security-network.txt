Environment Variables:
  - REST_PROXY_GRPC_PORT (default: 8033) - Upstream gRPC port
  - REST_PROXY_LISTEN_PORT (default: 8008) - HTTP listener port
  - REST_PROXY_USE_TLS (default: false) - Enable TLS for gRPC upstream
  - REST_PROXY_SKIP_VERIFY (default: false) - Skip TLS certificate verification
  - MM_TLS_KEY_CERT_PATH - Path to TLS certificate for HTTP listener
  - MM_TLS_PRIVATE_KEY_PATH - Path to TLS private key for HTTP listener
  - REST_PROXY_GRPC_MAX_MSG_SIZE (default: 16MB) - Max gRPC message size

RBAC Summary:
  - None. This component does not interact with the Kubernetes API.

Secrets:
  - TLS cert/key (via MM_TLS_KEY_CERT_PATH / MM_TLS_PRIVATE_KEY_PATH)
    Type: kubernetes.io/tls
    Purpose: TLS termination for HTTP listener
    Provisioned by: ModelMesh Serving operator
    Auto-rotate: Unknown

Authentication & Authorization:
  - No authentication at proxy level
  - Relies on:
    1. Network-level isolation (pod-local gRPC on loopback)
    2. Platform ingress authentication (if exposed externally)
    3. ModelMesh-level auth (if configured)

FIPS Compliance:
  - Build: CGO_ENABLED=1 GOEXPERIMENT=strictfipsruntime -tags strictfipsruntime
  - Crypto: stdlib crypto/tls backed by OpenSSL (FIPS validated)
  - Risk: InsecureSkipVerify configurable via REST_PROXY_SKIP_VERIFY
  - Base image: UBI9 minimal (FIPS-capable)
