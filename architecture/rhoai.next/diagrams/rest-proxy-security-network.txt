RBAC Summary:
- No Kubernetes RBAC. This component does not interact with the Kubernetes API.
- No ServiceAccount requirements specific to rest-proxy (uses pod's SA).
- No ClusterRoles, Roles, or RoleBindings.

Authentication & Authorization:
- rest-proxy does NOT implement authentication or authorization.
- All auth is delegated to the platform layer (ModelMesh Serving / platform ingress).
- No Bearer tokens, mTLS identity, or API keys are validated by this component.

Secrets:
- TLS certificate (path via MM_TLS_KEY_CERT_PATH)
  - Type: TLS cert file
  - Purpose: TLS termination for incoming REST connections
  - Provisioned by: External (ModelMesh / cert-manager)
  - Auto-Rotate: Unknown
- TLS private key (path via MM_TLS_PRIVATE_KEY_PATH)
  - Type: TLS key file
  - Purpose: TLS termination for incoming REST connections
  - Provisioned by: External (ModelMesh / cert-manager)
  - Auto-Rotate: Unknown

Environment Variables (Security-relevant):
- MM_TLS_KEY_CERT_PATH: Path to TLS certificate for HTTPS listener
- MM_TLS_PRIVATE_KEY_PATH: Path to TLS private key for HTTPS listener
- REST_PROXY_USE_TLS: Enable TLS on outbound gRPC connection (to inference server)
- REST_PROXY_SKIP_VERIFY: Disable TLS certificate verification on gRPC upstream
  ⚠️  WARNING: Setting this to "true" disables cert verification. Acceptable for
     development/testing only. Should be "false" or unset in production.
- REST_PROXY_GRPC_MAX_MSG_SIZE_BYTES: Max gRPC message size (default: 16 MiB)

FIPS Compliance:
- Build: GOEXPERIMENT=strictfipsruntime, CGO_ENABLED=1, -tags strictfipsruntime
- Base image: ubi9/ubi-minimal (includes OpenSSL)
- Crypto: stdlib crypto/tls (FIPS-compliant via OpenSSL at runtime)
- Risk: InsecureSkipVerify can be enabled (does not break FIPS, but reduces security)

Network Topology:
- All communication is pod-internal (localhost) between rest-proxy and inference server
- No external egress from rest-proxy itself
- External ingress to 8008/TCP is managed by ModelMesh Serving infrastructure
- No Kubernetes Service objects defined by this component
- No NetworkPolicy defined by this component
