RBAC Summary:
  None. This component does not interact with the Kubernetes API.
  No ClusterRoles, Roles, or RoleBindings.

Secrets:
  - TLS cert/key (via MM_TLS_KEY_CERT_PATH / MM_TLS_PRIVATE_KEY_PATH)
    Type: kubernetes.io/tls
    Purpose: TLS termination for the HTTP listener
    Provisioned By: ModelMesh Serving operator
    Auto-Rotate: Unknown

Authentication & Authorization:
  - No authentication enforced at proxy level
  - Relies on:
    1. Network-level isolation (pod-local gRPC communication)
    2. Platform ingress authentication (if exposed externally)
    3. Service mesh policies (if deployed with Istio/OSSM)

FIPS Compliance:
  - Build: CGO_ENABLED=1 GOEXPERIMENT=strictfipsruntime -tags strictfipsruntime
  - Linking: Dynamic (CGO_ENABLED=1) for OpenSSL backend
  - Base Image: UBI9 minimal (includes OpenSSL)
  - Crypto: stdlib crypto/tls → FIPS-validated via OpenSSL

Security Risks:
  - REST_PROXY_SKIP_VERIFY env var can disable TLS cert verification
    (not a FIPS issue, but bypasses certificate validation)
  - No authentication at proxy level (by design - relies on network isolation)
  - No health/readiness/liveness probes (relies on parent pod lifecycle)
  - No metrics/monitoring endpoint
