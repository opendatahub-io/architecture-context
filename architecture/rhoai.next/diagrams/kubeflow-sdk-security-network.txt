FIPS Compliance Notes:
  - SDK is a pure Python library (pip package, not container image)
  - No Konflux Dockerfiles - does not go through check-payload validation
  - Delegates all TLS/crypto to: kubernetes Python client, urllib3, fsspec/s3fs
  - FIPS compliance depends on runtime environment (e.g., UBI-based images)
  - Risk: verify_cloud_storage_ssl=False can disable SSL for S3 (default: True)

Authentication Summary:
  - Kubernetes API: Bearer Token (ServiceAccount JWT) or kubeconfig credentials
  - Model Registry: Bearer Token (optional, PEM-encoded) + Custom CA (optional)
  - HuggingFace Hub: Bearer Token (ACCESS_TOKEN env var)
  - S3 Storage: AWS IAM credentials from Kubernetes Secret
  - Spark Connect: None (no authentication)
  - Metrics endpoint: None (no authentication, HTTP only)
