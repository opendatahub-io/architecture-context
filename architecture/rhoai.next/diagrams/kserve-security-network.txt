Secrets:
  - kserve-webhook-server-cert (kubernetes.io/tls) — Webhook TLS, OpenShift serving cert, auto-rotate
  - llmisvc-webhook-server-cert (kubernetes.io/tls) — LLM webhook TLS, OpenShift serving cert, auto-rotate
  - localmodel-webhook-server-cert (kubernetes.io/tls) — LocalModel webhook TLS, OpenShift serving cert, auto-rotate
  - {llmSvc-name}-kserve-self-signed-certs (kubernetes.io/tls) — LLM workload TLS, controller-generated, rotate on expiry
  - kserve-metrics-reader-sa-secret (kubernetes.io/service-account-token) — Metrics reader SA token, no auto-rotate
  - storage-config (Opaque) — S3/GCS/Azure/HF credentials, user-managed, no auto-rotate

Authentication & Authorization:
  - InferenceService (RHOAI raw): Bearer token via kube-rbac-proxy (8443/TCP) → SubjectAccessReview
  - InferenceGraph (raw): Bearer token via TokenReview + SubjectAccessReview in router binary
  - Controller metrics: Bearer token via kube-rbac-proxy sidecar
  - LLMInferenceService scheduler: Bearer token (tokenReview) at EPP
  - Webhook endpoints: Kubernetes API server mTLS
