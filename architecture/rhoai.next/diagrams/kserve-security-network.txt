================================================================================
FIPS Compliance Notes
================================================================================

- Controllers (kserve, llmisvc): CGO_ENABLED=1, GOEXPERIMENT=strictfipsruntime → FIPS compliant
- Agent, Router: CGO_ENABLED=1, GOEXPERIMENT=strictfipsruntime → FIPS compliant
- localmodel-controller, localmodelnode-agent: CGO_ENABLED=0 → NOT FIPS compliant (static linking)
  ⚠️  May fail check-payload on FIPS-required clusters
- storage-initializer: Python on UBI 9 base → uses system OpenSSL (FIPS compliant)

Service Mesh Configuration
================================================================================

- PeerAuthentication: mTLS STRICT (namespace-scoped via Istio/OSSM)
- VirtualService: Traffic routing for InferenceService endpoints
- DestinationRule: Created by llmisvc-controller on OpenShift (distro build tag)
- HTTPRoute: Modern Gateway API routing for both InferenceService and LLMInferenceService
