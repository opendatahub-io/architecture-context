================================================================================
AUTHENTICATION & AUTHORIZATION
================================================================================

| Endpoint                          | Auth Mechanism                          | Enforcement Point              |
|-----------------------------------|-----------------------------------------|--------------------------------|
| InferenceService (RHOAI raw)      | Bearer token via kube-rbac-proxy        | kube-rbac-proxy sidecar (8443) |
| InferenceGraph (raw)              | Bearer + TokenReview + SAR              | Router binary                  |
| Controller metrics                | Bearer token via kube-rbac-proxy        | kube-rbac-proxy sidecar        |
| LLMISvc scheduler metrics         | Bearer token (tokenReview)              | Scheduler EPP                  |
| Webhook endpoints                 | Kubernetes API server mTLS              | kube-apiserver                 |

================================================================================
NETWORK POLICIES (ODH Overlay)
================================================================================

- Pod-to-pod traffic isolation via NetworkPolicies in ODH overlay
- Service mesh mTLS (Istio DestinationRules) for LLMInferenceService traffic
- OpenShift SCC for multi-node workloads and LocalModel agents
