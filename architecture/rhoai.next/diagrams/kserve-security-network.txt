================================================================================
  SECURITY FINDINGS / RISKS
================================================================================

  ⚠  FIPS Gap: localmodel-controller and localmodelnode-agent use CGO_ENABLED=0
     with static linking — these binaries will FAIL Konflux check-payload FIPS
     validation. All other Go components use CGO_ENABLED=1 with strictfipsruntime.

  ⚠  InsecureSkipVerify=true: PodMonitor TLS config for vLLM metrics scraping
     bypasses certificate validation. DestinationRule for scheduler TLS also uses
     InsecureSkipVerify. Pragmatic for self-signed certs but weakens security posture.

  ⚠  storage-config secret: Contains cloud storage credentials (AWS_ACCESS_KEY_ID,
     AWS_SECRET_ACCESS_KEY, etc.). Manual provisioning, no auto-rotation.
     Recommendation: Use workload identity (IRSA/WIF) where possible.

  ⚠  Self-signed TLS certificates: LLMInferenceService workload pods use self-signed
     certs (10-year validity, ECDSA P-256) for internal comms when OpenShift service-ca
     is not available. Long validity period increases exposure window.
