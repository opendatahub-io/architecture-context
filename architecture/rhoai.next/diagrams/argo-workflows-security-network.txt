================================================================================
  FIPS COMPLIANCE NOTES
================================================================================

workflow-controller:
  - Build: CGO_ENABLED=1 GOEXPERIMENT=strictfipsruntime -tags strictfipsruntime
  - Linking: Dynamic (uses OpenSSL via UBI9)
  - Single FIPS binary only

argoexec:
  - FIPS binary: /bin/argoexec-fips (CGO_ENABLED=1, dynamic)
  - Non-FIPS binary: /bin/argoexec (CGO_ENABLED=0, static) ← DEFAULT ENTRYPOINT
  - WARNING: FIPS compliance requires explicit command override to /bin/argoexec-fips

TLS Configuration:
  - Metrics endpoint: TLS 1.2+ default enabled, self-signed ECDSA P-256
  - InsecureSkipVerify=true in self-signed cert config (util/tls/tls.go:127)
  - No MD5/RC4/DES usage detected
