Security Considerations:
- ⚠️  Plaintext HTTP (28080/TCP) for progression metrics polling to training pods
  - Mitigated by: NetworkPolicy restricting ingress to controller namespace pods only
  - Risk: Metrics data traverses pod network unencrypted
- ⚠️  golang.org/x/crypto used for SSH key generation (MPI)
  - Mitigated by: strictfipsruntime routes to OpenSSL backend at runtime
- FIPS: CGO_ENABLED=1 GOEXPERIMENT=strictfipsruntime on UBI9 base
- HTTP/2 disabled by default (CVE-2023-44487 mitigation)
