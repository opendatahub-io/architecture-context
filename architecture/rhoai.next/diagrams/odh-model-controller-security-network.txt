  - Supports: Anonymous, UserDefined auth types via ISVC annotations
  - CEL objective expressions supported on gateway annotations
- Authorino TLS: EnvoyFilter injection for Gateway-Authorino communication
- PeerAuthentication: Not directly managed (depends on Service Mesh config)
- "Unmanaged" mode: Users can take ownership of auth resources
