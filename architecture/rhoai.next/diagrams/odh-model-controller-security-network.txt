================================================================================
  FIPS COMPLIANCE
================================================================================

Build-Time:
  - GOEXPERIMENT=strictfipsruntime, CGO_ENABLED=1, -tags strictfipsruntime
  - Dynamic linking (OpenSSL via UBI9 base image)
  - Both binaries: Dockerfile.konflux (controller), Containerfile.server.konflux (API)

Application-Level:
  - model-serving-api: FIPS-compliant TLS 1.2+ with restricted ECDHE+AES-GCM
    cipher suites and P-256/P-384 curves
  - Controller: default Go TLS via controller-runtime (FIPS via strictfipsruntime)
  - Certificate handling: cert-manager for webhooks, self-signed CA for Ray TLS,
    fsnotify-based hot reload for model-serving-api
  - No non-FIPS crypto detected (no MD5, RC4, DES usage)
