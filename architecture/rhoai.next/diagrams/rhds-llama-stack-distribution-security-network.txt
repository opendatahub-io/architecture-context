RBAC Summary:
  No RBAC resources defined in this repository.
  This is a container image, not an operator.
  RBAC for the pod service account is managed by the deploying operator
  (rhods-operator / RHOAI platform).

Authentication Summary:
  ⚠️  No application-level authentication on any endpoint.
  All /v1/* endpoints on port 8321 are open within the pod network.
  Production deployments MUST use:
    - kube-rbac-proxy sidecar (injected by RHOAI operator)
    - Network policies to restrict pod-to-pod access
    - Service mesh mTLS (if deployed within service mesh)

  - rhel-ai-private-index-auth (BOT_PAT)
    Type: Build-time secret
    Purpose: GitLab PAT for downloading pre-built wheel releases
    Provisioned: Manual / Konflux secret management
    Auto-Rotate: No

  - git_auth_secret
    Type: Build-time secret
    Purpose: Git authentication for Tekton pipeline runs
    Provisioned: Pipelines-as-Code

FIPS Compliance:
  - Base image: AIPCC CPU (RHEL 9.6 UBI) — FIPS-capable OpenSSL
  - Python: Uses system ssl/hashlib (delegates to OpenSSL)
  - TLS: System trust store (RHEL 9.6); custom CA via FMS_SSL_CERT_PATH
  - vLLM TLS verify: Configurable via VLLM_TLS_VERIFY
  - No non-FIPS crypto detected in repository source
