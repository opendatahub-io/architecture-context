RBAC Summary:
- No RBAC roles defined by this component
- Runs as container without direct Kubernetes API access
- TrustyAI LMEval provider may use K8s API (when TRUSTYAI_LMEVAL_USE_K8S=true)
  but RBAC is managed by the platform operator (rhods-operator)
- All API authentication is delegated to platform ingress
  (kube-rbac-proxy or OAuth proxy)

Service Mesh Configuration:
- Not directly configured by this component
- Platform operator manages service mesh integration
- In-cluster communication is plaintext HTTP on port 8321/TCP
- TLS termination occurs at platform ingress layer

Secrets (Environment Variables):
- VLLM_API_TOKEN (Bearer) - vLLM inference auth, provisioned by platform/user
- VLLM_EMBEDDING_API_TOKEN (Bearer) - vLLM embedding auth, provisioned by platform/user
- AWS_BEARER_TOKEN_BEDROCK (Bearer) - AWS Bedrock auth, 12h TTL, user-provisioned
- WATSONX_API_KEY (API Key) - IBM WatsonX auth, user-provisioned
- AZURE_API_KEY (API Key) - Azure OpenAI auth, user-provisioned
- OPENAI_API_KEY (API Key) - OpenAI auth, user-provisioned
- POSTGRES_PASSWORD (Password) - PostgreSQL auth, platform operator provisioned
- MILVUS_TOKEN (Token) - Remote Milvus auth, user-provisioned
- PGVECTOR_PASSWORD (Password) - pgvector PostgreSQL auth, user-provisioned
- QDRANT_API_KEY (API Key) - Qdrant auth, user-provisioned
- BRAVE_SEARCH_API_KEY (API Key) - Brave Search auth, user-provisioned
- TAVILY_SEARCH_API_KEY (API Key) - Tavily Search auth, user-provisioned
- AWS_ACCESS_KEY_ID (IAM) - S3 storage auth, user-provisioned
- AWS_SECRET_ACCESS_KEY (IAM) - S3 storage auth, user-provisioned
- KUBEFLOW_PIPELINES_TOKEN (Bearer) - Kubeflow auth, user-provisioned
- rhel-ai-private-index-auth/BOT_PAT (BuildSecret) - Konflux build only

Security Notes:
- No secrets auto-rotation configured for any credentials
- All secrets passed as environment variables (not mounted files)
- Provider activation is environment-variable-driven (conditional)
- Single PostgreSQL instance is a single point of failure
- In-cluster communication is unencrypted (plaintext HTTP)
- External service communication uses TLS 1.2+ minimum
