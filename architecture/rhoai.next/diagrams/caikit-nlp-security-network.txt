- No secrets defined by this library
- TGIS TLS certificates (ca_cert, client_cert, client_key) are file-path references
  configured in runtime_config.yaml, provisioned by the deployment
- HuggingFace Hub token (if used) is provided via environment variable

Container Security:
- Base image: registry.access.redhat.com/ubi9/ubi-minimal:latest
- Runs as: non-root (UID 1001, group: caikit)
- AIPCC base images: NOT used (migration recommended)
- Python deps: NOT from RHEL AI PyPI (uses default PyPI)

Security Controls:
- allow_downloads: false (default) — prevents arbitrary model downloads
- Path validation: TWO_DOTS_REGEX check prevents directory traversal in PEFT configs
- Thread-safe tokenization: per-thread tokenizer copies avoid race conditions

FIPS Compliance:
- OpenSSL available via UBI 9 base image
- Crypto delegated to grpcio (uses OpenSSL for TLS)
- No direct crypto operations in library source
- No hardcoded cipher suites
