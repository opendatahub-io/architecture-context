- TGIS CA cert (File: ca_cert_file) - TLS verification for TGIS, provisioned by platform operator, no auto-rotate
- TGIS client cert (File: client_cert_file) - mTLS client auth to TGIS, provisioned by platform operator, no auto-rotate
- TGIS client key (File: client_key_file) - mTLS client key for TGIS, provisioned by platform operator, no auto-rotate

- HTTP/gRPC endpoints (8080, 8085): Auth delegated to platform (kube-rbac-proxy / KServe / Gateway API)
- TGIS backend connection: mTLS (optional, configurable via cert files)
- HuggingFace Hub: Token auth (optional)
