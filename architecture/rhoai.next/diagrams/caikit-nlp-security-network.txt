RBAC Summary:
- No RBAC defined — caikit-nlp is a Python library, not a Kubernetes operator.
- Authentication is enforced externally by kube-rbac-proxy sidecars in RHOAI 3.x.

TLS Configuration:
- Server TLS (gRPC + HTTP):
  - RUNTIME_TLS_SERVER_KEY: TLS private key (file path, manual provisioning)
  - RUNTIME_TLS_SERVER_CERT: TLS certificate (file path, manual provisioning)
  - RUNTIME_TLS_CLIENT_CERT: Client cert for mutual TLS (file path, optional)
  - No auto-rotation configured

- TGIS Backend TLS (per-connection):
  - ca_cert_file: CA certificate for TLS verification
  - client_cert_file: Client certificate for mTLS
  - client_key_file: Client private key for mTLS
  - Configured via caikit config YAML per TGIS connection

Secrets:
- RUNTIME_TLS_SERVER_KEY (file path) - Server TLS private key, manual provisioning, no auto-rotate
- RUNTIME_TLS_SERVER_CERT (file path) - Server TLS certificate, manual provisioning, no auto-rotate
- RUNTIME_TLS_CLIENT_CERT (file path) - Client mTLS certificate, manual provisioning, no auto-rotate
- ca_cert_file (config) - TGIS backend CA cert, manual provisioning, no auto-rotate
- client_cert_file (config) - TGIS backend client cert, manual provisioning, no auto-rotate
- client_key_file (config) - TGIS backend client key, manual provisioning, no auto-rotate

Security Notes:
- Model downloads from HuggingFace Hub are DISABLED by default (allow_downloads=false)
- Distributed training traffic (29550/TCP) is PLAINTEXT — relies on network isolation
- No in-library authentication — all auth enforced by platform (kube-rbac-proxy sidecar)
- Container image: odh-caikit-nlp-rhel9 (built via Konflux)
