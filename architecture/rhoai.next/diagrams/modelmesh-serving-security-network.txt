NETWORK POLICIES:
================================================================================
Policy: modelmesh-controller
  Pod Selector: control-plane=modelmesh-controller
  Ingress:
    - Port 8443/TCP from any
  Purpose: Controller metrics access

Policy: modelmesh-runtimes
  Pod Selector: modelmesh-service label exists
  Ingress:
    - Port 8033/TCP from modelmesh-controller pods
    - Port 8033/TCP, 8008/TCP from any
    - Port 2112/TCP from any
  Purpose: Internal model-mesh communication, inference access, Prometheus metrics

Policy: modelmesh-webhook
  Pod Selector: control-plane=modelmesh-controller
  Ingress:
    - Port 9443/TCP from any
  Purpose: Webhook access from API server

RBAC SUMMARY:
================================================================================

  - modelmesh-controller (controller namespace)
  - modelmesh-serving-sa (runtime namespaces)

ClusterRole: modelmesh-controller-role
  - "" : configmaps, secrets, services, services/finalizers → full CRUD
  - "" : endpoints, persistentvolumeclaims → get, list, watch
  - "" : namespaces, namespaces/finalizers → get, list, patch, update, watch
  - apps : deployments, deployments/finalizers → full CRUD
  - monitoring.coreos.com : servicemonitors → full CRUD
  - serving.kserve.io : inferenceservices, predictors, servingruntimes,
                         clusterservingruntimes (+ finalizers, status) → full CRUD
  - autoscaling : horizontalpodautoscalers → full CRUD

Role: modelmesh-controller-leader-election-role (namespace-scoped)
  - coordination.k8s.io : leases → create, get, list, update
  - "" : configmaps → full CRUD
  - "" : events → create

Role: modelmesh-controller-restricted-scc-role (namespace-scoped)
  - security.openshift.io : securitycontextconstraints (restricted) → use
