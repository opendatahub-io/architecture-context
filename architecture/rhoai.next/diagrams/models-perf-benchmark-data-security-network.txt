================================================================================
 SECURITY NOTES
================================================================================

1. Shipped image runs as non-root (UID 1001)
2. No executables in shipped image (data files only)
3. No network services or endpoints in shipped image
4. UBI9-minimal base (minimal attack surface)
5. Dev tool api-client can disable TLS hostname verification
   (API_TLS_HOSTNAME_CHECK=false) -- intended for self-signed certs only
6. Repository contains confidential benchmark data (private repo)
7. FIPS: Not directly applicable (no crypto in shipped image)
