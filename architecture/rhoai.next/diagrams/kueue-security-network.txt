- kueue-webhook-server-cert (kubernetes.io/tls)
  - Purpose: TLS certificate for webhook server on port 9443
  - Provisioned by: Internal cert management (pkg/util/cert)
  - Auto-Rotate: Yes (controller-managed)

- controller-manager-metrics-token (kubernetes.io/service-account-token)
  - Purpose: Bearer token for Prometheus to scrape metrics
  - Provisioned by: Kubernetes SA token controller
  - Auto-Rotate: Yes

- MultiKueue kubeconfig secrets (Opaque, key: kubeconfig)
  - Purpose: Kubeconfig for connecting to remote worker clusters
  - Provisioned by: Manual / external provisioning
  - Auto-Rotate: No
  - Note: Disabled in RHOAI (MultiKueue=false)
