Headless Service:
- Service: model-mesh (ClusterIP: None, headless)
  - Port: 8033/TCP → grpc (8033)
  - Protocol: gRPC/HTTP2
  - Encryption: TLS (optional)
  - Purpose: Client-facing inference and model management API

NetworkPolicy:
- Defined in config/base/networkpolicy.yaml
- Restricts ingress/egress to necessary ports and services

Secrets:
- model-mesh-tls-secret (kubernetes.io/tls) - TLS cert and private key for gRPC and internal comm; Manual/cert-manager provisioned; No auto-rotate
- model-mesh-icd-etcd (Opaque) - etcd connection configuration (host, port, TLS certs); Manual/platform operator provisioned; No auto-rotate

RBAC:
- ModelMesh does not define its own RBAC resources
- RBAC is managed by the modelmesh-serving controller
