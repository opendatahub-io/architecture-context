Secrets:
  - codeflare-operator-webhook-server-cert (kubernetes.io/tls)
    Provisioned by: cert-controller (open-policy-agent/cert-controller)
    Auto-Rotate: Yes
    Purpose: Webhook server TLS certificate

  - {cluster}-proxy-tls-secret (kubernetes.io/tls) [per RayCluster]
    Provisioned by: OpenShift service-serving-cert-signer
    Auto-Rotate: Yes
    Purpose: OAuth proxy TLS serving certificate

  - {cluster}-oauth-config (Opaque) [per RayCluster]
    Provisioned by: codeflare-operator controller
    Auto-Rotate: No
    Purpose: OAuth cookie secret (SHA1 hash of cluster name + salt)
    Note: Uses crypto/sha1 (not FIPS-approved for new applications)

  - {cluster}-ca-secret (Opaque) [per RayCluster]
    Provisioned by: codeflare-operator controller (self-signed)
    Auto-Rotate: No
    Purpose: CA private key and certificate for mTLS between Ray head/workers
    Details: RSA 2048-bit, x509 self-signed, 1-year validity
    Note: Uses math/rand for CA serial number (non-cryptographic)

FIPS Compliance:
  Build-time: GOEXPERIMENT=strictfipsruntime, CGO_ENABLED=1, dynamic linking
  Risks: crypto/sha1 for OAuth cookie, math/rand for cert serial numbers
