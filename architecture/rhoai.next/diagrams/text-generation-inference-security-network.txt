- Name: inference-server

- Note: Headless for direct pod-to-pod communication (KServe routing)

RBAC Summary:
- No RBAC resources defined at TGIS level
- TGIS runs as non-root user (UID 2000, GID 0)
- Kubernetes RBAC handled at KServe/platform level

- TLS cert/key (optional, kubernetes.io/tls) - gRPC server TLS encryption

- HuggingFace auth token (optional, Opaque) - Private model access
  - Provisioned by: Manual
  - Auto-Rotate: No

- /metrics (HTTP/3000): Unauthenticated (Prometheus scraping)

  - kube-rbac-proxy sidecar
  - Gateway API HTTPRoute

- Container runs as non-root user 'tgis' (UID 2000)
- allowPrivilegeEscalation: false
- readOnlyRootFilesystem: supported via kustomize patch
- seccompProfile: RuntimeDefault
- No privilege escalation paths
- UDS communication (no network exposure for internal APIs)
