NetworkPolicy (when enableNetworkPolicy=true):
- Ingress restricted to service port from:
  1. All pods in the same namespace
  2. Operator namespace
  3. Explicitly listed namespaces (spec.network.allowedFrom.namespaces)
  4. Namespaces matching labels (spec.network.allowedFrom.namespaceLabels)
  5. OpenShift router namespaces (network.openshift.io/policy-group=ingress)
  6. Wildcard "*" in namespaces disables all restrictions
