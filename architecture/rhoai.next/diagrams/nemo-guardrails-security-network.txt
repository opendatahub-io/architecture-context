RBAC Summary:
- No Kubernetes ClusterRoles or RoleBindings defined in this repository
- RBAC is managed by the platform operator (RHOAI)
- No ServiceAccount-specific permissions defined

Secrets:
- LLM API keys (Opaque) - API keys for upstream LLM providers
  - OPENAI_API_KEY
  - ANTHROPIC_API_KEY
  - AZURE_OPENAI_API_KEY
  - COHERE_API_KEY
  - Provisioned by: Manual / platform operator
  - Auto-Rotate: No

Authentication & Authorization:
- No built-in authentication on the guardrails server
- All auth relies on upstream LLM provider authentication
- X-* headers forwarded from client to upstream LLM
- Auth tokens forwarded but sensitive tokens redacted in logs
- Authorization header forwarded for /v1/models endpoint

Security Notes:
- Server listens on HTTP (port 8000), TLS termination handled externally
- Open-source guardrail profile strips 15 proprietary modules at build time
- Container runs on UBI9 with Python 3.12
- Embedding models baked into image (air-gap compatible)
- ConfigMap-based config (config.yaml + rails.co) validated at startup
- Entrypoint validates required config files before server start
