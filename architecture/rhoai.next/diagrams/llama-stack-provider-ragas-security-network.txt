Secrets:
- kubeflow-pipelines-token (Opaque) - Bearer token for KFP API auth, manual provisioning, no auto-rotate
- aws-credentials (Opaque) - AWS_ACCESS_KEY_ID, AWS_SECRET_ACCESS_KEY, AWS_DEFAULT_REGION, manual provisioning, no auto-rotate
- llama-stack-inference-model-secret (Opaque) - INFERENCE_MODEL, VLLM_URL, VLLM_TLS_VERIFY, VLLM_API_TOKEN, manual provisioning, no auto-rotate
- S3 credentials secret (Opaque) - AWS credentials mounted into KFP pipeline pods, manual provisioning, no auto-rotate

RBAC Summary:
- No RBAC manifests defined in this repository
- Component runs within the Llama Stack server pod and inherits its RBAC
- Remote mode requires read access to ConfigMaps in redhat-ods-applications or opendatahub namespaces

Authentication & Authorization:
- Llama Stack eval API: No application-level auth; delegated to platform ingress (Gateway API / kube-rbac-proxy)
- KFP API: Bearer Token (OpenShift user token), enforced by KFP server
- S3 storage: AWS IAM credentials from Kubernetes Secret, enforced by AWS/MinIO IAM

FIPS Compliance:
- Pure Python library; delegates all TLS/crypto to platform OpenSSL via urllib3/requests
- UBI9 base image provides FIPS-validated OpenSSL
- No direct crypto imports detected
- WARNING: VLLM_TLS_VERIFY="false" in demo README - must not be used in production
