  Auth: Bearer Token (OpenShift OAuth or explicit token)
  Enforcement: KFP server
  Policy: Token must have pipeline submission permissions

- S3 result storage:
  Methods: PUT, GET
  Auth: AWS IAM credentials (from Kubernetes Secret)
  Enforcement: S3/MinIO server
  Policy: Write access to configured bucket/prefix

- Llama Stack inference (from KFP pods):
  Methods: POST
  Auth: None (configurable)
  Enforcement: Llama Stack server
  Policy: Network-level access; no auth enforced by default
