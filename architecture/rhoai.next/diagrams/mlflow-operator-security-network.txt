Authentication & Authorization:
- MLflow API (/mlflow/api/*, /mlflow/v1/*):
  Auth: Kubernetes RBAC via SelfSubjectAccessReview
  Enforcement: MLflow kubernetes-auth app plugin
  Policy: Checks user permissions on pseudo-resources (experiments, datasets,
          registeredmodels) in mlflow.kubeflow.org API group
  Security Middleware: Hardcoded enabled, CEL-validated (cannot be disabled)

- Operator metrics (/metrics):
  Auth: Bearer Token (ServiceAccount token)
  Enforcement: controller-runtime authn/authz filter
  Policy: Kubernetes TokenReview/SubjectAccessReview

- MLflow health (/mlflow/health):
  Auth: None (unauthenticated health check)
