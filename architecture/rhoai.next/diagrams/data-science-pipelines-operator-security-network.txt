- DSP components: ingress allowed from workbench pods (opendatahub.io/workbenches=true)
- DSP components: ingress allowed from pipeline pods (pipelines.kubeflow.org/v2_component=true)
- DSP components: ingress allowed from monitoring namespace
- MariaDB: ingress restricted to DSP component pods only
- Inter-component: restricted to DSP namespace labels
