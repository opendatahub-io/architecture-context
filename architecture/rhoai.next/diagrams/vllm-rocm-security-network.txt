================================================================================
  RBAC SUMMARY
================================================================================

This component defines NO RBAC resources. RBAC is managed by KServe and
the RHOAI platform operator.

- Container runs as: USER 2000 (non-root)
- Auth enforcement: kube-rbac-proxy sidecar (injected by KServe/platform)
- Platform RBAC: SubjectAccessReview via kube-rbac-proxy

================================================================================
  FIPS COMPLIANCE
================================================================================

- No TLS configured at container level (relies on platform ingress)
- Crypto libraries inherited from RHEL 9 UBI base (OpenSSL)
- Python ssl module backed by system OpenSSL (FIPS-capable)
- All crypto dependencies in RHAIIS base image (not this repo)

================================================================================
  BUILD SECURITY
================================================================================

- Build: Non-hermetic (hermetic: false in Tekton pipeline)
- No packages installed in this layer (thin wrapper)
- All dependencies inherited from AIPCC base image
- Architecture: linux/amd64 only (ROCm x86_64 constraint)
- Pipeline timeouts: 8h pipeline, 4h task (large GPU image)
