================================================================================
  Security Considerations
================================================================================

  ⚠️  ZMQ Event Stream (5557/TCP):
     - No authentication or encryption on KV-Events
     - Relies solely on network policy for pod-to-pod isolation
     - Risk: In multi-tenant deployments, malicious pods could inject
       or eavesdrop on cache locality information
     - Mitigation: Network policies restricting ZMQ traffic to trusted namespaces

  ⚠️  No FIPS Build Flags:
     - Go binary built without GOEXPERIMENT=strictfipsruntime
     - No CGO_ENABLED=1 for OpenSSL linkage
     - Python tokenizer inherits OpenSSL from AIPCC UBI base image
     - Only tokenizer service has Konflux Dockerfile

  Note: gRPC connections use insecure credentials for UDS (acceptable for
  pod-local communication via Unix Domain Socket with filesystem permissions)
