================================================================================
  SECURITY NOTES
================================================================================

1. NO TLS on any network-facing interface:
   - ZMQ PUB/SUB (5557/TCP) — no encryption, no auth
   - gRPC IndexerService (configurable TCP) — insecure credentials
   - Health probes (8082/TCP, 8000/TCP) — plain HTTP
   - Redis/Valkey (6379/TCP) — supports TLS but not configured by default
   - OTLP export (4317/TCP) — insecure flag set

2. UDS tokenizer is secured by kernel file permissions:
   - Shared via emptyDir volume mount
   - Only containers in the same pod can access

3. Pod discovery uses minimal RBAC (pods: get, list, watch only)

4. No NetworkPolicies defined — relies on namespace isolation

5. FNV-64a hashing used for block keys is NOT cryptographic
   (intentional — used for content addressing, not security)
