- Pod runs with ServiceAccount provided by PyTorchJob spec

- HF_TOKEN (Opaque, env var) - HuggingFace Hub authentication for gated model downloads
  - Provisioned by: User/ConfigMap
  - Auto-Rotate: No
- imagePullSecrets (kubernetes.io/dockerconfigjson) - Container registry credentials
  - Provisioned by: User
  - Auto-Rotate: No

Security Notes:
- ⚠️  NCCL/Gloo inter-process communication on port 29500/TCP is PLAINTEXT with NO authentication
  - Risk: Data in transit between training ranks is unencrypted
  - Mitigation: Typically pod-internal or within trusted cluster network
- Container image uses UBI9 base with CUDA 12.1 development layer
- Release image strips Python dev packages and dnf to minimize CVE surface
- No inbound network services exposed (batch job pattern)
- All egress connections use TLS except NCCL rendezvous
- Triton cache directories configured for qLoRA kernel compilation at runtime
