Security Notes:
- CLI tools (model-extractor, metadata-report) are NOT shipped in final container
- Final container is data-only (no executables beyond sleep)
- Container runs as non-root (UID 1001)
- No runtime network access required (all data pre-generated at build time)
- No RBAC permissions needed (data served via volume mount)
- Path traversal protection: filepath.Clean + rejection of absolute/.. paths
  in MCP server input file resolution
