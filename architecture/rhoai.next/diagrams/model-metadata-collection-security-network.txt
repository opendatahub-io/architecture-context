RBAC Summary:
=============
- No ClusterRoles defined
- No ClusterRoleBindings defined
- No Roles or RoleBindings defined
- Data container runs as non-root (UID 1001)
- No Kubernetes API access required

Secrets:
========
- HF_TOKEN (env var)
  - Type: Environment variable
  - Purpose: HuggingFace API authentication for private collections
  - Provisioned: Manual (developer)
  - Auto-Rotate: No
  - Context: Build-time only (CI/CD)

- git_auth_secret (Opaque, Tekton)
  - Purpose: Git authentication for Konflux pipeline checkout
  - Provisioned: Konflux/PaC
  - Auto-Rotate: Yes
  - Context: CI/CD pipeline only

- QUAY_USERNAME / QUAY_PASSWORD (Opaque, GitHub Actions)
  - Purpose: Container registry push credentials for upstream builds
  - Provisioned: Manual (GitHub Secrets)
  - Auto-Rotate: No
  - Context: CI/CD pipeline only

Security Posture:
=================
- ZERO runtime attack surface (data-only container, no binaries)
- No network listeners, no exposed ports
- No service mesh participation required
- Non-root execution (UID 1001)
- All external API calls are build-time only
- TLS 1.2+ for all outbound connections
- Go binary uses stdlib crypto (not FIPS-validated, but build-time only)
- UBI9-minimal base image (minimal OS packages)
