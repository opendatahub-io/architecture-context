VOLUME MOUNTS:
- /storage-config     ← Kubernetes Secret (Opaque)
                        Contains per-provider JSON credential files
                        Keys: S3 access_key_id/secret_access_key,
                              GCS private_key/client_email,
                              Azure client_id/client_secret/tenant_id
                        Provisioned by: User / ModelMesh controller
                        Auto-Rotate: No

- /models             ← EmptyDir or PVC
                        Root directory for downloaded model artifacts
                        Shared between puller, adapter, and runtime containers

- /pvc_mounts/{name}  ← PVC volumes (optional)
                        Pre-provisioned model storage
                        Symlinked into /models by puller

RBAC Summary:
- No RBAC roles or bindings
- This component does not interact with the Kubernetes API directly
- All operations are via gRPC (model lifecycle) and cloud storage APIs

Secrets:
- storage-config (Opaque) - Cloud storage credentials
  - S3: access_key_id, secret_access_key, region, bucket, endpoint_url
  - GCS: private_key, client_email, project_id, bucket
  - Azure: client_id, client_secret, tenant_id, container_name, storage_account
  - HTTP: custom headers, client certificate, client key, CA certificate
  - Provisioned By: User / ModelMesh controller
  - Auto-Rotate: No

FIPS Compliance:
- Build flags: GOEXPERIMENT=strictfipsruntime, CGO_ENABLED=1, -tags strictfipsruntime
- Dynamic linking to OpenSSL (UBI 9 ubi-minimal base)
- TLS uses crypto/tls with Go defaults (FIPS-restricted in strictfipsruntime mode)
- No custom cipher suites, no InsecureSkipVerify
- No non-FIPS crypto (no MD5, RC4, DES)
- Local gRPC connections use insecure credentials (expected - localhost only)

Network Policy Recommendations:
- ALLOW egress: TCP/443 to S3/GCS/Azure endpoints (TLS required)
- ALLOW egress: TCP/configurable to HTTP model endpoints (optional TLS)
- DENY all ingress from outside pod (all listeners are localhost-only)
- All intra-pod communication is plaintext on localhost (acceptable per pod boundary)
