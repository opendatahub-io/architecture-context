- Base Image: registry.redhat.io/rhai/base-image-cpu-rhel9
- Runs as: UID 1001 (non-root)

- MPLCONFIGDIR: /tmp/matplotlib (prevents writes to read-only paths)
- No privilege escalation
- No host network access
- No Kubernetes API access from container

Authentication & Authorization:
- No auth at container level
- All auth delegated to:
  1. Istio sidecar (mTLS, SPIFFE identity)
  2. KServe Gateway (RequestAuthentication, JWT validation)
  3. kube-rbac-proxy (if configured by platform)
- Endpoint-level policy enforcement at mesh/gateway layer, not in application
