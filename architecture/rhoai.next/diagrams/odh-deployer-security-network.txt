Security Notes:
  - /metrics endpoints on Prometheus and Blackbox Exporter bypass OAuth (skip-auth-regex: ^/metrics)
  - Blackbox exporter external config uses insecure_skip_verify: true (non-FIPS crypto risk)
  - Pod security: anyuid SCC granted broadly to application service accounts
  - Component metrics scraped over plaintext HTTP (8080/TCP) - no encryption or auth
  - sed-based template substitution for alerting config is fragile (silent failures)
