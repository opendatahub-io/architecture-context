NETWORK POLICIES:
- allow-kfp-to-llamastack (NetworkPolicy)
  - Allows KFP pods to reach Llama Stack on port 8321/TCP
  - Namespace: tai-garak-lls
  - Source: KFP pipeline pods
  - Destination: Llama Stack server pod

RBAC Summary:
- Role: lsd-garak-dspa-api-access
  - API Groups: datasciencepipelinesapplications.opendatahub.io
  - Resources: datasciencepipelinesapplications/api
  - Verbs: get, create

- RoleBinding: lsd-garak-dspa-api-access
  - Namespace: tai-garak-lls
  - Role: lsd-garak-dspa-api-access
  - ServiceAccount: llamastack-garak-distribution-sa

- RoleBinding: lsd-garak-pipeline-management
  - Namespace: tai-garak-lls
  - Role: ds-pipeline-dspa (predefined)
  - ServiceAccount: llamastack-garak-distribution-sa

Secrets:
- llama-stack-inference-model-secret (Opaque)
  - Keys: VLLM_API_TOKEN
  - Purpose: vLLM API authentication token
  - Provisioned By: Manual
  - Auto-Rotate: No

- postgres-secret (Opaque)
  - Keys: POSTGRES_DB, POSTGRES_USER, POSTGRES_PASSWORD
  - Purpose: PostgreSQL database credentials
  - Provisioned By: Manual
  - Auto-Rotate: No
  - WARNING: Reference manifests use default password "llamastack" — MUST change for production

- model-auth (Opaque) [KFP mode]
  - Keys: Per-role API keys (TARGET, SDG, JUDGE, ATTACKER, EVALUATOR)
  - Mount: /mnt/model-auth
  - Purpose: Model endpoint API keys
  - Provisioned By: Manual / External
  - Auto-Rotate: No

- Data Connection Secret (Opaque) [eval-hub mode]
  - Keys: access key, secret key, region, bucket, endpoint
  - Purpose: S3 credentials for artifact storage
  - Provisioned By: Manual / DSPA
  - Auto-Rotate: No

SECURITY CONSIDERATIONS:
- Reference deployment manifests contain placeholder credentials (dev-only)
- TLS verification disabled by default for vLLM in reference config
- PostgreSQL uses plaintext connections (no TLS)
- Llama Stack server HTTP is plaintext within cluster
- All Secrets are manual-provisioned with no auto-rotation
- Garak subprocess runs with process group isolation (os.setsid)
- Inline mode uses asyncio.Semaphore for concurrency control
