Secret Redaction:
  - All configs are redacted via redact_api_keys() before upload
  - Placeholder values (DUMMY, __FROM_ENV__, ***) replaced at runtime
  - Secrets never persist in S3, Files API, or OCI storage

FIPS Compliance:
  - UBI9 base image with system OpenSSL
  - TLS via httpx + Python ssl module (uses system OpenSSL)
  - boto3 uses system OpenSSL for S3 connections
  - No direct MD5, RC4, DES, or custom crypto detected
  - Configurable TLS verification via GARAK_TLS_VERIFY

Build Hermeticity:
  - RPM: rpms.lock.yaml (rpm-lockfile-prototype)
  - Python: requirements.txt (uv pip compile)
  - Rust: Cargo.lock (cargo, build-time only for tokenizer)
  - Hermeto prefetch: pip + Rust crate prefetching in Tekton
