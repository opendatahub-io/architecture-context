Security Notes:
- KFP mode uses __FROM_ENV__ placeholder in config — keys resolved at pod level, never in S3
- Config.json uploaded to S3 is redacted via redact_api_keys() before persistence
- TLS verification enabled by default; configurable via get_http_client_with_tls()
- FIPS: Python-based, inherits RHEL 9.6 UBI OpenSSL; no custom crypto detected
- No MD5, RC4, or non-FIPS algorithms in application code
- Build: Hermetic via Konflux Tekton (hermetic: true), Hermeto prefetch (pip, permissive mode)
