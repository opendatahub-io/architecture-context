================================================================================
SECURITY CONCERNS SUMMARY
================================================================================

CRITICAL:
  - fms-guardrails-orchestrator: rustls/ring backend NOT FIPS-validated
    Rust has no certified FIPS crypto module

HIGH:
  - batch-gateway: CGO_ENABLED=0 in Konflux Dockerfiles breaks Go FIPS
  - argo-workflows: Dual FIPS/non-FIPS binary defaults to non-FIPS
  - data-science-pipelines: Launcher ships dual FIPS/non-FIPS binary
  - fms-guardrails-orchestrator: NoVerifier bypasses TLS cert verification

MEDIUM:
  - gateway-api-inference-extension: InsecureSkipVerify=true for metrics
  - feast: insecureSkipVerify: true in ServiceMonitor
  - codeflare-sdk: verify_tls=False option available
  - ai-gateway-payload-processing: Plaintext HTTP to NeMo Guardrails
  - fms-hf-tuning: FSDP port 29500 is plaintext TCP
  - caikit-tgis-serving: Non-hermetic Konflux builds (hermetic: false)
  - guardrails-detectors: LLM Judge missing Konflux Dockerfile
  - gateway-api-inference-extension: No Konflux Dockerfiles on branch

LOW:
  - codeflare-operator: SHA1-based cookie secret generation

================================================================================
AUTH MECHANISMS
================================================================================

Kubernetes-native Auth:
  - eval-hub: TokenReview + SubjectAccessReview
  - data-science-pipelines: K8s SA token validation
  - feast: OIDC + K8s RBAC auto-access bridging

Proxy-based Auth:
  - codeflare-operator: OAuth proxy sidecar injection
  - data-science-pipelines-operator: kube-rbac-proxy sidecar
  - codeflare-sdk: OpenShift OAuth token refresh

Service Mesh Auth:
  - caikit-tgis-serving: Istio mTLS (PERMISSIVE on metrics)

API Key Auth:
  - ai-gateway-payload-processing: Injects API keys from Secrets

Header-based Auth:
  - batch-gateway: X-MaaS-Username header (no built-in auth!)

No Auth:
  - caikit, fms-guardrails-orchestrator, fms-hf-tuning,
    gateway-api-inference-extension, guardrails-detectors
    (delegate to deployment environment)
