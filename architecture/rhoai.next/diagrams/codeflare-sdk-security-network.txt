================================================================================
TLS / Crypto Configuration
================================================================================

  - Certificate Generation: RSA-2048, SHA-256 (via Python cryptography 40.0.2)
  - OpenSSL Backend: Inherits from notebook image (FIPS-compatible when host
    OpenSSL is FIPS-enabled)
  - Custom CA Trust: /etc/pki/tls/custom-certs/ca-bundle.crt or
    CF_SDK_CA_CERT_PATH env var
  - ODH Trusted CA: Mounted in Ray pods via odh-trusted-ca-bundle ConfigMap

  SECURITY RISKS:
  - verify_tls=False disables ALL TLS verification (InsecureSkipVerify)
  - urllib3.disable_warnings suppresses SSL warnings when skip_tls=True
  - CA secret auto-rotation: Not implemented
