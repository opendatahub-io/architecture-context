RBAC Summary:
- No RBAC resources defined in this repository
- RBAC is managed by KServe and the RHOAI operator
- kube-rbac-proxy sidecar enforces Kubernetes RBAC via SubjectAccessReview

Secrets:
- Model storage credentials (Opaque) - S3/MinIO access keys, provisioned by User/RHOAI Dashboard, no auto-rotation
- HuggingFace token (Opaque) - Hub auth for gated models, provisioned by User, no auto-rotation

Authentication & Authorization:
- /v1/* endpoints: Bearer Token (JWT) enforced by kube-rbac-proxy sidecar
- /health, /metrics: Unauthenticated (direct access, no sidecar interception)
- RHOAI 3.x: kube-rbac-proxy sidecar
- RHOAI 2.x: oauth-proxy sidecar (legacy)
