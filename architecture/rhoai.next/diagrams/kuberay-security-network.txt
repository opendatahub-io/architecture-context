RBAC Summary:
- ClusterRole: kuberay-operator
  - ray.io: rayclusters, rayjobs, rayservices (full CRUD + finalizers + status)
  - core: pods, services, endpoints, events, serviceaccounts, configmaps, secrets
  - batch: jobs (full CRUD)
  - apps: deployments (get, list, watch, update, patch)
  - networking.k8s.io: ingresses, ingressclasses, networkpolicies (full CRUD)
  - rbac.authorization.k8s.io: roles, rolebindings (CRUD); clusterroles, clusterrolebindings (read)
  - route.openshift.io: routes (full CRUD)
  - config.openshift.io: authentications, oauths (read)
  - operator.openshift.io: kubeapiservers (read)
  - authentication.k8s.io: tokenreviews (create)
  - authorization.k8s.io: subjectaccessreviews (create)
  - cert-manager.io: issuers, certificates (full CRUD)
  - gateway.networking.k8s.io: httproutes, referencegrants (full CRUD); gateways (read)
  - coordination.k8s.io: leases (get, list, create, update)

Authentication & Authorization:
- /ray/{ns}/{cluster}/* (HTTPRoute) → kube-rbac-proxy (8443) → SubjectAccessReview
  - Requires: verb=get, resource=services, resourceName={cluster}-head-svc
- Ray Dashboard (8265): Direct access or mTLS; NetworkPolicy limits source namespaces
- Webhook (9443): TLS client cert from Kubernetes API server

Secrets:
- kuberay-webhook-server-cert (kubernetes.io/tls) - Webhook TLS, OpenShift serving cert, auto-rotate
- ray-head-secret-{cluster} (kubernetes.io/tls) - Head mTLS cert, cert-manager, 90-day/15-day renewal
- ray-worker-secret-{cluster} (kubernetes.io/tls) - Worker mTLS cert, cert-manager, 90-day/15-day renewal
- {cluster}-ca-secret-{uid} (kubernetes.io/tls) - CA cert, cert-manager, 10-year/30-day renewal

Network Policies (when enabled via annotation):
- Intra-cluster: All ports allowed between cluster pods
- In-namespace: Ports 10001, 8265 from same namespace
- Operator namespace: Management access to head pod
- Monitoring namespace: Metrics port access
- Gateway namespace: Dashboard port (8265) access
- Port 8443 (kube-rbac-proxy): Open to all (auth enforced at application layer)
