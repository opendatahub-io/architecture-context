- OIDC CA certificate (Opaque) - Verify OIDC issuer TLS, manual, no auto-rotate
- Client CA certificate (Opaque) - Verify x509 client certs, manual/operator, dynamic reload

- CI: check-payload scan via GitHub Actions

- Token sanitization: Bearer tokens masked in log output (SanitizingFilter)
- Non-root: Runs as UID 65534
- HTTP/2 disable mode: Mitigates CVE-2023-44487 (Rapid Reset)
- ALPN enforcement: h2 + http/1.1 (or http/1.1 only in disable mode)
