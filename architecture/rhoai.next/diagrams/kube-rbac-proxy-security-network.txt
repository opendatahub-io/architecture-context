Sidecar Injection:
- Injected by: rhods-operator (platform operator)
- Pattern: Added as container in component Deployment pod spec
- Replaces: oauth-proxy sidecar (RHOAI 2.x)
- Typical port mapping: 8443 (kube-rbac-proxy) → application port (upstream)

Secrets:
- (TLS cert/key pair) (kubernetes.io/tls) - TLS termination, cert-manager/operator, auto-rotate via CertReloader
- (client CA bundle) (Opaque) - mTLS client cert CA, platform operator, dynamic reload via DynamicFileCAContent
- (upstream client cert) (kubernetes.io/tls) - mTLS to upstream, platform operator, no auto-rotate
- (OIDC CA bundle) (Opaque) - OIDC issuer TLS verification, platform operator, dynamic reload

Security Controls:
- Token Sanitization: Bearer tokens masked in klog output (SanitizingFilter)
- Container Security: Runs as UID 65534 (nobody), non-root
- TLS Configuration: Configurable min version (default TLS 1.2), cipher suite selection
- HTTP/2 Disable: Optional --http2-disable flag (CVE-2023-44487 mitigation)
- Path Bypass: --ignore-paths allows specific paths to skip authn/authz (e.g., health checks)
- Caching: TokenReview auth 2-min cache, SAR allow 5-min cache / deny 30-sec cache
