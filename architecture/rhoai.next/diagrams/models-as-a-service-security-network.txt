Security Policies:

  Gateway Default Deny:
    - AuthPolicy: gateway-default-auth → rejects all requests to ungoverned model paths (401)
    - TokenRateLimitPolicy: gateway-default-deny → 0 tokens/1m for model paths (defense-in-depth)

  Per-Model Policies (dynamically created):
    - AuthPolicy: CEL expressions extract identity from 3 auth sources
      (API key, K8s token, OIDC JWT) into unified user/group/subscription tuple
    - TokenRateLimitPolicy: per-subscription token rate limits
    - Authorization header stripped after auth (prevents credential exfiltration)

  NetworkPolicies:
    - maas-authorino-allow: Authorino pods → maas-api /internal/v1/*
    - maas-api-cleanup-restrict: CronJob pods → maas-api /internal/v1/api-keys/cleanup
    - monitoring: openshift-monitoring namespace → maas-api :9090

  Conflict Detection:
    - Monitors for non-MaaS AuthPolicies targeting MaaS-managed HTTPRoutes
    - Emits events and degrades MaaSAuthPolicy status on conflicts

  Build-time:
    - GOEXPERIMENT=strictfipsruntime, CGO_ENABLED=1 (dynamic linking)
    - UBI9 base images (OpenSSL included)

  Application-level:
    - TLS: configurable MinVersion (1.2 or 1.3)
    - Crypto: stdlib crypto/tls (FIPS via OpenSSL with strictfipsruntime)
    - API keys: SHA-256 hashing (FIPS-approved)
