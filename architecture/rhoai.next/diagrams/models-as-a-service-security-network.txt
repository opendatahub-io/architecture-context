FIPS Compliance:
- Build: GOEXPERIMENT=strictfipsruntime, CGO_ENABLED=1, dynamic linking
- Runtime: UBI9 base (OpenSSL FIPS provider), stdlib crypto/tls via OpenSSL
- TLS: Configurable MinVersion (default 1.2), supports 1.2 and 1.3
- Known exception: InsecureSkipVerify=true for cluster-internal model probing
- No MD5/RC4/DES usage detected

API Key Security Design:
- Format: sk-oai-{key_id}_{secret}
- Storage: SHA-256 hash with per-key salt (key_id + null byte delimiter)
- Plaintext shown once at creation, never stored
- Subscription binding immutable at creation (no swap attacks)
- Group snapshots frozen at creation (no privilege escalation)
- Auth failures return 404 (not 403) to prevent enumeration (IDOR protection)
- Constant-time comparison, async last_used_at update (timing side-channel mitigation)
