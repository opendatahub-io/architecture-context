- RBAC is managed by the TrustyAI LMEval Operator that creates the Job
- Pod runs as non-root user 65532:65532

Secrets:
- Model auth secret (model.auth.secret_ref) - Opaque
  Contains: OPENAI_API_KEY, HF_TOKEN (optional)
  Provisioned by: User / LMEvalJob CR
  Auto-Rotate: No

- OCI auth config (/tmp/.docker/config.json) - Docker config
  Contains: Registry authentication credentials
  Provisioned by: Platform / Operator
  Auto-Rotate: No

- S3 credentials - Environment variables
  Contains: AWS_ACCESS_KEY_ID, AWS_SECRET_ACCESS_KEY
  Provisioned by: User / ConfigMap
  Auto-Rotate: No

- CA certificates (verify_certificate path) - TLS CA bundle
  Contains: Custom CA certificate for inference endpoint TLS verification
  Provisioned by: Platform / Operator
  Auto-Rotate: No

Security Notes:
- Container runs as non-root (UID 65532, GID 65532)
- No inbound network connections (batch job, egress-only)
- Supports custom CA certificates for TLS verification
- Error messages sanitized to avoid leaking raw exception details
- Concurrent request count bounded to max 128 to prevent abuse
- Supports air-gapped/disconnected operation via offline mode
