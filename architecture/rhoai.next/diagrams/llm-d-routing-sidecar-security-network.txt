- All endpoints: TLS 1.2+ with configurable cipher suites

Security Notes:
- InsecureSkipVerify flags available for prefiller and decoder TLS connections
  (typical for internal/development traffic where cert management is challenging)
- SSRF allowlist tracks host IPs only (not ports) - any port on allowed host is permitted
- Component is experimental, noted for removal in upcoming llm-d architecture iteration
