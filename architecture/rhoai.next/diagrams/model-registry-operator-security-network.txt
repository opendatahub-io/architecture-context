Authentication & Authorization:
  Model Registry REST API:
    - Enforcement: kube-rbac-proxy sidecar (port 8443)
    - Mechanism: SubjectAccessReview (services/get on registry Service)
    - User must have "get" permission on the Service resource in registry namespace

  Model Catalog REST API:
    - Enforcement: kube-rbac-proxy sidecar (port 8443)
    - Mechanism: SubjectAccessReview (services/get on catalog Service)
    - User must have "get" permission on the model-catalog Service

  Operator Metrics:
    - Enforcement: controller-runtime metrics server (port 8443)
    - Mechanism: Bearer Token → TokenReview + SubjectAccessReview

  Webhooks:
    - Enforcement: Kubernetes API server
    - Mechanism: API server client cert (only API server can call)
