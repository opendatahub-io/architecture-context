Secrets:
  - {name}-kube-rbac-proxy (kubernetes.io/tls) — OpenShift serving cert, auto-rotate
  - {name}-postgres-credentials (Opaque) — auto-provisioned DB password (crypto/rand 16-byte)
  - model-catalog-postgres (Opaque) — catalog DB credentials
  - User-provided DB secrets (Opaque) — external PostgreSQL/MySQL credentials + SSL certs

NetworkPolicies:
  - Per-registry: Restricts ingress to model-registry pods from authorized sources
  - Per-database: Restricts ingress to PostgreSQL pod from model-registry pod only

TLS Security Profile:
  - Operator integrates with OpenShift APIServer TLS profile
  - Cipher suites and min TLS version fetched from apiservers.config.openshift.io
  - Automatic pod restart on profile change via TLS Security Profile Watcher

FIPS Compliance:
  - Build: CGO_ENABLED=1, GOEXPERIMENT=strictfipsruntime, -tags strictfipsruntime
  - Runtime: Dynamic linking to OpenSSL via UBI9 minimal base image
  - No non-FIPS crypto (MD5/RC4/DES) detected in source
