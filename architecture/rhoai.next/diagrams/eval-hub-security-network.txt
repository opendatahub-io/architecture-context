Multi-Tenancy:
- Tenant scoping via X-Tenant header → mapped to Kubernetes namespace
- SAR checks scoped to tenant namespace
- Each evaluation job runs in tenant's namespace

Secrets:
- TLS cert/key (kubernetes.io/tls) - HTTPS for eval-hub server, cert-manager, auto-rotate
- Database credentials (Opaque) - PostgreSQL connection URL, operator-provisioned
- MLflow token (Projected SA Token) - MLflow API auth, 3600s expiry, kubelet auto-rotate
- S3 credentials (Opaque) - AWS access key/secret/region/endpoint, user-provisioned
- OCI credentials (kubernetes.io/dockerconfigjson) - Docker registry auth, user-provisioned
- Model auth (Opaque) - Model endpoint authentication, user-provisioned
- Service CA (ConfigMap) - OpenShift service CA certificate, service-ca operator, auto-rotate
