- OCI credentials (kubernetes.io/dockerconfigjson) - OCI registry push auth, user-created, no auto-rotation
- Model auth (Opaque) - Model serving endpoint auth, user-created, no auto-rotation
- S3 credentials (Opaque) - AWS_ACCESS_KEY_ID, AWS_SECRET_ACCESS_KEY, etc., user-created, no auto-rotation
- Database credentials (Opaque) - PostgreSQL connection credentials, operator/user-created, no auto-rotation
