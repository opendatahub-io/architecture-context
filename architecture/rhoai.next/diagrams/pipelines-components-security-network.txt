Security Concerns:
⚠ SSL verification disabled as fallback in multiple components (boto3 verify=False, httpx verify=False, ssl.CERT_NONE)
⚠ Kubeflow Model Registry uses plaintext HTTP/8080 with no authentication
⚠ Secrets are user-provisioned with no auto-rotation
⚠ Custom CA bundles supported via REQUESTS_CA_BUNDLE/SSL_CERT_FILE but not proactively configured

FIPS Compliance:
- Build-time: Python-only (no Go binaries), UBI9 base images with system OpenSSL
- Application-level: Python ssl module uses system default context (FIPS via system OpenSSL on UBI9)
- Risk: SSL verify=False fallback bypasses FIPS-compliant TLS validation
