Security Notes:
- All external egress uses TLS 1.2+ (system OpenSSL from UBI9 base images)
- No ingress endpoints (ephemeral batch task pods only)
- Model Registry communication is plaintext HTTP (cluster-internal only)
- WARNING: documents_indexing component supports OGX_TLS_INSECURE=1 (dev-only TLS bypass)
  Source: components/data_processing/autorag/documents_indexing/component.py:66-82
  Risk: If enabled in production, disables certificate verification for OGX API calls
- FIPS: Python components inherit FIPS-compliant OpenSSL from UBI9/RHAI base images
- No custom TLS configuration; all crypto via system ssl module
- Build hermeticity via uv.lock (Python) + Hermeto prefetch (docling models for AutoRAG)
