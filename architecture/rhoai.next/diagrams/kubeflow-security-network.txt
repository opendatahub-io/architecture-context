  - Notebook access (/notebook/{ns}/{name}/*):
    Method: SubjectAccessReview (get notebooks in namespace)
    Enforcement: kube-rbac-proxy sidecar (8443/TCP)
    Policy: Users must have `get` on `notebooks.kubeflow.org` in notebook namespace

  - Webhook endpoints (/mutate-notebook-v1, /validate-notebook-v1):
    Method: mTLS (API server client cert)
    Enforcement: Webhook server TLS
    Policy: Only Kubernetes API server can call
