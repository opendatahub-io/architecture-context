NetworkPolicy (ODH/RHOAI overlays):
- Allows same-namespace pod communication on ports:
  • 7078/TCP (driver-rpc)
  • 7079/TCP (blockmanager)
  • 4040/TCP (spark-ui)
- Allows OpenShift ingress to reach Spark pods

RBAC Summary:
- ClusterRole: spark-operator-controller
  - API Groups: core, extensions, networking.k8s.io, sparkoperator.k8s.io, apiextensions.k8s.io
  - Resources: pods, configmaps, services, PVCs, events, nodes, ingresses, sparkapplications, scheduledsparkapplications, sparkconnects, CRDs
  - Verbs: create, delete, get, list, watch, update, patch

- ClusterRole: spark-operator-webhook
  - API Groups: admissionregistration.k8s.io, sparkoperator.k8s.io, core
  - Resources: mutatingwebhookconfigurations, validatingwebhookconfigurations, sparkapplications, scheduledsparkapplications, pods
  - Verbs: get, list, watch, update

- Aggregated Roles (user-facing):
  - spark-operator-sparkapplication-viewer-role: get, list, watch
  - spark-operator-sparkapplication-editor-role: full CRUD
  - spark-operator-scheduledsparkapplication-viewer-role: get, list, watch
  - spark-operator-scheduledsparkapplication-editor-role: full CRUD

Secrets:
- spark-operator-webhook-certs (Opaque)
  - Purpose: TLS certificate and CA for webhook server
  - Provisioned by: Internal certificate provider or cert-manager
  - Auto-Rotate: Yes (validated 180 days ahead, 10-year validity)
  - Contents: TLS cert, key, CA bundle

TLS Configuration:
- Webhook: Go default cipher suites (no hardcoded restrictions)
- HTTP/2: Explicitly DISABLED by default (NextProtos=["http/1.1"])
- Mitigation: CVE-2023-44487 (HTTP/2 Rapid Reset)
- Certificate: RSA keys, 10-year validity, proactive validation 180 days ahead

FIPS Compliance Notes:
- CGO_ENABLED=0 in source (static linking, does NOT use OpenSSL)
- Downstream Konflux may override to CGO_ENABLED=1 with GOEXPERIMENT=strictfipsruntime
- Non-cryptographic MD5 use for DNS name hash truncation (cosmetic, not a FIPS concern)
