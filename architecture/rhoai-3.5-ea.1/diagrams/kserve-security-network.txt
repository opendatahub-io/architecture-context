Authentication & Authorization:
===============================
  - InferenceService (raw, auth enabled): OAuth proxy / kube-rbac-proxy on 8443/TCP,
    SubjectAccessReview enforcement
  - LLMInferenceService (OCP): Kuadrant AuthPolicy at Gateway API level
  - Webhook endpoints: mTLS, CA-signed certificate validation by K8s API server
  - Controller /metrics: kube-rbac-proxy sidecar, RBAC-based access
  - Scheduler /metrics: Bearer Token via ServiceMonitor, kserve-metrics-reader-sa-secret
