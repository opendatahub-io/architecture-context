RBAC Summary:
- No Kubernetes RBAC. This is a Python library with no cluster-level permissions.

Secrets:
- No Kubernetes secrets managed by this component.
- Runtime credentials passed via environment variables or function arguments:
  - HF_TOKEN / HUGGING_FACE_HUB_TOKEN: HuggingFace Hub access (optional)
  - WANDB_API_KEY: Weights & Biases API key (optional)
  - MLflow credentials: Configurable per deployment (optional)

FIPS Compliance:
- No container images produced. FIPS compliance depends on runtime environment.
- TLS handled by Python's standard library (requests/urllib3) inheriting OS crypto.
- No direct cryptographic operations in application code.

Supply Chain Notes:
- Distributed via PyPI, bypasses Konflux build pipelines.
- litellm==1.63.11 pinned for supply chain security (GRPO extra).
- No lock files present; dependencies resolved at install time.

Distributed Training Network:
- torchrun uses dynamic TCP ports for NCCL communication.
- NO encryption on distributed training traffic (standard for NCCL).
- NO authentication on distributed training traffic.
- Traffic is intra-cluster only (same network segment).
- verl backend uses FSDP with multiple worker processes.
