Security Notes:
- No TLS on inference endpoints (8000/TCP) by default — TLS termination at Gateway
- CUDA image runs as non-root (UID 2000); CPU, HPU, ROCm images run as root
- No FIPS-specific build flags or cryptographic hardening
- PROMETHEUS_TLS_INSECURE_SKIP_VERIFY=true in WVA autoscaler configs
- RDMA/IB traffic between pods is unencrypted (performance-critical path)
- KV cache ZMQ events (5556/TCP) are unencrypted
- Gateway auth is provider-dependent and not configured by default
