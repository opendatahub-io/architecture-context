================================================================================
Security Observations
================================================================================

⚠  ZMQ transport between vLLM/SGLang pods and the indexer has NO encryption
   and NO authentication. KVEvents contain block hashes and metadata (not model
   weights or inference data), but cache topology information could leak which
   models are being served and usage patterns in multi-tenant environments.

⚠  Redis/Valkey connection has no authentication configured by default.
   TLS is available via rediss:// scheme but must be explicitly configured.

⚠  No GOEXPERIMENT=strictfipsruntime build flag set for the Go binary.
   The Go crypto stack does not use FIPS-validated primitives.

✓  UDS Tokenizer uses Unix Domain Sockets (pod-local), minimizing attack surface.
✓  Kubernetes API access uses mTLS with ServiceAccount tokens.
✓  UBI9 base image provides system-level OpenSSL (FIPS-capable).
✓  Python dependencies use hash-pinned requirements.txt for supply chain integrity.
