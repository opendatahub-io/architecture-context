NetworkPolicy (deployed by Helm chart):
  Ingress:
  - Allow: Any namespace → MLflow 8443/TCP (HTTPS)
    Reason: Gateway and Prometheus source namespaces vary

  Egress:
  - Allow: DNS 53/UDP, 53/TCP, 5353/UDP, 5353/TCP
  - Allow: HTTPS 443/TCP (external services)
  - Allow: Kubernetes API 6443/TCP
  - Allow: PostgreSQL 5432/TCP
  - Allow: MySQL 3306/TCP
  - Allow: S3-compatible 9000/TCP (MinIO)
  - Allow: S3-compatible 8333/TCP, 8334/TCP (SeaweedFS)
  - Extensible: networkPolicyAdditionalEgressRules in CR spec
