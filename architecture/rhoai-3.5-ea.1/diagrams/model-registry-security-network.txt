  - Prevents: Identity spoofing via kubeflow-userid header on internal calls

Security Notes:
- ⚠ Controller has InsecureSkipVerify: true hardcoded for Model Registry API calls
  (mitigated by Istio mTLS but bypasses TLS certificate validation)
- ⚠ CSI init container connects to Model Registry without auth or encryption
  (mitigated by being within same mesh namespace)
- ✅ Proxy server has proper FIPS-compliant TLS config (MinVersion TLS 1.2,
  FIPS-approved cipher suites only, insecure ciphers explicitly rejected)
- ✅ Build uses CGO_ENABLED=1, GOEXPERIMENT=strictfipsruntime for FIPS compliance
