Gateway Security Posture:
- Default-deny: gateway-default-auth AuthPolicy rejects requests to unconfigured model paths
- Default rate limit: gateway-default-deny TokenRateLimitPolicy → 0 tokens/min for unconfigured paths
- Per-model AuthPolicy: Generated from MaaSAuthPolicy (groups/users access control)
- Per-model TokenRateLimitPolicy: Generated from MaaSSubscription (rate limiting)
- Conflict detection: Controller warns on external AuthPolicies targeting MaaS HTTPRoutes
