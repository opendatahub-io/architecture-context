## Purpose
**Short**: Kubernetes operator that embeds and manages the Multi-Cluster Application Dispatcher (MCAD) and InstaScale controllers for distributed AI/ML workload scheduling and cluster auto-scaling on OpenShift.

**Detailed**: The CodeFlare Operator is a meta-operator that bundles two sub-controllers into a single deployment: the Multi-Cluster Application Dispatcher (MCAD) for intelligent batch job queuing and scheduling of distributed AI/ML workloads, and InstaScale for automatic cluster scaling by creating OpenShift MachineSets/MachinePools when additional compute is needed.

The operator runs as a single Go binary that starts the MCAD queue controller directly (via `mcad.NewJobController`) and optionally starts the InstaScale AppWrapper reconciler based on configuration. Configuration is stored in a ConfigMap (`codeflare-operator-config`) that the operator creates if it doesn't exist. The operator manages AppWrapper CRDs which wrap arbitrary Kubernetes resources (including RayClusters, PyTorch jobs, etc.) with scheduling policies, priority, and quota management. It is deployed as part of the OpenShift AI (RHOAI) and Open Data Hub (ODH) platforms, managed by the ODH/RHODS operator via the DataScienceCluster CR's `codeflare` component.

The operator does not define its own CRDs locally but imports and deploys CRDs from the upstream MCAD project (AppWrapper, QuotaSubtree, SchedulingSpec) via kustomize remote references.
