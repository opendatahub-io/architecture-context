RBAC Summary:
  None. This component does not interact with the Kubernetes API.

Secrets:
  - TLS certificate (via MM_TLS_KEY_CERT_PATH env)
    - Type: TLS certificate file
    - Purpose: HTTPS listener TLS termination
    - Provisioned By: ModelMesh Serving operator
    - Auto-Rotate: Unknown

  - TLS private key (via MM_TLS_PRIVATE_KEY_PATH env)
    - Type: TLS private key file
    - Purpose: HTTPS listener TLS termination
    - Provisioned By: ModelMesh Serving operator
    - Auto-Rotate: Unknown

Authentication & Authorization:
  - All REST endpoints: No authentication enforced
  - Auth is delegated to upstream ModelMesh Serving ingress layer
  - The rest-proxy trusts all traffic arriving at port 8008

Network Security Notes:
  - No Kubernetes Service defined (sidecar pattern)
  - Communication with model server is over localhost (pod-internal)
  - No external egress (all calls are pod-local to gRPC backend)
  - No RBAC permissions required (no K8s API interaction)
  - Runs as non-root user (UID 2000)
  - Statically linked binary (CGO_ENABLED=0)
  - Minimal base image (ubi-micro)
