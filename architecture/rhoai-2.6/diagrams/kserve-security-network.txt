DEPLOYMENT MODES
═══════════════════════════════════════════════════════════════════════════

1. Serverless Mode (default):
   ├─ Uses: Knative Serving
   ├─ Features: Scale-to-zero, request-based autoscaling
   ├─ Network: Knative Activator → Model Server Pod
   └─ Dependencies: Knative Serving 0.26+, Istio 1.15+

2. RawDeployment Mode:
   ├─ Uses: Kubernetes Deployment
   ├─ Features: No scale-to-zero, manual HPA configuration
   ├─ Network: Direct ClusterIP Service → Model Server Pod
   └─ Dependencies: Kubernetes 1.23+ only

3. ModelMesh Mode (separate component):
   ├─ Uses: ModelMesh Serving
   ├─ Features: High-density multi-model serving
   └─ Network: ModelMesh router → Multiple model servers
