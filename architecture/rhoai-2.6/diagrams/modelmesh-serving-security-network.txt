Network Policies:
================================================================================
- modelmesh-controller:
  - Pod Selector: control-plane: modelmesh-controller
  - Ingress: port 8443/TCP from any
  - Purpose: Controller metrics access

- modelmesh-runtimes:
  - Pod Selector: modelmesh-service: Exists
  - Ingress: port 8033/TCP + 8080/TCP from modelmesh pods
  - Ingress: port 8033/TCP + 8008/TCP from any
  - Ingress: port 2112/TCP from any
  - Purpose: Runtime inference, internal communication, Prometheus metrics

- modelmesh-webhook:
  - Pod Selector: control-plane: modelmesh-controller
  - Ingress: port 9443/TCP from any
  - Purpose: Webhook admission requests from API server

Authentication & Authorization:
================================================================================
- gRPC inference (8033/TCP):
  - Auth: mTLS client certificates (optional, configurable via tls.clientAuth)
  - Enforcement: ModelMesh container
  - Policy: TLS client auth config in model-serving-config ConfigMap

- HTTPS inference (8443/TCP):
  - Auth: OpenShift oauth-proxy with SAR (SubjectAccessReview)
  - Enforcement: oauth-proxy sidecar
  - Policy: Requires "get" on "services" in runtime namespace

- REST inference (configurable):
  - Auth: Depends on TLS config
  - Enforcement: REST proxy container
  - Policy: Proxies to ModelMesh gRPC, inherits TLS settings

- Webhook (9443/TCP):
  - Auth: Kubernetes API server TLS
  - Enforcement: Kubernetes admission controller
  - Policy: ValidatingWebhookConfiguration

- Prometheus metrics (2112/TCP):
  - Auth: None (open within NetworkPolicy)
  - Enforcement: NetworkPolicy
  - Policy: Restricted to modelmesh-labeled pods
