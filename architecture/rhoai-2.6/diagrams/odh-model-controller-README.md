## Component Overview

The **odh-model-controller** is a Kubernetes operator that extends KServe and ModelMesh InferenceService with OpenShift-specific capabilities:

- **External Access**: Creates OpenShift Routes (TLS Passthrough for KServe, Edge/Reencrypt for ModelMesh)
- **Service Mesh Integration**: Manages ServiceMeshMemberRoll, PeerAuthentication (mTLS with metric port exceptions), and Telemetry
- **Monitoring**: Provisions ServiceMonitors, PodMonitors, and RoleBindings for Prometheus
- **Network Security**: Creates NetworkPolicies for monitoring, ingress, and ODH cross-namespace traffic
- **Storage Configuration**: Aggregates ODH-managed data connection secrets into unified storage-config
- **Dual Mode Support**: Handles both KServe (via Istio) and ModelMesh deployment modes
