================================================================================
NETWORK POLICIES
================================================================================

Policy: model-mesh
  Pod Selector: app=model-mesh, component=model-mesh

  Ingress Rules:
  - Allow 8033/TCP, 8080/TCP from model-mesh pods (intra-cluster)
  - Allow 8033/TCP from any source (inference clients)
  - Allow 2112/TCP from any source (Prometheus scraping)

================================================================================
TLS CONFIGURATION
================================================================================

Channel                      | TLS Mode           | Certificate Source
-----------------------------|--------------------|----------------------------
External gRPC API (8033)     | Optional TLS/mTLS  | User-provided (k8s secret)
Internal inter-pod (8080)    | Optional TLS/mTLS  | User-provided (same certs)
Prometheus metrics (2112)    | HTTPS self-signed   | Auto-generated (BouncyCastle)
Sidecar-to-runtime (8085)   | plaintext           | N/A (localhost/UDS)

Client Auth Modes (MM_TLS_CLIENT_AUTH):
  - NONE: No client certificate required
  - REQUIRE: Client certificate required (full mTLS)
  - OPTIONAL: Client certificate optional
