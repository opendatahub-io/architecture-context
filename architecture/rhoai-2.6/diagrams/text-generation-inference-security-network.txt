- allowPrivilegeEscalation: false

- capabilities.drop: ALL

TLS Configuration:
- Server TLS: Optional, configured via TLS_CERT_PATH + TLS_KEY_PATH
- Client mTLS: Optional, configured via TLS_CLIENT_CA_CERT_PATH
- When enabled: tonic gRPC server validates client certificates against CA

Secrets:
- TLS certificates (kubernetes.io/tls) - gRPC server TLS + optional client mTLS, User/cert-manager provisioned
- HuggingFace token (Opaque) - Model download auth (build-time only), User provisioned, no auto-rotation

Network Services:
- inference-server: ClusterIP (Headless), 8033/TCP gRPC, Optional TLS/mTLS, Internal only

Internal Process Communication:
- Router → Shards: Unix Domain Socket (/tmp/text-generation-server-{rank}), gRPC, no encryption
- Launcher → Router/Shards: Process management (stdout/stderr pipes)
- Shard ↔ Shard: NCCL TCP (port 29500), no encryption (pod-local)

Authentication Summary:
- gRPC 8033: Optional mTLS (client certificates validated by tonic when TLS_CLIENT_CA_CERT_PATH set)
- HTTP 3000: No authentication (/health and /metrics are open)
- No RBAC (not a Kubernetes operator)
- No CRDs (standalone inference service)
