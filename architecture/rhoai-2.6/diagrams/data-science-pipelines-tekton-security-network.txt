Authentication & Authorization:
==========================================================================

- API Server (/apis/v1beta1/*):
  - Auth: Bearer Token (API Key in header)
  - Enforcement: API Server interceptor
  - Multi-user: SubjectAccessReview

- UI (ds-pipeline-ui):
  - Auth: OpenShift OAuth Proxy
  - Enforcement: oauth-proxy sidecar
  - Policy: OpenShift RBAC (services/get on ml-pipeline-ui)

- Metrics (/metrics):
  - Auth: None (unauthenticated)

==========================================================================
Security Notes:
==========================================================================

- No NetworkPolicies found in manifests
- No Service Mesh/Istio configuration in ODH manifests
- Internal service-to-service communication is plaintext (no mTLS)
- MySQL and MinIO connections are plaintext (configurable)
- Only external-facing endpoint (UI Route) uses TLS
- persistence agent has cluster-wide access to PipelineRuns/TaskRuns
- pipeline-runner SA has broad permissions (wildcard on kubeflow.org/*)
