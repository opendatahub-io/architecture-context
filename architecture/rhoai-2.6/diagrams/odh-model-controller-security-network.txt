Security Notes
════════════════════════════════════════════════════════════════════════════

1. Controller Pod Security:
   - runAsNonRoot: true
   - allowPrivilegeEscalation: false
   - No privileged containers

2. TLS/Encryption:
   - External ingress: TLS 1.2+ (Edge termination at OpenShift Route)
   - Control plane: TLS 1.3 (Kubernetes API communication)
   - Model storage: TLS 1.2+ (S3/object storage)
   - Service mesh: Optional mTLS (PERMISSIVE or STRICT mode)

3. Authentication:
   - Controller: ServiceAccount token authentication
   - Prometheus: ServiceAccount token via RoleBinding
   - S3 access: AWS Signature v4 (IAM or access keys)
   - InferenceService: Optional Bearer Token (configured per namespace)

4. Recent CVE Fixes:
   - CVE-2022-21698 and CVE-2023-45142: otelhttp dependency
   - CVE-2023-48795: golang.org/x/crypto (Terrapin attack)
   - Stack-based buffer overflow: protobuf dependency

5. Network Isolation:
   - NetworkPolicies restrict ingress to:
     - OpenShift monitoring (metrics scraping)
     - OpenShift ingress (router traffic)
     - ODH application namespaces (labeled)
   - Default deny for unlabeled namespaces
