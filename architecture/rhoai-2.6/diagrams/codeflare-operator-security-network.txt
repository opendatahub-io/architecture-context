Security Notes:
- No external ingress exposed
- Metrics endpoint (8080/TCP) is HTTP without auth (cluster-internal only)
- Health probes (8081/TCP) are HTTP without auth (pod-internal only)
- All Kubernetes API communication uses TLS 1.2+ with ServiceAccount tokens
- Broad RBAC permissions: machine.openshift.io/* grants full access to Machine API
- pods/binding permission allows scheduler-level pod placement
- tokenreviews/subjectaccessreviews allow authentication/authorization delegation
