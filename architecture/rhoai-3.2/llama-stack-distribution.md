## Purpose
**Short**: A containerized distribution of Meta's Llama Stack providing a unified API server for inference, agents, evaluation, safety, and RAG capabilities within Red Hat OpenShift AI.

**Detailed**: The Llama Stack Distribution repository builds an OCI-compliant container image that packages the Meta Llama Stack server with a curated set of providers for inference, vector I/O, safety, evaluation, scoring, agents, tool runtimes, datasets, and file storage. The image is designed for deployment within the Red Hat OpenShift AI (RHOAI) platform and ships with pre-configured providers for remote inference backends (vLLM, AWS Bedrock, Azure, Vertex AI, OpenAI, WatsonX), vector databases (Milvus, FAISS, pgvector), safety (TrustyAI FMS), evaluation (TrustyAI LM Eval, RAGAS), and various tool runtimes (Brave Search, Tavily, MCP, RAG).

The distribution uses PostgreSQL as its default persistent storage backend (both key-value and SQL stores), supports environment-variable-driven provider activation for flexible deployment configurations, and ships a pre-downloaded embedding model (ibm-granite/granite-embedding-125m-english). The container is built via Konflux CI/CD pipelines targeting RHEL 9 UBI base images, with multi-architecture support for x86_64 and arm64.
