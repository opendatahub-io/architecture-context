## Purpose
**Short**: A Kubernetes operator that manages AI/ML trustworthiness, evaluation, and guardrails services on OpenShift AI.

**Detailed**: The TrustyAI Service Operator provides a unified control plane for deploying and managing four distinct AI trustworthiness services within Red Hat OpenShift AI (RHOAI). It operates as a single binary with selectable service controllers enabled via the `--enable-services` flag.

The operator manages: (1) **TrustyAI Service (TAS)** - an explainability and bias monitoring service for ML models served via ModelMesh or KServe, supporting PVC or database-backed storage; (2) **LMEval (LMES)** - a language model evaluation framework that creates Pods to run evaluation jobs against LLM models using the Unitxt framework, with optional Kueue integration for job queuing; (3) **Guardrails Orchestrator (GORCH)** - an orchestration layer for guardrails that coordinates content detection across multiple detector services with auto-configuration from KServe InferenceServices; and (4) **NeMo Guardrails** - NVIDIA NeMo Guardrails server deployment for programmable guardrails.

Each controller dynamically creates Deployments, Services, OpenShift Routes, ConfigMaps, and kube-rbac-proxy sidecars at runtime based on Custom Resource specifications. The operator uses OpenShift serving certificate annotations for automatic TLS provisioning.

| LMEval Controller (LMES) | Go Controller | Reconciles LMEvalJob CRs; creates Pods with driver + eval containers for language model evaluation; manages job lifecycle (New/Scheduled/Running/Complete/Cancelled) |

| Guardrails Orchestrator Controller (GORCH) | Go Controller | Reconciles GuardrailsOrchestrator CRs; creates multi-container Deployments (orchestrator + gateway + built-in detectors + kube-rbac-proxy sidecars), Services, OpenShift Routes, ServiceMonitors |
| NeMo Guardrails Controller | Go Controller | Reconciles NemoGuardrails CRs; creates Deployments with optional kube-rbac-proxy sidecar, Services, OpenShift Routes, CA bundle ConfigMaps |
| Operator Manager | Go Binary | controller-runtime manager; serves metrics on :8080, health probes on :8081; leader election via Lease |
