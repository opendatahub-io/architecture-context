## Purpose
**Short**: Central build configuration repository that defines the RHOAI 3.2 operator bundle, FBC catalog fragments, container image references, and CI/CD pipelines for the Red Hat OpenShift AI product.

**Detailed**: RHOAI-Build-Config is the "source of truth" for the Red Hat OpenShift AI (RHOAI) 3.2 product build and release pipeline. It does not contain operator or application source code. Instead, it aggregates and pins the exact container image references (with SHA256 digests) for all 78 shipped container images that make up the RHOAI product. These references are assembled into an OLM (Operator Lifecycle Manager) operator bundle and per-OCP-version FBC (File-Based Catalog) fragments.

The repository contains the ClusterServiceVersion (CSV) manifest that defines how the rhods-operator is installed via OLM, including all CRD ownership declarations, RBAC rules, and related image references needed for disconnected/air-gapped installations. It also houses Tekton PipelineRun definitions for Konflux-based CI/CD, GitHub Actions workflows for automated bundle processing, Enterprise Contract integration tests, and release artifacts for stage/production releases. The build configuration tracks upstream source repositories and their pinned commit SHAs for every component image, enabling full provenance tracing from product release back to source commits.

### Ingress

This repository defines the operator bundle (not the operator itself). The operator deployed by this bundle manages Gateway API resources (HTTPRoute, Gateway, GatewayClass) and OpenShift Routes for platform ingress. See the rhods-operator repository for controller-level ingress architecture details.

The CSV RBAC declarations show the operator manages:
- `gateway.networking.k8s.io`: GatewayClasses, Gateways, HTTPRoutes (RHOAI 3.x primary ingress)
- `route.openshift.io`: Routes, Routes/custom-host (legacy/redirect ingress)
- `networking.istio.io`: DestinationRules, EnvoyFilters (service mesh integration)
- `networking.k8s.io`: Ingresses, NetworkPolicies
