## Purpose
**Short**: TrustyAI is a Java library and containerized REST service providing AI fairness metrics, model explainability algorithms, and data drift detection for ML models served via ModelMesh and KServe.

**Detailed**: TrustyAI Explainability is a multi-module Java project that provides Explainable AI (XAI) capabilities for the Red Hat OpenShift AI (RHOAI) platform. The core library implements fairness metrics (Statistical Parity Difference, Disparate Impact Ratio), explainability algorithms (LIME, SHAP, Counterfactual, Partial Dependence Plots, Time-Series Saliency), and data drift detectors (KS Test, Fourier MMD, Meanshift).

The explainability-service module packages these capabilities as a Quarkus-based REST service that integrates with ModelMesh and KServe inference serving platforms. It consumes inference payloads (both via REST endpoint for ModelMesh and via Knative CloudEvents for KServe), stores them in PVC-backed CSV storage or a database (MariaDB/MySQL), and provides scheduled Prometheus metric publishing for continuous monitoring. The service exposes a comprehensive REST API for on-demand metric calculations, explainability requests, and service metadata management.

The project also includes the explainability-connectors module which provides gRPC client connectivity to KServe V2 Inference Protocol endpoints, enabling the explainer endpoints to query model servers directly for generating explanations.
