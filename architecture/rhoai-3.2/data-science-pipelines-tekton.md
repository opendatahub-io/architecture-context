## Purpose
**Short**: Kubeflow Pipelines backend and frontend services adapted to use Tekton as the workflow execution engine, enabling ML pipeline orchestration on OpenShift.

**Detailed**: Data Science Pipelines Tekton (DSP-Tekton) is a fork/adaptation of Kubeflow Pipelines that replaces the Argo Workflows execution backend with Tekton Pipelines. It provides the complete pipeline management platform including an API server for pipeline CRUD operations, a scheduling controller for recurring pipeline runs via the ScheduledWorkflow CRD, a persistence agent for synchronizing Tekton PipelineRun status back to the API server, and a viewer controller for managing visualization deployments.

The component is based on Kubeflow Pipelines 1.8.4 and Tekton >= 0.41.0. It includes a web UI frontend (React/TypeScript) for pipeline management with an OAuth proxy sidecar for OpenShift authentication, a visualization server for rendering pipeline output visualizations, and ML Metadata (MLMD) services (metadata gRPC server + Envoy proxy) for artifact and execution lineage tracking. The pipeline compiler SDK translates KFP DSL pipelines into Tekton PipelineRun YAML. A custom Tekton CRD (PipelineLoop) extends Tekton with looping/iteration capabilities needed for ML workflows.
