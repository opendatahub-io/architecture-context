Authentication & Authorization:
===============================

  Ray Dashboard API:
    - Methods: GET, POST
    - Auth: Bearer Token (OpenShift SA token)
    - Enforcement: kube-rbac-proxy / Route auth
    - Policy: Per-cluster RBAC

  Prometheus API:
    - Methods: GET
    - Auth: Bearer Token (OpenShift SA token)
    - Enforcement: OAuth proxy
    - Policy: openshift-monitoring RBAC

  Kubernetes API:
    - Methods: All
    - Auth: Bearer Token / kubeconfig
    - Enforcement: kube-apiserver
    - Policy: Cluster RBAC

    - 8265/TCP   Ray Dashboard API (head node)
    - 10001/TCP  Ray Client port (head-to-workers)

  External-facing (TLS):

    - 443/TCP    S3 Storage (TLS 1.2+, AWS IAM)

    - 443/TCP    Container registries (TLS 1.2+)
