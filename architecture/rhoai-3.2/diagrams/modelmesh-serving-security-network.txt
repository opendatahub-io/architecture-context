Network Policies:
- modelmesh-controller: Allow 8443/TCP from any
- modelmesh-runtimes: Allow 8033,8080/TCP from modelmesh pods;
                      Allow 8033,8008/TCP from any;
                      Allow 2112/TCP from any
- modelmesh-webhook: Allow 9443/TCP from any

Secrets:
- model-serving-etcd (Opaque) - etcd credentials (host, port, userid, password), no auto-rotate
- storage-config (Opaque) - S3/storage backend credentials per model entry, no auto-rotate
- model-serving-proxy-tls (kubernetes.io/tls) - oauth-proxy TLS cert, OpenShift serving cert, auto-rotate
- TLS Secret (kubernetes.io/tls) - ModelMesh gRPC TLS cert/key (configurable), no auto-rotate

Authentication & Authorization:
- gRPC inference (8033): Optional mTLS via TLS.ClientAuth config
- REST inference (8443): OpenShift OAuth Bearer token, SAR: services/get in namespace
- Webhook (9443): K8s API server client certificate
- Controller metrics (8080): None (NetworkPolicy restricted)
- pprof debug (9999): Basic Auth (etcd credentials), optional via ENABLE_PPROF
