RBAC Summary:
================================================================================

  This component does not require Kubernetes RBAC permissions.
  It runs as a sidecar container and reads storage credentials
  from pre-mounted Secrets via the filesystem.

================================================================================
Secrets:
================================================================================

  Secret: storage-config
    - Type: Opaque
    - Mount Path: /storage-config
    - Purpose: Per-key JSON with storage credentials
      - access_key_id, secret_access_key (S3)
      - endpoint_url, region, bucket
      - certificate (custom CA)
    - Provisioned By: User/Admin
    - Auto-Rotate: No

================================================================================
FIPS Compliance:
================================================================================

  - Go binaries compiled with: -tags strictfipsruntime
  - Build flag: GOEXPERIMENT=strictfipsruntime
  - Ensures FIPS-compliant cryptographic operations at runtime
  - Base image: UBI 9 Minimal (FIPS-capable)

================================================================================
Network Security Notes:
================================================================================

  - NO ingress traffic (sidecar only, no Kubernetes Services)
  - NO external gRPC exposure (all listeners on localhost/loopback)
  - Egress is TLS-encrypted to cloud storage (S3, Azure, GCS)
  - Pod network isolation provides the trust boundary
  - No service mesh sidecar injection (internal pod communication)
  - Credentials never transit the network (mounted as filesystem)
