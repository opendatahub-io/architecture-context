Secrets:
========
- teacher-secret (Opaque) - Teacher model endpoint, model_name, api_token | User-provisioned | No auto-rotate
- judge-secret (Opaque) - Judge model endpoint, model_name, api_token | User-provisioned | No auto-rotate
- taxonomy-repo-secret (basic-auth/ssh-auth) - Git credentials for taxonomy | User-provisioned | No auto-rotate
- oci-registry-push-secret (dockerconfigjson) - OCI registry push creds | User-provisioned | No auto-rotate
- hugging-face-token (Opaque) - HuggingFace API token for model downloads | User-provisioned | No auto-rotate

Authentication & Authorization:
================================
- Kubernetes API: Bearer Token (SA JWT) -> kube-apiserver -> RBAC
- Teacher Model: Bearer Token (api_token) -> KServe/Model server -> Application-level
- Judge Model: Bearer Token (api_token) -> KServe/Model server -> Application-level
- OCI Registry: Docker auth (.dockerconfigjson) -> Registry server -> Registry auth
- Model Registry: Bearer Token (SA JWT) -> Model Registry API -> RBAC (registry-user role)
- Git Repository: Basic-auth or SSH key -> Git server -> Git auth

Notes:
======
- Pipeline does NOT expose external ingress; triggered via DSPA API server
- PyTorchJob pods have Istio sidecar injection DISABLED for NCCL performance
- vLLM eval server runs localhost-only within eval pods (no network exposure)
- Service mesh (OSSM v2) required by KServe for traffic management
- Red Hat Authorino required for RHOAI authorization
