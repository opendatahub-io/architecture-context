Authentication & Authorization:
  - HTTP API (8080): Auth delegated to platform (kube-rbac-proxy or Istio)
  - gRPC API (8085): Auth delegated to platform (kube-rbac-proxy or Istio)
  - TGIS backend: mTLS (optional, certificate-based)

  - Base image: registry.access.redhat.com/ubi9/ubi-minimal:latest
  - Model downloads disabled by default (allow_downloads: false)
  - No privileged capabilities required
