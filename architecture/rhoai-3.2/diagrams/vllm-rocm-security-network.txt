RBAC Summary:
================================================================================
- RBAC is NOT defined by this repository
- Managed by platform components:
  - KServe controller: ServiceAccount for model serving pods
  - kube-rbac-proxy sidecars: RHOAI 3.x auth enforcement via SubjectAccessReview
  - Platform Gateway API: Authorization policies

Container Security:
================================================================================
- Run-as User: 2000 (non-root)
- Base Image: registry.redhat.io/rhaiis/vllm-rocm-rhel9 (Red Hat certified, vulnerability-scanned)
- FIPS Compliance: Enabled (platform-level)
- Security Hardening:
  - HTTP header count and size limits (v0.10.0.0-29)
