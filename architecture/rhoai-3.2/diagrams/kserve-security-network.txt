Deployment Modes:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. Serverless Mode (Knative + Istio):
   - Autoscaling: Knative KPA (request concurrency based)
   - Scale to zero: Yes
   - Routing: Istio VirtualService
   - mTLS: Enforced via Istio sidecar

2. RawDeployment Mode (Standard K8s):
   - Autoscaling: HPA or KEDA ScaledObject
   - Scale to zero: Only with KEDA
   - Routing: Gateway API HTTPRoute or Istio VirtualService
   - mTLS: Optional (depends on service mesh configuration)

3. ModelMesh Mode (High-density serving):
   - Autoscaling: HPA
   - Multi-model: Yes (many models per pod)
   - Routing: ModelMesh router
   - mTLS: Enforced via service mesh
