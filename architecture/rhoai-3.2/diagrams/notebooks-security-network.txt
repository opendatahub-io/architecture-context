  - Auth: Bearer Token validated against Kubernetes RBAC
  - Policy: Per-user ServiceAccount binding

- Workbench UI access: kube-rbac-proxy (RHOAI 3.x) / oauth-proxy (RHOAI 2.x)
- Enforcement: odh-notebook-controller injects auth sidecar into every workbench pod
- Policy: Per-user RBAC via ServiceAccount
