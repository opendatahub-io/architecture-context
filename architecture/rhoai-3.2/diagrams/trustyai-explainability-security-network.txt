Authentication & Authorization:
- All REST endpoints: No auth in service code
  - In RHOAI 3.x: kube-rbac-proxy sidecar provides external auth
- /q/metrics: Bearer Token (ServiceAccount token)
  - Enforcement: ServiceMonitor configuration
  - Token path: /var/run/secrets/kubernetes.io/serviceaccount/token

Secrets:
- TLS certs at /etc/tls/internal/ (kubernetes.io/tls)
  - Purpose: HTTPS on port 4443
  - Provisioned by: TrustyAI Service Operator / cert-manager
  - Auto-Rotate: Unknown

- Database credentials (Opaque)
  - Keys: QUARKUS_DATASOURCE_USERNAME, QUARKUS_DATASOURCE_PASSWORD
  - Provisioned by: Operator / Manual
  - Auto-Rotate: No

- MinIO credentials (Opaque)
  - Keys: minio.access-key, minio.secret-key
  - Provisioned by: Operator / Manual
  - Auto-Rotate: No

- ENABLED: Fairness metrics (SPD, DIR), Drift metrics (KS Test, MMD, Meanshift)
