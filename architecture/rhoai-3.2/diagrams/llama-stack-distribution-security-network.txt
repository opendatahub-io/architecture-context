Security Notes:
- The Llama Stack server does NOT enforce authentication at the container level
- All authentication is delegated to the platform ingress layer
- TLS is not terminated by the container; platform manages TLS
- Provider connections support configurable TLS verification
- PostgreSQL connection uses plaintext by default (configurable)
