UNPROTECTED ENDPOINTS:
- /health       (GET) - Health check, no auth
- /version      (GET) - Server version, no auth
- /metrics      (GET) - Prometheus metrics, no auth (when enabled)
- /             (GET) - React UI index.html, no auth
- /static-files/* (GET) - Static UI assets, no auth

PROTECTED ENDPOINTS (mlflow.kubeflow.org RBAC):
- /api/2.0/mlflow/*        → experiments (get, list, create, update, delete)
- /ajax-api/2.0/mlflow/*   → experiments (get, list, create, update, delete)
- /api/3.0/mlflow/traces/* → experiments (get, list)
- /ajax-api/3.0/jobs/*     → jobs (get, list, create)
- /v1/traces               → experiments (create)
- /graphql                 → experiments + registeredmodels (per-operation)
