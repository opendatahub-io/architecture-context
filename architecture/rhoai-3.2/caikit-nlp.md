## Purpose
**Short**: Caikit-NLP is a Python library providing Natural Language Processing capabilities (text generation, embeddings, reranking, classification) built on the Caikit AI framework for serving NLP models in RHOAI.

**Detailed**: Caikit-NLP implements the Caikit framework's "task" and "module" abstractions to provide a comprehensive set of NLP capabilities. It supports text generation via prompt tuning (PEFT), fine-tuning, and remote inference through TGIS (Text Generation Inference Server); text embeddings via sentence-transformers; reranking via both bi-encoder and cross-encoder models; sentence similarity scoring; and text/token classification.

The library is packaged as a container image that serves as the NLP runtime within the RHOAI platform. When deployed, it runs as a Caikit runtime server exposing both HTTP (port 8080) and gRPC (port 8085) interfaces for model inference. It integrates with TGIS backend for efficient large language model serving, supporting both local model loading and remote model connections. The library supports Hugging Face model bootstrapping, allowing direct download and conversion of models from the Hugging Face Hub.

Caikit-NLP is not a Kubernetes operator — it is a Python library and container image that is deployed by other RHOAI components (such as KServe/ModelMesh) as part of the model serving infrastructure. It contains no CRDs, controllers, or Kubernetes manifests.
