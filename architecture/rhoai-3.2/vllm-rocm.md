## Purpose

**Short**: GPU-accelerated vLLM container image built with AMD ROCm for high-performance LLM inference serving on Red Hat OpenShift AI.

**Detailed**: This repository is a **container image build/packaging repository** for the vLLM inference serving runtime, specifically targeting AMD ROCm GPUs. It does not contain the vLLM source code itself; instead, it layers a thin configuration on top of the upstream Red Hat AI Infrastructure Services (RHAIIS) base image `registry.redhat.io/rhaiis/vllm-rocm-rhel9`.

The repository provides the `Dockerfile.konflux.rocm` build file used by the Konflux CI/CD system to produce the `odh-vllm-rocm-rhel9` container image. The image is configured to run vLLM with the TGIS (Text Generation Inference Server) adapter, exposing both an OpenAI-compatible HTTP API (port 8000) and a gRPC endpoint (port 8033). This enables the image to serve as a drop-in inference runtime for KServe/ModelMesh within the RHOAI platform.

The actual vLLM engine provides PagedAttention-based memory management, continuous batching, speculative decoding, quantization support (GPTQ, AWQ, INT4, INT8, FP8), and distributed inference capabilities (tensor/pipeline/data/expert parallelism) — all optimized for AMD ROCm/HIP GPU acceleration.
