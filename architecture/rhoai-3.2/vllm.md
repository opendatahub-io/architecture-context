## Purpose
**Short**: GPU-accelerated LLM inference serving runtime built on vLLM with NVIDIA CUDA, packaged as a container image for RHOAI model serving.

**Detailed**: This repository is a Konflux build-wrapper that produces the `odh-vllm-cuda-rhel9` container image for Red Hat OpenShift AI (RHOAI). It does not contain application source code; instead, it layers on top of the Red Hat AI Inference Server (RHAIIS) `vllm-cuda-rhel9` base image, which contains the full vLLM inference engine with CUDA support.

The resulting container image provides high-performance LLM inference using vLLM's PagedAttention memory management, continuous batching, and CUDA kernel optimizations. The entrypoint invokes the `vllm-tgis-adapter`, which wraps the vLLM OpenAI-compatible API server and exposes an additional gRPC interface compatible with the Text Generation Inference Server (TGIS) protocol. This enables the runtime to be used with KServe/ModelMesh in RHOAI for serving large language models on NVIDIA GPU hardware.

The build pipeline is managed entirely through Konflux (Tekton-based CI/CD), building multi-architecture images for amd64 and arm64 platforms. The image is intended to be deployed as a KServe InferenceService or ServingRuntime pod, not as a standalone service.
