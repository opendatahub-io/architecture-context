- No ClusterRoles defined
- No RoleBindings defined
- No ServiceAccounts created by this component
- The init container runs under the model-catalog Pod's ServiceAccount
  (managed by model-registry-operator)

Secrets:
- Kafka credentials (SASL_USERNAME/SASL_PASSWORD)
  Type: Environment variables
  Scope: Build-time only (NOT deployed to cluster)
  Provisioned: Manual configuration
  Auto-Rotate: No

- SSL certificates (SSL_CA_LOCATION, SSL_CERTIFICATE_LOCATION, SSL_KEY_LOCATION)
  Type: File paths
  Scope: Build-time only (NOT deployed to cluster)
  Provisioned: Manual configuration
  Auto-Rotate: No

FIPS Compliance:
- Shipped image: Data-only container based on UBI9-minimal
- No compiled binaries, no cryptographic operations in shipped image
- No TLS configuration in runtime (no network services)
- Kafka processor uses SASL_SSL but is NOT shipped/deployed

Network Attack Surface: MINIMAL
- Runtime: Zero network exposure (data-only init container)
- Build-time: Single outbound Kafka connection with SASL_SSL + SCRAM-SHA-512
