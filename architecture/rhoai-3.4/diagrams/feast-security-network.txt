OIDC Token Validation:

- JWKS validation: Yes (fetched from OIDC discovery endpoint)
- Leeway: 10 seconds
- verify_aud: False (audience not validated)
- K8s SA token detection: via "kubernetes.io" claim

================================================================================
  FIPS COMPLIANCE
================================================================================

Build-Time:
- feast-operator: GOEXPERIMENT=strictfipsruntime, CGO_ENABLED=1 (dynamic linking)
- feature-server: UBI9 Python 3.12 base (OpenSSL-backed)
- Base image: UBI9 ubi-minimal

Application-Level:
- TLS: Go stdlib crypto/tls, HTTP/2 disabled (NextProtos: ["http/1.1"])
- No custom cipher suites configured
- Redis TLS: default tls.Config{} (inherits system defaults)
- OIDC: PyJWT RS256 with JWKS

Risks:
- InsecureSkipVerify=true for intra-cluster registry HTTP client
- Intra-comm JWT token stored in ConfigMap (not encrypted at rest)
- No rpms.lock.yaml for OS package pinning
