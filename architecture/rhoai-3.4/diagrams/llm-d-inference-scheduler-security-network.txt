  - Sidecar: -tags strictfipsruntime, CGO_ENABLED=1, dynamic linking
  - Base image: UBI9 ubi-minimal (OpenSSL included)

  - Risk: InsecureSkipVerify configurable per stage; models scraping
    defaults to InsecureSkipVerify=true

Security Concerns:
  1. No encryption on ext-proc gRPC (9002) — relies on network policy
  2. No encryption on ZMQ KV-Events (5556/5557) — cluster-internal only
  3. OpenTelemetry export (4317) uses insecure gRPC
  4. SSRF protection is opt-in (--enable-ssrf-protection=false by default)
  5. InsecureSkipVerify=true default for /v1/models scraping
  6. Self-signed sidecar certs have 10-year validity (no auto-rotation)
