Authentication & Authorization:

- /health, /info endpoints: No authentication (open access)
- Header rewriting: X-Forwarded-Access-Token → Authorization: Bearer (forwarded to backends)
- Backend auth: Per-service TLS config (mTLS certs) or API token (Bearer) for OpenAI backends
