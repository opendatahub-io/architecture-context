TLS Certificate Chain:
  KubeRay Operator creates CA Secret → SDK reads CA cert+key →
  SDK generates RSA-3072 client cert (x509v3: BasicConstraints,
  KeyUsage=digitalSignature+keyEncipherment,
  ExtendedKeyUsage=serverAuth+clientAuth,
  SAN={head-svc DNS names}) →
  Stored at ~/.local/share/codeflare/tls/ (0600) →
  Used for mTLS connection to Ray Head (10001/TCP)
  Cleaned up on cluster.down()
