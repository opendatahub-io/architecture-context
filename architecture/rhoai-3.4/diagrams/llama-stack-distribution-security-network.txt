RBAC Summary:
  This component does not define Kubernetes RBAC resources.
  RBAC is managed at the platform level by the RHOAI operator.

Authentication Summary:
  - Container-level: No authentication enforced by Llama Stack server
  - Platform-level: Authentication enforced via kube-rbac-proxy sidecars
    or Gateway API (HTTPRoute) managed by RHOAI operator
  - All /v1/* endpoints: Platform-managed authentication
