Notes

- Ingress to Llama Stack server is managed by llama-stack-operator
- All S3 communication uses HTTPS/TLS 1.2+
- KFP API communication uses HTTPS/TLS 1.2+ with Bearer Token auth
- Inference backend communication is typically plaintext HTTP in-cluster
- No CRDs are defined by this component
- LlamaStackDistribution CR (llamastack.io/v1alpha1) is from llama-stack-operator
