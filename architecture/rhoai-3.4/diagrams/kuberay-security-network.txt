NetworkPolicy Rules (when odh.ray.io/secure-trusted-network: true):

  {cluster}-head NetworkPolicy:
  - Pod Selector: ray.io/cluster={name}, ray.io/node-type=head
  - Ingress Rules:
    1. Intra-cluster (all ports) - ray.io/cluster={name}
    2. Same-namespace pods (10001, 8265)
    3. Operator namespace (8265, 10001) - opendatahub
    4. Monitoring namespace (8080) - optional
    5. Gateway namespace (8265) - openshift-ingress
    6. All sources (8443 only) - secured via kube-rbac-proxy

  {cluster}-workers NetworkPolicy:
  - Pod Selector: ray.io/cluster={name}, ray.io/node-type=worker
  - Ingress Rules:
    1. Intra-cluster only (all ports) - ray.io/cluster={name}

  Webhook endpoints:
