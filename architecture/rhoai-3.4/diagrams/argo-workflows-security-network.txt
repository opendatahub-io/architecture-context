- workflow-controller: GOEXPERIMENT=strictfipsruntime, CGO_ENABLED=1 (FIPS only)
- argoexec: Dual binary — argoexec-fips (CGO_ENABLED=1, FIPS) + argoexec (CGO_ENABLED=0, static)
- Base image: UBI9 ubi-minimal (OpenSSL available)

Known TLS Risks:
- InsecureSkipVerify: true on self-signed TLS configs (util/tls/tls.go:127)
- InsecureSkipVerify: true on argoexec agent HTTP client (workflow/executor/agent.go:303)
- InsecureSkipVerify configurable per HTTP artifact (pkg/apis/workflow/v1alpha1/http_types.go:53-54)
- Mitigated: In RHOAI, controller and argoexec communicate via K8s API, not direct connections
