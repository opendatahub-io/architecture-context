Authentication & Authorization:
==============================================================================

  Argo Server API (2746/TCP):
  - Client mode: Bearer Token (Kubernetes ServiceAccount) -> Kubernetes RBAC
  - Server mode: Server ServiceAccount (impersonation) -> Server SA permissions
  - SSO mode: OIDC Token (via SSO prefix) -> OIDC + optional RBAC mapping
  - Enforcement: Gatekeeper interceptor

  - argo-workflows-agent-ca-certificates (Opaque)
