Secrets:
- No secrets directly consumed by the library
- Redis/Valkey credentials: Passed via connection URL (redis:// or rediss://)
- HuggingFace tokens: Passed via environment variables by consuming deployment
- No TLS certificates managed by this component

Security Notes:
- ALL internal communication is PLAINTEXT (architecturally intentional):
  - ZMQ PUB/SUB (5557/TCP): No encryption, no authentication
  - gRPC over UDS: No encryption (local IPC, filesystem permissions only)
  - IndexerService gRPC (50051/TCP, example only): No encryption, no auth
  - OTLP export (4317/TCP): Insecure gRPC by default
- Security boundary: Relies entirely on Kubernetes network policies
  and namespace isolation
- Redis/Valkey: Supports optional TLS via rediss:// URL scheme
- External egress (Hugging Face Hub): Uses TLS 1.2+

FIPS Compliance Notes:
- CGO_ENABLED=1 (required for Python/ZMQ cgo linking)
- No explicit GOEXPERIMENT=strictfipsruntime build flag
- UBI 9 base image includes OpenSSL
- gRPC connections use insecure.NewCredentials() (plaintext)
- Non-FIPS crypto risks: ZMQ, gRPC, OTLP all use plaintext by default
