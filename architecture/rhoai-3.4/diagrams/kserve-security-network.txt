================================================================================
  FIPS Compliance
================================================================================

Build-Time:
  - GOEXPERIMENT=strictfipsruntime, CGO_ENABLED=1, -tags strictfipsruntime
  - Dynamic linking with OpenSSL via UBI 9 base image
  - Python: cryptography >= 46.0.5 (OpenSSL backend)

Application-Level:
  - llmisvc controller disables HTTP/2 (NextProtos: ["http/1.1"])
  - Self-signed certs use Go stdlib crypto (FIPS-compliant with strictfipsruntime)

Non-FIPS Crypto Risks:
  - PodMonitor uses InsecureSkipVerify=true for vLLM engine scraping
  - DestinationRules use InsecureSkipVerify=true for scheduler TLS
  - Self-signed workload certs (10-year validity)
