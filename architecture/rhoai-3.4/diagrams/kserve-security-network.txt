================================================================================
  FIPS COMPLIANCE
================================================================================

Build flags: GOEXPERIMENT=strictfipsruntime, -tags strictfipsruntime
Linking: Dynamic (CGO_ENABLED=1)
OpenSSL: Yes (UBI9 base image: registry.access.redhat.com/ubi9/ubi-minimal)
TLS config: LLMIsvc controller disables HTTP/2; InferenceGraph router uses
            Go default TLS. All crypto via stdlib crypto/tls (FIPS via OpenSSL).
Python: storage-initializer uses UBI9 OpenSSL backend (FIPS-compliant).
