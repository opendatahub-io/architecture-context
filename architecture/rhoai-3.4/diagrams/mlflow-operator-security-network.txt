Authentication & Authorization:
================================================================================
- MLflow API (/mlflow/*): Kubernetes RBAC via SelfSubjectAccessReview
  Enforcement: MLflow kubernetes-auth plugin
  Checks caller's K8s token against pseudo-resources

- Operator Metrics (/metrics): Bearer Token + TokenReview + SubjectAccessReview
  Enforcement: controller-runtime WithAuthenticationAndAuthorization
  Only authorized SAs with metrics-reader role

- MLflow Prometheus (/prometheus): Kubernetes RBAC (same as API)
  Enforcement: MLflow kubernetes-auth plugin
  Prometheus ServiceMonitor uses bearer token
