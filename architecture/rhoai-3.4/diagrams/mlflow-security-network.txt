RBAC Summary:

  Virtual Resources (mlflow.kubeflow.org API Group - enforced via SubjectAccessReview):
  - experiments:              get, list, create, update, delete
  - registeredmodels:         get, list, create, update, delete
  - datasets:                 get, list, create, update, delete
  - gatewaysecrets:           get, list, create, update, delete
  - gatewayendpoints:         get, list, create, update, delete
  - gatewaymodeldefinitions:  get, list, create, update, delete
  - assistants:               get, list, create, update, delete

  Server ServiceAccount Permissions (required for MLflow server pod):
  - core/namespaces:                              list, watch
  - core/secrets:                                 get
  - mlflow.kubeflow.org/mlflowconfigs:            list, watch
  - authentication.k8s.io/tokenreviews:           create
  - authorization.k8s.io/subjectaccessreviews:    create

CRDs:
  - MLflowConfig (mlflow.kubeflow.org/v1) - Namespaced
    Purpose: Per-namespace artifact storage overrides
    Fields: artifactRootSecret, artifactRootPath
    Note: Optional; CRD definition installed separately

Secrets:
  - Per-namespace artifact secret (Opaque) - AWS_S3_BUCKET for per-NS artifact storage
    Provisioned by: Namespace administrator | Auto-Rotate: No
  - MLFLOW_CRYPTO_KEK_PASSPHRASE (env var) - Key Encryption Key for gateway secrets
    Provisioned by: Server administrator | Rotate: mlflow crypto rotate-kek
  - MLFLOW_FLASK_SERVER_SECRET_KEY (env var) - Flask session signing key
    Provisioned by: Server administrator | Auto-Rotate: No
  - Database connection string (env var) - PostgreSQL credentials
    Provisioned by: Deployment configuration | Auto-Rotate: No

Security Middleware:
  - HostValidationMiddleware: DNS rebinding protection via Host header validation
  - CORSMiddleware: Cross-origin request protection
  - CORSBlockingMiddleware: Active blocking of unauthorized cross-origin state-changing requests
  - SecurityHeadersMiddleware: X-Content-Type-Options: nosniff, configurable X-Frame-Options
