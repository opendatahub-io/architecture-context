RBAC:
  None — Batch Gateway does not define RBAC roles. It operates as a user-space
  application with no Kubernetes API access. Each component uses a dedicated
  ServiceAccount with automount enabled but no RBAC bindings.

FIPS COMPLIANCE WARNING:
  CGO_ENABLED=0 in all Konflux Dockerfiles → static Go binaries with pure-Go
  crypto. Will fail check-payload FIPS validation. No GOEXPERIMENT=strictfipsruntime
  flag present. TLS 1.2+ minimum with stdlib crypto/tls (no MD5/RC4/DES).

SERVICE MESH:
  Not configured — Batch Gateway does not use Istio/OSSM sidecar injection
  or PeerAuthentication policies. mTLS to inference gateway is application-level
  (configured via TLS client cert/key in processor config).
