  Supported Cipher Suites:
  - TLS_ECDHE_RSA_WITH_AES_128_GCM_SHA256

  - TLS_ECDHE_RSA_WITH_CHACHA20_POLY1305
  - TLS_ECDHE_ECDSA_WITH_CHACHA20_POLY1305
  Minimum TLS Version: 1.2

TLS Configuration (Inference Client):
  - Custom CA certificates for private CAs
  - Client certificate + key for mTLS
  - Configurable TLS min/max version
  - InsecureSkipVerify (testing only)

Secrets:
  - {release}-apiserver-tls (kubernetes.io/tls) - API server HTTPS cert/key, Manual or cert-manager, Auto-rotate if cert-manager
  - {release}-processor-tls (kubernetes.io/tls) - Processor observability server TLS cert/key, Manual, No auto-rotate
  - Processor database secret (Opaque) - Database connection URL, User-provided, No auto-rotate
