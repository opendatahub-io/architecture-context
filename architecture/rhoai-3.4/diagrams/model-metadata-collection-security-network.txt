================================================================================
  ATTACK SURFACE SUMMARY
================================================================================

  Runtime Attack Surface: MINIMAL
  - No network listeners (no ports exposed)
  - No network egress
  - No compiled binaries in shipped image
  - No Kubernetes API access
  - No secrets at runtime
  - Container runs as data-only volume (sleep infinity)
  - Base image: ubi9-minimal (minimal OS packages)

  Build-Time Attack Surface:
  - HTTPS egress to huggingface.co (443/TCP) - model metadata
  - HTTPS egress to registry.redhat.io (443/TCP) - OCI image pulls
  - HF_TOKEN secret in CI environment
  - Hermetic build enforced (gomod prefetch via Tekton)
