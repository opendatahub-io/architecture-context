FIPS Compliance:
================

  Build-Time:
  - CGO_ENABLED=1, GOEXPERIMENT=strictfipsruntime (both binaries)
  - Dynamic linking (CGO_ENABLED=1), OpenSSL via UBI9 base
  - Base image: registry.access.redhat.com/ubi9/ubi-minimal

  Application-Level:
  - TLS: Configurable MinVersion (default 1.2), Go default cipher suites
  - Crypto: stdlib crypto/tls, crypto/sha256, crypto/subtle (FIPS via OpenSSL)
  - Risk: InsecureSkipVerify=true for cluster-internal model probes (documented)
  - No MD5/RC4/DES usage detected

Defense-in-Depth Security Layers:
=================================

  Layer 1: Authorino validates credentials + injects identity headers
  Layer 2: OPA rules in AuthPolicy enforce group-based access
  Layer 3: Limitador enforces token rate limits (TokenRateLimitPolicy)
  Layer 4: NetworkPolicy restricts internal endpoint access
  Layer 5: Authorization header stripped before forwarding to backends
