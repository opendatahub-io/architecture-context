## Purpose
**Short**: A secure, batch-oriented container runtime for executing language model evaluations using the lm-evaluation-harness framework, integrated with EvalHub for job orchestration and result reporting.

**Detailed**: The LM Evaluation Harness (LMES Job) is a Python-based container image that provides a standardized runtime environment for evaluating large language models against a comprehensive suite of benchmark tasks. It is a downstream fork of the EleutherAI lm-evaluation-harness project, adapted for Red Hat OpenShift AI (RHOAI) with multi-architecture container builds, FIPS compliance, disconnected operation support, and EvalHub integration.

The component operates as a Kubernetes batch Job, not as a long-running service. When launched, it reads a job specification (typically mounted as a ConfigMap at `/meta/job.json`), configures the evaluation parameters, connects to an OpenAI-compatible model inference endpoint to run benchmark evaluations, and reports results back to EvalHub via callback URLs. It supports over 150 evaluation task families (including standard benchmarks like ARC, BBH, MMLU, and many more) and includes offline metric computation for disconnected environments.

The container also supports OCI artifact persistence via `skopeo` and `olot`, enabling evaluation results and traces to be pushed to OCI registries. S3-compatible storage is supported for downloading model assets and datasets, with configurable SSL/TLS certificate verification for enterprise environments.

### Multi-Architecture Build

The Konflux build (`Dockerfile.konflux.lmes-job`) is a 5-stage multi-architecture build:
1. **Base builder** (UBI9 Python 3.11): Common tooling, skopeo, architecture-specific compilers
2. **Arrow builder**: Compiles Apache Arrow from source for ppc64le and s390x
3. **Torch builder**: Compiles PyTorch from source for ppc64le and s390x
4. **OpenBLAS builder**: Compiles OpenBLAS for ppc64le (POWER9 target)
5. **Final image**: Assembles all wheels, installs Python dependencies, configures runtime
