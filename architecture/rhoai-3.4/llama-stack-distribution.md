## Purpose
**Short**: A containerized Llama Stack distribution providing a unified AI/ML API server for inference, agents, safety, evaluation, and vector I/O within the Red Hat OpenShift AI platform.

**Detailed**: The Llama Stack Distribution is a container image that packages Meta's Llama Stack server with a curated set of providers and dependencies tailored for the Red Hat OpenShift AI (RHOAI) platform. It provides a single API endpoint (port 8321) that exposes multiple AI/ML capabilities including inference (via vLLM, AWS Bedrock, Azure, Vertex AI, WatsonX, OpenAI), vector storage (Milvus, pgvector, Qdrant, FAISS), safety evaluation (TrustyAI FMS, passthrough), model evaluation (LM Eval, RAGAS, Garak), agents with tool use, and dataset I/O.

The distribution is built using Konflux CI/CD pipelines for RHOAI production releases (multi-arch: x86_64, arm64) and GitHub Actions for upstream community builds. It uses PostgreSQL as its primary persistence backend for key-value storage, inference logging, agent state, and file metadata. The server supports OpenTelemetry instrumentation for traces and metrics export. Providers are conditionally enabled via environment variables, allowing flexible deployment configurations.
