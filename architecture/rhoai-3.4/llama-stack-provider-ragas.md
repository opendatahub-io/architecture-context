## Purpose
**Short**: Ragas evaluation framework as an out-of-tree Llama Stack provider for LLM evaluation via the Llama Stack Eval API.

**Detailed**: This component implements [Ragas](https://github.com/explodinggradients/ragas) as an external evaluation provider for Meta's [Llama Stack](https://github.com/meta-llama/llama-stack) platform. It exposes Ragas' evaluation metrics (answer relevancy, context precision, faithfulness, context recall) through the Llama Stack Eval API, enabling standardized LLM evaluation workflows within the RHOAI ecosystem.

The provider operates in two modes: **inline** and **remote**. The inline provider runs Ragas evaluation in the same process as the Llama Stack server, using the server's built-in inference and embedding APIs. The remote provider offloads evaluation to Kubeflow Pipelines (Data Science Pipelines in RHOAI), where evaluation jobs execute as KFP pipeline runs with results persisted to S3-compatible storage. Both modes wrap Llama Stack's inference API to provide LLM and embedding capabilities to the Ragas framework.

The component is not an operator or standalone service. It is a Python library that plugs into a Llama Stack server distribution, and a container image used as the base image for Kubeflow Pipeline components when running in remote mode. In OpenShift, it is typically deployed as part of a LlamaStackDistribution custom resource managed by the llama-stack-operator.

### Llama Stack Eval API (Provider Implementation)

This component implements the Llama Stack `Eval` API interface:

| Method | Purpose | Provider Modes |
|--------|---------|----------------|
| `run_eval(benchmark_id, benchmark_config)` | Submit evaluation job against a registered benchmark | inline, remote |
| `evaluate_rows(benchmark_id, input_rows, scoring_functions, benchmark_config)` | Evaluate specific rows (not implemented) | N/A |
| `job_status(benchmark_id, job_id)` | Check evaluation job status | inline, remote |
| `job_result(benchmark_id, job_id)` | Retrieve evaluation results | inline, remote |
| `job_cancel(benchmark_id, job_id)` | Cancel a running evaluation job | remote only |
| `register_benchmark(task_def)` | Register a benchmark definition | inline, remote |

### HTTP Endpoints

This component does not directly expose HTTP endpoints. It runs within a Llama Stack server which exposes its own HTTP API (default port 8321/TCP). The provider is accessed through the Llama Stack server's eval API endpoints.

### Evaluation Metrics Supported

| Metric Name | Ragas Function | Purpose |
|-------------|----------------|---------|
| answer_relevancy | `ragas.metrics.answer_relevancy` | Measures how relevant the answer is to the question |
| context_precision | `ragas.metrics.context_precision` | Measures precision of retrieved contexts |
| faithfulness | `ragas.metrics.faithfulness` | Measures factual consistency of answer with context |
| context_recall | `ragas.metrics.context_recall` | Measures recall of relevant context |
