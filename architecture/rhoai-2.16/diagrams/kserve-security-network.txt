DEPLOYMENT MODES
════════════════════════════════════════════════════════════════════════════════════════

1. Serverless Mode (Default - Knative)
   - Autoscaling: Knative Pod Autoscaler (KPA) - scale to zero, rapid scale-up
   - Dependencies: Knative Serving, Istio
   - Use Case: Variable load, cost optimization with scale-to-zero
   - Min Replicas: 0 (scale to zero after idle period)
   - Max Replicas: 100 (configurable)

2. RawDeployment Mode
   - Autoscaling: Kubernetes HPA (Horizontal Pod Autoscaler)
   - Dependencies: None (standard K8s only)
   - Use Case: Lightweight, no Knative dependency
   - Min Replicas: 1
   - Max Replicas: 10 (configurable)

3. ModelMesh Mode
   - Autoscaling: Fixed replicas (not autoscaling)
   - Dependencies: ModelMesh installation
   - Use Case: High-density multi-model serving, frequently changing models
   - Replicas: 3-5 (typical)
