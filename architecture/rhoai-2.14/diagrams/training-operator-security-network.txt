Security Best Practices:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
✓ Non-root user (UID 65532) for operator container
✓ FIPS mode enabled for cryptographic operations
✓ No privilege escalation allowed
✓ ServiceAccount with least-privilege RBAC
✓ TLS 1.2+ for all external communication
✓ Red Hat UBI base image with security updates

⚠ Training pod inter-worker communication uses plaintext (23456/TCP)
  - Mitigation: Use network policies to restrict pod-to-pod traffic
  - Alternative: Run training in isolated namespaces

⚠ Metrics and health endpoints use HTTP (no encryption)
  - Acceptable: Internal-only access via ClusterIP service
  - Mitigation: Network isolation ensures endpoints not exposed externally
