  - Enforcement Point: OAuth Proxy Sidecar

  - Auth Mechanism: mTLS client certificates (optional)
  - Enforcement Point: Ray Head Pod

  - Auth Mechanism: Kubernetes API Server mTLS
  - Enforcement Point: Kubernetes API Server
  - Policy: Valid API server client certificate required

Endpoint: Metrics Endpoint (port 8080)
  - Methods: GET
  - Auth Mechanism: None (protected by NetworkPolicy)
  - Enforcement Point: NetworkPolicy
  - Policy: Accessible from openshift-monitoring namespace only
