Service Mesh Configuration (Optional):
- PeerAuthentication: Optional STRICT mode (namespace-scoped)
- AuthorizationPolicy: Optional policies for /v2/models/* endpoints
  - Can restrict inference requests by source service identity
  - Default: No restrictions (open to all)

Network Policies:
- Policy: modelmesh-controller
  - Pod Selector: control-plane=modelmesh-controller
  - Ingress: Allow 8443/TCP (metrics from Prometheus)
  - Egress: Allow all

- Policy: modelmesh-runtimes
  - Pod Selector: modelmesh-service exists
  - Ingress: Allow 8033/TCP (gRPC), 8008/TCP (HTTP), 8080/TCP (internal), 2112/TCP (metrics)
  - Egress: Allow all (for S3, Etcd communication)

- Policy: modelmesh-webhook
  - Pod Selector: control-plane=modelmesh-controller
  - Ingress: Allow 9443/TCP (webhook from Kubernetes API)
  - Egress: Allow all

Secrets:
- model-serving-etcd (Opaque)
  - Keys: etcd_connection_url, etcd_username, etcd_password
  - Provisioned: Manual/Operator
  - Auto-Rotate: No

- storage-config (Opaque)
  - Keys: AWS_ACCESS_KEY_ID, AWS_SECRET_ACCESS_KEY, storage endpoint config
  - Provisioned: User/Administrator
  - Auto-Rotate: No

- webhook-server-cert (kubernetes.io/tls)
  - Purpose: TLS certificate for webhook server (9443/TCP)
  - Provisioned: cert-manager or Manual
  - Auto-Rotate: Yes (if cert-manager)
  - Validity: 90 days (cert-manager default)

- modelmesh-serving-cert (kubernetes.io/tls)
  - Purpose: Optional mTLS between ModelMesh components
  - Provisioned: cert-manager or Manual
  - Auto-Rotate: Yes (if cert-manager)

Deployment Modes:
- Cluster-scoped (default): Controller watches all namespaces
- Namespace-scoped: Controller watches single namespace (NAMESPACE_SCOPE=true)

High Availability:
- Controller replicas: 1 (default) or 3 (HA mode)
- Leader election: Lease-based (15s duration, 10s renew, 2s retry)
- Runtime pods: Horizontal Pod Autoscaling supported based on CPU/memory

Resource Limits:
- Controller: 50m CPU request, 96Mi memory request, 1 CPU limit, 512Mi memory limit
- Runtime pods: Configured per ServingRuntime definition
