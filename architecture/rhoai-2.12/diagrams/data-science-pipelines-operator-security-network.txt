Trust Boundaries:
===============================================================================
1. External → Ingress: TLS 1.2+ encryption, public internet to OpenShift Router
2. Ingress → DMZ: TLS reencryption, OAuth2 authentication enforced
3. DMZ → Application: Plaintext HTTP (localhost sidecar), delegated auth
4. Application → Data: Plaintext MySQL/HTTP (internal services), credential-based auth
5. Application → External: TLS 1.2+ (S3, external DB), credential or IAM-based auth
