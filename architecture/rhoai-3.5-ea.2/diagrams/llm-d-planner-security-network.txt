Authentication Summary:
- External access: No user authentication (relies on Route-level access control)
- Admin DB endpoints: x-admin-password header with HMAC comparison (optional)
- Model Catalog: Bearer Token (ServiceAccount mount or explicit)
- Kubernetes API: ServiceAccount Token (auto-mounted)
- HuggingFace: Bearer Token (HF_TOKEN env, optional)
- OpenAI: Bearer Token (OPENAI_API_KEY env, conditional)
- Vertex AI: GCP IAM (GCP_CREDENTIALS_JSON env, conditional)
- PostgreSQL: Password (via DATABASE_URL connection string)
- Ollama: No authentication
