RBAC Summary:
- ClusterRole: argo-cluster-role (workflow-controller)
  - argoproj.io: workflows, workflowtasksets, workflowartifactgctasks (full CRUD + finalizers)
  - argoproj.io: workflowtemplates, clusterworkflowtemplates (get, list, watch)
  - argoproj.io: workflowtaskresults (list, watch, deletecollection)
  - argoproj.io: cronworkflows (full CRUD + finalizers)
  - core: pods, pods/exec (create, get, list, watch, update, patch, delete)
  - core: configmaps (get, watch, list)
  - core: persistentvolumeclaims (create, update, delete, get)
  - core: serviceaccounts (get, list)
  - core: events (create, patch)
  - core: secrets - RESTRICTED to "argo-workflows-agent-ca-certificates" (get only)
  - policy: poddisruptionbudgets (create, get, delete)

- ClusterRole: argo-server-cluster-role (argo-server)
  - argoproj.io: eventsources, sensors, workflows, workfloweventbindings,
    workflowtemplates, cronworkflows, clusterworkflowtemplates (full CRUD)
  - core: configmaps (get, watch, list)
  - core: secrets (get, create)
  - core: pods, pods/exec, pods/log (get, list, watch, delete)
  - core: events (watch, create, patch)
  - core: serviceaccounts (get, list, watch)

- Aggregate Roles:
  - argo-aggregate-to-view: read-only on Argo CRDs
  - argo-aggregate-to-edit: CRUD on Argo CRDs
  - argo-aggregate-to-admin: full admin on Argo CRDs + workflowtasksets

Secrets:
- argo-workflows-agent-ca-certificates (Opaque) - CA certs for agent HTTPS, Manual, No rotation
- argo-workflows-webhook-clients (Opaque) - Webhook creds for GitHub/GitLab/Bitbucket, Manual, No rotation
- sso/cookieEncryptionPrivateKey (Opaque) - RSA-2048 for SSO session cookies, Auto/Manual, No rotation
- argo-server-sso-secret (Opaque) - OIDC client ID/secret, Manual, No rotation
- workflow-controller-configmap (ConfigMap) - Controller config (artifact repos, persistence), Manual

FIPS Compliance:
- Build: CGO_ENABLED=1, GOEXPERIMENT=strictfipsruntime, godebug fips140=on
- workflow-controller: FIPS-only binary
- argoexec: Dual binaries (FIPS + standard), runtime selection
- Base image: UBI9 ubi-minimal with OpenSSL
- RISKS: agent executor defaults InsecureSkipVerify=true; SSO has InsecureSkipVerify option

Authentication Mechanisms:
- Argo Server API: Bearer Token (SA) / SSO (OIDC JWT) / Client (kubeconfig)
- Controller → K8s API: ServiceAccount Token (Bearer)
- Executor → K8s API: ServiceAccount Token (Bearer)
- Artifact Repos: Access Key/Secret Key (S3), SA Key (GCS), Account Key (Azure)
- SQL Database: Username/Password
- OIDC SSO: Client ID/Secret with OAuth2 redirect
