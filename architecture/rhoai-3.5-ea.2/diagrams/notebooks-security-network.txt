================================================================================
  RBAC SUMMARY
================================================================================

This component does NOT define RBAC resources.
- All container images run as unprivileged user UID 1001
- RBAC is managed by the odh-notebook-controller which creates workbench pods
- kube-rbac-proxy sidecar enforces SubjectAccessReview against namespace RBAC

================================================================================
  AUTHENTICATION & AUTHORIZATION
================================================================================

Endpoint                  Auth Mechanism              Enforcement Point
─────────────────────────────────────────────────────────────────────────
JupyterLab (8888/TCP)     Bearer Token (OCP user)     kube-rbac-proxy sidecar
code-server (8787/TCP)    Bearer Token (OCP user)     kube-rbac-proxy sidecar

- Authentication delegated entirely to platform (kube-rbac-proxy)
- Workbenches serve plaintext HTTP — no application-level TLS
- TLS termination at Platform Gateway (Envoy) layer

================================================================================
  SECRETS
================================================================================

No secrets defined by this component.
- S3 credentials: User-configured at runtime
- Registry pull secrets: Managed by OpenShift platform
- TLS certificates: Managed by platform Gateway / cert-manager

================================================================================
  FIPS COMPLIANCE
================================================================================

Build-Time:
- Go binaries (mongocli): GOEXPERIMENT=strictfipsruntime, CGO_ENABLED=1
- compat-openssl11 installed in all Konflux Dockerfiles
- OpenSSL from UBI9 base image for system crypto

Application-Level:
- No application-level TLS (plaintext HTTP in pod)
- Python cryptography/pyOpenSSL use system OpenSSL
- System trust store (RHEL CA bundle) for certificates
- No non-FIPS crypto detected (no MD5/RC4)
