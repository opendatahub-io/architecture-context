- All endpoints: No built-in auth - relies on external enforcement

- Base image: UBI 9 (RHEL OpenSSL)
- MD5 usage: Cache key hashing only (non-cryptographic)
- FIPS fallback: Detects hashlib.md5 unavailability, switches to SHA256
- TLS: No explicit config; uvicorn serves HTTP only; TLS termination at ingress
- cryptography==44.0.3 via presidio-anonymizer (uses system OpenSSL)
