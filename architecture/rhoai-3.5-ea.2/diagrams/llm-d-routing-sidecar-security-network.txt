Security Notes:
- InsecureSkipVerify available for prefiller and decoder TLS (disabled by default)
- SSRF protection validates prefiller target IPs against InferencePool pod list
- Allowlist normalizes host:port to extract host only (any port on allowed host OK)
- Upstream Dockerfile uses CGO_ENABLED=0 (NOT FIPS compliant for check-payload)
- RHOAI downstream must use CGO_ENABLED=1 + GOEXPERIMENT=strictfipsruntime
- All cipher suites are FIPS-approved (ECDHE with AES-GCM, ChaCha20-Poly1305)
