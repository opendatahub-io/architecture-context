Network Policies (per notebook):
  - Policy 1: Allow controller namespace → notebook pod port 8888/TCP (idle culling)
  - Policy 2: Allow external traffic → kube-rbac-proxy port 8443/TCP (authenticated access)

Admission Webhooks:
  - notebooks.opendatahub.io (mutating, failurePolicy: Fail)
    - Injects: kube-rbac-proxy sidecar, CA bundles, proxy env vars, DSPA secrets,
      Feast config, MLflow tracking URI, ImageStream resolution, reconciliation lock
  - notebooks-validation.opendatahub.io (validating, failurePolicy: Fail)
    - Prevents: MLflow annotation removal on running notebooks
  - conversion.notebooks.kubeflow.org (conversion webhook for CRD versioning)

Secrets:
  - odh-notebook-controller-webhook-cert (kubernetes.io/tls)
    Purpose: Webhook TLS cert
    Provisioned: OpenShift service-serving-cert (auto-rotate: Yes)
  - ds-pipeline-config (Opaque, per notebook)
    Purpose: Elyra runtime config with DSPA API + S3 credentials
    Provisioned: odh-notebook-controller (auto-rotate: No)
  - {notebook-name}-kube-rbac-proxy-tls (kubernetes.io/tls, per notebook)
    Purpose: kube-rbac-proxy sidecar TLS cert
    Provisioned: External (must be pre-provisioned, auto-rotate: No)

TLS Configuration:
  - Webhook server: MinVersion TLS 1.2, cipher suites from OpenShift TLS security profile
  - Build: GOEXPERIMENT=strictfipsruntime, CGO_ENABLED=1 (FIPS-compliant)
  - Base image: UBI9 ubi-minimal (includes OpenSSL for FIPS runtime crypto)
